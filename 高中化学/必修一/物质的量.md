# 物质的量

## 一、物质的量的概念

### 1.1 引入背景

宏观物质由大量微观粒子（分子、原子、离子等）构成。化学反应中，粒子按一定数目关系进行反应，但单个粒子的数目过于庞大，无法直接计量。

为解决此问题，引入 **物质的量** 这一物理量，用于将宏观可称量的质量与微观粒子数联系起来。

### 1.2 定义与符号

| 项目 | 内容 |
|------|------|
| 物理量名称 | 物质的量 |
| 符号 | $n$ |
| 单位 | 摩尔（mol，简称"摩"） |
| 基准 | 1 mol 粒子集体所含粒子数 = 0.012 kg ¹²C 中所含碳原子数 |

> **关键区分**：物质的量 ≠ 质量，"物质的量"四个字是一个整体物理量名词，不可拆分为"物质的"+"量"。

### 1.3 阿伏伽德罗常数 ($N_A$)

**定义**：1 mol 任何粒子所含的粒子数。

**近似值**：$N_A \approx 6.02 \times 10^{23} \text{ mol}^{-1}$

**精确值**：$N_A = 6.02214076 \times 10^{23} \text{ mol}^{-1}$（2019年新定义）

> **注意**：$N_A$ 有单位 $\text{mol}^{-1}$，而 $6.02 \times 10^{23}$ 没有单位。

### 1.4 物质的量与粒子数关系

$$n = \frac{N}{N_A}$$

| 变形公式 | 用途 |
|----------|------|
| $N = n \times N_A$ | 已知物质的量求粒子数 |
| $n = \dfrac{N}{N_A}$ | 已知粒子数求物质的量 |

---

## 二、摩尔质量 ($M$)

### 2.1 定义

**单位物质的量的物质所具有的质量**。

### 2.2 符号与单位

| 项目 | 内容 |
|------|------|
| 符号 | $M$ |
| 单位 | $\text{g/mol}$（$\text{g·mol}^{-1}$） |
| 数值 | 以 g/mol 为单位时，数值上 = 相对原子质量（或相对分子质量） |

### 2.3 公式

$$n = \frac{m}{M}$$

| 变形公式 | 用途 |
|----------|------|
| $m = n \times M$ | 已知物质的量求质量 |
| $M = \dfrac{m}{n}$ | 已知质量和物质的量求摩尔质量 |
| $n = \dfrac{m}{M}$ | 已知质量求物质的量 |

### 2.4 计算示例

> 例：36 g H₂O 的物质的量是多少？
>
> 解：$M(\text{H₂O}) = 18 \text{ g/mol}$
>
> $$n = \frac{m}{M} = \frac{36}{18} = 2 \text{ mol}$$

## 三、气体摩尔体积 ($V_m$)

### 3.1 影响气体体积的因素

| 因素 | 说明 |
|------|------|
| 粒子数 | 粒子数越多，体积越大 |
| 粒子间平均距离 | **决定因素**——温度和压强 |
| 粒子自身大小 | 相对于间距可忽略不计 |

> **核心结论**：相同温度和压强下，相同物质的量的任何气体，体积大致相等。

### 3.2 定义与公式

**定义**：单位物质的量的气体所占的体积。

$$V_m = \frac{V}{n}$$

| 项目 | 内容 |
|------|------|
| 符号 | $V_m$ |
| 单位 | $\text{L/mol}$（$\text{L·mol}^{-1}$） |

### 3.3 标准状况下的气体摩尔体积

**标准状况（标况）**：温度 0°C（273 K），压强 101 kPa

在标况下，**1 mol 任何气体**的体积都约等于：

$$V_m \approx 22.4 \text{ L/mol}$$

> ⚠️ **适用条件**：
> 1. 必须是**气体**（水在标况下是液体/固体，不适用）
> 2. 必须是**标准状况**
> 3. $22.4 \text{ L/mol}$ 是**近似值**

### 3.4 阿伏伽德罗定律

**内容**：同温同压下，相同体积的任何气体含有相同数目的分子。

**数学表达**（同温同压）：

$$\frac{V_1}{V_2} = \frac{n_1}{n_2} = \frac{N_1}{N_2}$$

**推论**（阿伏伽德罗定律的四个"正比"）：

| 条件 | 关系 | 公式 |
|------|------|------|
| 同温同压 | $V \propto n$ | $\dfrac{V_1}{V_2} = \dfrac{n_1}{n_2}$ |
| 同温同容 | $P \propto n$ | $\dfrac{P_1}{P_2} = \dfrac{n_1}{n_2}$ |
| 同温同压 | $\rho \propto M$ | $\dfrac{\rho_1}{\rho_2} = \dfrac{M_1}{M_2}$ |
| 同温同压同质量 | $V \propto \dfrac{1}{M}$ | $\dfrac{V_1}{V_2} = \dfrac{M_2}{M_1}$ |

## 四、物质的量浓度 ($c_B$)

### 4.1 定义

**单位体积溶液中所含溶质 B 的物质的量**。

### 4.2 公式

$$c_B = \frac{n_B}{V}$$

| 符号    | 含义           | 单位                                  |
| ----- | ------------ | ----------------------------------- |
| $c_B$ | 溶质 B 的物质的量浓度 | $\text{mol/L}$（$\text{mol·L}^{-1}$） |
| $n_B$ | 溶质 B 的物质的量   | mol                                 |
| $V$   | **溶液的体积**    | L                                   |
|       |              |                                     |

> ⚠️ **易错点**：$V$ 是溶液体积，不是溶剂体积，也不是溶质体积 + 溶剂体积之和！

### 4.3 计算示例

> 例：将 4 g NaOH 固体溶于水配成 500 mL 溶液，求物质的量浓度。
>
> 解：$M(\text{NaOH}) = 40 \text{ g/mol}$
>
> $$n = \frac{m}{M} = \frac{4}{40} = 0.1 \text{ mol}$$
>
> $$V = 500 \text{ mL} = 0.5 \text{ L}$$
>
> $$c = \frac{n}{V} = \frac{0.1}{0.5} = 0.2 \text{ mol/L}$$

### 4.4 稀释定律

稀释前后，溶质的物质的量不变：

$$c_1 V_1 = c_2 V_2$$

## 五、各物理量之间的换算关系

### 5.1 核心换算网络

```
                    ┌──────────────────────┐
                    │   物质的量 (n/mol)     │
                    └──────┬───┬───┬───────┘
                ┌──────────┘   │   └──────────┐
          ÷NA  │        ÷Vm   │   ×M         │
               ↓              │              ↓
        粒子数 N       气体体积 V(L)       质量 m(g)
               \                           /
                \         ×c              /
                 └─────→ 溶液 V  ←────────┘
```

### 5.2 全部核心公式汇总

| 公式 | 适用范围 |
|------|----------|
| $n = \dfrac{N}{N_A}$ | 任何物质 |
| $n = \dfrac{m}{M}$ | 任何物质 |
| $n = \dfrac{V}{V_m}$ | **气体**（标况下 $V_m = 22.4 \text{ L/mol}$） |
| $n = c \times V_{溶液}$ | 溶液 |

## 六、一定物质的量浓度溶液的配制

### 6.1 主要仪器

| 仪器 | 用途 |
|------|------|
| 容量瓶 | 配制**准确体积**的溶液（有 50、100、250、500、1000 mL 规格） |
| 托盘天平（或分析天平） | 称量固体溶质 |
| 量筒 | 量取液体体积 |
| 烧杯 | 溶解/稀释溶质 |
| 玻璃棒 | 搅拌、引流 |
| 胶头滴管 | 定容时逐滴加入 |

### 6.2 容量瓶使用注意事项

1. **检漏**：加水→塞瓶塞→倒立→不漏水→正立旋转瓶塞 180°→再倒立→不漏水方可使用
2. 不可用于溶解、稀释、加热
3. 不可长期贮存溶液
4. 选择容量瓶规格时，略大于所需溶液体积（配 480 mL 选 500 mL 容量瓶）

### 6.3 配制步骤（以配制 100 mL 1.00 mol/L NaCl 溶液为例）

| 步骤 | 操作 | 要点 |
|------|------|------|
| **计算** | $m = c \times V \times M = 1.00 \times 0.100 \times 58.5 = 5.85 \text{ g}$ | 保留相应精度 |
| **称量** | 用托盘天平/分析天平称取 5.85 g NaCl | "左物右码"；NaOH 等易潮解固体用烧杯称 |
| **溶解** | 将 NaCl 倒入烧杯，加适量蒸馏水，用玻璃棒搅拌至溶解 | 加水不宜过多，洗烧杯 |
| **冷却** | 冷却至室温 | 热溶液会使容量瓶体积不准 |
| **转移** | 将溶液沿**玻璃棒**注入容量瓶 | 玻璃棒下端靠容量瓶刻度线下方内壁 |
| **洗涤** | 用蒸馏水洗涤烧杯内壁和玻璃棒 2~3 次 | 洗涤液**全部**注入容量瓶 |
| **定容** | 加蒸馏水至液面距刻度线 1~2 cm → 换**胶头滴管**逐滴加入 | 至凹液面最低点与刻度线相切 |
| **摇匀** | 盖好瓶塞，反复上下颠倒摇匀 | — |
| **装瓶贴签** | 将配好的溶液倒入试剂瓶，贴标签 | 容量瓶不能久存 |

### 6.4 误差分析

**核心公式**：$c = \dfrac{n}{V}$

| 操作 | $n$ 变化 | $V$ 变化 | $c$ 结果 |
|------|----------|----------|--------|
| 称量时砝码沾污（增重） | ↑ | — | **偏大** |
| 溶解后未冷却即转移 | — | ↓（冷缩） | **偏大** |
| 转移时有溶液溅出 | ↓ | — | **偏小** |
| 未洗涤烧杯和玻璃棒 | ↓ | — | **偏小** |
| 定容时仰视刻度线 | — | ↑ | **偏小** |
| 定容时俯视刻度线 | — | ↓ | **偏大** |
| 定容后摇匀，液面下降再加水 | — | ↑ | **偏小** |
| 定容时超过刻度线（需重配） | — | ↑ | **偏小** |

> **仰俯视口诀**：仰大俯小——仰视则体积偏大，浓度偏小；俯视则体积偏小，浓度偏大。

## 七、典型题型与解题方法

### 7.1 基本换算型

将已知物理量转换到 $n$（物质的量），再从 $n$ 转换到目标物理量。

> 例：标况下 44.8 L CO₂ 的质量是多少？
>
> 解：$n = \dfrac{V}{V_m} = \dfrac{44.8}{22.4} = 2 \text{ mol}$
>
> $m = n \times M = 2 \times 44 = 88 \text{ g}$

### 7.2 混合体系型

> 例：28 g CO 和 N₂ 的混合气体，标况下体积是多少？
>
> 解析：$M(\text{CO}) = M(\text{N₂}) = 28 \text{ g/mol}$，二者摩尔质量相同，
> 混合气的平均摩尔质量仍为 28 g/mol。
>
> $n = \dfrac{28}{28} = 1 \text{ mol}$，$V = 1 \times 22.4 = 22.4 \text{ L}$

### 7.3 阿伏伽德罗定律应用型

> 例：同温同压下，等质量的 SO₂ 和 CO₂ 体积比是多少？
>
> 解：$\dfrac{V_1}{V_2} = \dfrac{n_1}{n_2} = \dfrac{m/M_1}{m/M_2} = \dfrac{M_2}{M_1} = \dfrac{44}{64} = \dfrac{11}{16}$

### 7.4 物质的量浓度计算型

> 例：将标准状况下 a L HCl 气体溶于 1000 g 水中，得到盐酸密度为 b g/mL，求该盐酸的物质的量浓度。
>
> 解：$n(\text{HCl}) = \dfrac{a}{22.4} \text{ mol}$
>
> $m(\text{溶液}) = \dfrac{a}{22.4} \times 36.5 + 1000 \text{ g}$
>
> $V(\text{溶液}) = \dfrac{m(\text{溶液})}{1000b} \text{ L}$
>
> $$c = \frac{n}{V} = \frac{\dfrac{a}{22.4}}{\dfrac{a \times 36.5/22.4 + 1000}{1000b}} = \frac{1000ab}{a \times 36.5 + 22400} \text{ mol/L}$$

## 八、常见误区与注意事项

| 误区 | 正确理解 |
|------|----------|
| "物质的量"可简称为"物量" | ❌ 不可，"物质的量"是整体名词 |
| $N_A = 6.02 \times 10^{23}$ | ❌ $N_A$ 数值上约为 $6.02 \times 10^{23}$，但 $N_A$ 有单位 $\text{mol}^{-1}$ |
| 标况下 $H_2O$ 的体积可用 $22.4 \text{ L/mol}$ 计算 | ❌ 标况下水是液体 |
| 气体摩尔体积 = $22.4 \text{ L/mol}$ | ❌ 仅标况下近似此值，其他条件不同 |
| 物质的量浓度公式中 $V$ 是溶剂体积 | ❌ $V$ 是**溶液**总体积 |
| 稀释计算：$c_1V_1 = c_2V_2$ 中的 $V$ 是溶质体积 | ❌ 均为溶液体积 |
| 容量瓶摇匀后液面低于刻度线属误差，需补水 | ❌ 正常现象，部分溶液附着瓶塞和瓶颈，不需补水 |

```
                             物质的量 (n) [mol]
                                   │
            ┌──────────────────────┼──────────────────────┐
            │                      │                      │
       ÷NA  │     ÷Vm (标况)       │   ×M                 │  ÷c
            ↓                      ↓                      ↓          ↓
      ┌──────────┐          ┌───────────┐          ┌──────────┐  ┌──────────┐
      │ 粒子数 N  │          │气体体积 V(L)│          │ 质量 m(g) │  │溶液体积 V(L)│
      └──────────┘          └───────────┘          └──────────┘  └──────────┘
         ×NA ↑                  ×Vm ↑                 ÷M ↑          ×c ↑
            └──────────────────────┼──────────────────────┘
                             物质的量 (n)
```

> **"见量先化n"**：遇到任何已知量，第一步先转化为物质的量 n，问题即可简化。
