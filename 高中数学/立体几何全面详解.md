# 1. 空间几何体

## 1.1 多面体

### 1.1.1 棱柱

**定义**：有两个面互相平行，其余各面都是四边形，并且每相邻两个四边形的公共边都互相平行，由这些面所围成的多面体。

**基本要素**：
- 底面：两个互相平行的面（全等多边形）
- 侧面：其余各面（平行四边形）
- 侧棱：相邻侧面的公共边
- 顶点：棱的公共交点
- 高：两个底面之间的垂直距离

**分类**：

| 分类标准 | 类型 | 特征 |
|---------|------|------|
| 按侧棱与底面关系 | 直棱柱 / 斜棱柱 | 侧棱⊥底面 vs 不⊥ |
| 按底面形状 | 三棱柱 / 四棱柱 / 五棱柱…… | |
| 特殊 | **正棱柱** | 底面为正多边形且侧棱⊥底面 |

> **特殊四棱柱的关系链**：
> $$\boxed{\{\text{四棱柱}\} \supset \{\text{平行六面体}\} \supset \{\text{直平行六面体}\} \supset \{\text{长方体}\} \supset \{\text{正方体}\}}$$

- **平行六面体**：底面是平行四边形的四棱柱（6 个面都是平行四边形）
- **直平行六面体**：侧棱⊥底面的平行六面体（侧面为矩形）
- **长方体**：底面是矩形的直平行六面体（所有面都是矩形）
- **正方体**：所有棱长相等的长方体（所有面为正方形）

**核心性质**：

| 性质 | 直棱柱 | 斜棱柱 |
|------|--------|--------|
| 侧面形状 | 矩形 | 平行四边形 |
| 侧棱与高 | 侧棱 = 高 | 侧棱 > 高 |
| 平行于底面的截面 | 与底面全等 | 与底面全等 |

**长方体对角线**：
$$\boxed{l = \sqrt{a^2 + b^2 + c^2}}$$
其中 $a, b, c$ 为从同一顶点出发的三条棱长。

**表面积**：
$$S_{\text{表}} = S_{\text{侧}} + 2S_{\text{底}}$$

**体积**：
$$\boxed{V = S_{\text{底}} \cdot h}$$
$h$ 为两底面间的垂直距离。

---

### 1.1.2 棱锥

**定义**：有一个面是多边形，其余各面都是有一个公共顶点的三角形，由这些面所围成的多面体。

**正棱锥的条件**：
1. 底面是正多边形
2. 顶点在底面的射影是底面正多边形的**中心**
3. 各侧棱相等，各侧面为全等的等腰三角形

**核心性质**：

| 性质 | 内容 |
|------|------|
| 截面性质 | 平行于底面的截面与底面**相似** |
| 面积比 | $\frac{S_{\text{截面}}}{S_{\text{底}}} = \left(\frac{h'}{h}\right)^2$ |
| 体积比 | $\frac{V_{\text{小}}}{V_{\text{大}}} = \left(\frac{h'}{h}\right)^3$ |

**正棱锥侧面积**：
$$S_{\text{侧}} = \frac{1}{2}Ch'$$
$C$ 为底面周长，$h'$ 为斜高（顶点到底面边中点距离）。

**体积**：
$$\boxed{V = \frac{1}{3}S_{\text{底}} \cdot h}$$

> **三棱锥向量体积公式**（扩展）：
> $$V = \frac{1}{6}|\vec{AB} \cdot (\vec{AC} \times \vec{AD})|$$
> 即三条棱混合积的绝对值的 $\frac{1}{6}$。

---

### 1.1.3 棱台

**定义**：用一个平行于棱锥底面的平面去截棱锥，底面与截面之间的部分。

**正棱台条件**：由正棱锥截得。

**核心性质**：
- 上、下底面是**相似多边形**
- 侧面都是**梯形**
- 各侧棱延长后**交于一点**（还原为棱锥）

**正棱台侧面积**：
$$S_{\text{侧}} = \frac{1}{2}(C + C')h'$$
$C, C'$ 为上、下底面周长，$h'$ 为斜高。

**体积**：
$$\boxed{V = \frac{1}{3}h\left(S_{\text{上}} + S_{\text{下}} + \sqrt{S_{\text{上}} \cdot S_{\text{下}}}\right)}$$

> **退化检验**：
> - $S_{\text{上}} = 0$ → $V = \frac{1}{3}hS_{\text{下}}$（棱锥公式 ✅）
> - $S_{\text{上}} = S_{\text{下}}$ → $V = hS_{\text{下}}$（棱柱公式 ✅）

## 1.2 旋转体

### 1.2.1 圆柱

| 要素 | 符号 | 关系 |
|------|------|------|
| 底面半径 | $r$ | |
| 高 | $h$ | |
| 母线 | $l$ | $l = h$ |

$$S_{\text{表}} = 2\pi r^2 + 2\pi rh = 2\pi r(r + h)$$
$$V = \pi r^2 h$$

**侧面展开图**：矩形（长 $2\pi r$，宽 $h$）

**轴截面**：矩形（长 $2r$，宽 $h$）

---

### 1.2.2 圆锥

**基本关系**：
$$\boxed{l^2 = r^2 + h^2}$$

$$S_{\text{表}} = \pi r^2 + \pi rl = \pi r(r + l)$$
$$V = \frac{1}{3}\pi r^2 h$$

**侧面展开图**：扇形
- 扇形半径 = $l$
- 扇形弧长 = $2\pi r$
- 扇形圆心角：$\theta = \frac{r}{l} \times 360°$（弧度制：$\theta = \frac{2\pi r}{l}$）

**轴截面**：等腰三角形（底 $2r$，高 $h$，腰 $l$）

---

### 1.2.3 圆台

**基本关系**：
$$\boxed{l^2 = h^2 + (R - r)^2}$$

$$S_{\text{侧}} = \pi(r + R)l$$
$$S_{\text{表}} = \pi(r^2 + R^2 + rl + Rl)$$
$$V = \frac{1}{3}\pi h(r^2 + R^2 + rR)$$

**侧面展开图**：扇环（大扇形 $-$ 小扇形）
- 圆心角：$\theta = \frac{R - r}{l} \times 360°$

---

### 1.2.4 球

$$S = 4\pi R^2 = \pi d^2$$
$$V = \frac{4}{3}\pi R^3 = \frac{1}{6}\pi d^3$$

**球冠**（球面被平面截得的一部分）：
$$S_{\text{球冠}} = 2\pi Rh \quad\text{（}h\text{为球冠高）}$$

**球缺**（球体被平面截得的一部分）：
$$V_{\text{球缺}} = \frac{1}{3}\pi h^2(3R - h) = \pi h^2\left(R - \frac{h}{3}\right)$$

**球面距离**（球面上两点间最短距离 = 大圆劣弧长）：
$$d = R \cdot \alpha$$
$\alpha$ 为两点对球心的圆心角的弧度数（$0 \leq \alpha \leq \pi$）。

---

## 1.3 斜二测画法

**规则**（记三点即可）：

1. **横不变**：平行于 $x$ 轴的线段，直观图中长度不变
2. **纵折半**：平行于 $y$ 轴的线段，直观图中长度变为一半
3. **角 45°**：$\angle x'O'y' = 45°$（或 $135°$）

**面积关系**（重要结论！）：
$$\boxed{S_{\text{直观图}} = \frac{\sqrt{2}}{4}S_{\text{原图}} \quad\Longleftrightarrow\quad S_{\text{原图}} = 2\sqrt{2}S_{\text{直观图}}}$$

> **口诀**："横不变，纵折半，45 度斜着看，面积乘上四分之根二。"

## 1.4 表面积与体积公式速查表

| 几何体 | 表面积 $S$ | 体积 $V$ | 记忆要点 |
|--------|-----------|---------|----------|
| 正方体（棱长 $a$） | $6a^2$ | $a^3$ | 全部相同 |
| 长方体（$a,b,c$） | $2(ab+bc+ca)$ | $abc$ | 面面积之和×2 |
| 棱柱 | $S_{\text{侧}}+2S_{\text{底}}$ | $S_{\text{底}}h$ | 底面积 × 高 |
| 棱锥 | $S_{\text{侧}}+S_{\text{底}}$ | $\frac{1}{3}S_{\text{底}}h$ | **除以 3** |
| 棱台 | $S_{\text{上}}+S_{\text{下}}+S_{\text{侧}}$ | $\frac{1}{3}h(S_{\text{上}}+S_{\text{下}}+\sqrt{S_{\text{上}}S_{\text{下}}})$ | 棱柱+棱锥的中间态 |
| 圆柱 | $2\pi r(r+h)$ | $\pi r^2 h$ | 底面积 × 高 |
| 圆锥 | $\pi r(r+l)$ | $\frac{1}{3}\pi r^2 h$ | **除以 3** |
| 圆台 | $\pi(r^2+R^2+rl+Rl)$ | $\frac{1}{3}\pi h(r^2+R^2+rR)$ | 圆柱+圆锥中间态 |
| 球 | $4\pi R^2$ | $\frac{4}{3}\pi R^3$ | 记住 4 和 $\frac{4}{3}$ |

## 1.5 截面问题专题（扩展）

截面问题是高考选填题的热门考点。

### 1.5.1 正方体的截面

用一个平面去截正方体，截面可能是：

| 截面形状 | 边的条数 | 条件 |
|---------|----------|------|
| 三角形 | 3 | 平面只与 3 个面相交 |
| 四边形 | 4 | 平面只与 4 个面相交 |
| 五边形 | 5 | 平面只与 5 个面相交 |
| 六边形 | 6 | 平面与全部 6 个面相交 |

> **结论**：正方体截面的边数 = 截面平面穿过的面的个数，范围为 3~6。

**特殊截面**：

| 截面 | 作法 | 形状 |
|------|------|------|
| 对角面 | 过体对角线 | 矩形 |
| 中垂面 | 垂直平分体对角线 | 正六边形 |
| 对角面 | 过相对的两棱中点 | 也可以是正六边形 |

---

### 1.5.2 球的截面

球被平面截得的面永远是**圆**。

- 截面圆半径 $r$，球心到截面距离 $d$，球半径 $R$：
$$\boxed{r^2 = R^2 - d^2}$$

> 当 $d = 0$ 时截面过球心 → **大圆**（$r = R$）；  
> 当 $d = R$ 时截面切于球 → $r = 0$（一个点）。

---

### 1.5.3 截面面积的最值

- **正方体中最大截面**：对角面，面积 $= \sqrt{2}a^2$
- **球中最大截面**：大圆，面积 $= \pi R^2$
- **圆锥中最大截面**：轴截面，面积 $= rh$

---

## 1.6 展开与折叠专题（扩展）

### 1.6.1 表面展开图

| 几何体 | 展开图形状 |
|--------|-----------|
| 正方体 | 6 个正方形（共 11 种展开图） |
| 圆柱 | 2 个圆 + 1 个矩形 |
| 圆锥 | 1 个圆 + 1 个扇形 |
| 圆台 | 2 个圆 + 1 个扇环 |
| 正三棱锥 | 1 个等边三角形 + 3 个等腰三角形 |

### 1.6.2 翻折问题三步曲

```
① 明确折线（不动线），找出翻折前后不变的量
② 画出折前折后的对比图（关键！）
③ 利用面面垂直的条件推出新的位置关系
```

**翻折前后不变的量**：
- 折线上各点到自身的距离（当然不变）
- 任意两点到折线上某点的距离**不变**
- 折线两侧的线段长度**不变**（但位置变了）

**翻折前后变化的量**：
- 空间中两点的距离（可能变）
- 异面直线的夹角（可能变）
- 点到平面的距离（可能变）

# 2. 点、线、面之间的位置关系

## 2.1 平面的基本性质

### 三大公理 + 一个平行公理

**公理 1**（直线在平面内的判定）：
一条直线上有两点在平面内 → 整条直线都在平面内。

**公理 2**（确定平面的条件）：
不共线的三点确定唯一平面。

> **三个推论**（证明共面的常用工具）：
> - 推论 1：一条直线 + 直线外一点 → 确定平面
> - 推论 2：两条相交直线 → 确定平面
> - 推论 3：两条平行直线 → 确定平面

**公理 3**（两平面相交）：
两平面有一个公共点 → 它们有过该点的一条交线。

**公理 4**（平行的传递性）：
$a \parallel b, b \parallel c \implies a \parallel c$

---

## 2.2 空间中直线与直线的位置关系

| 位置关系 | 共面？ | 公共点 |
|---------|--------|--------|
| 相交 | 共面 | 有且只有 1 个 |
| 平行 | 共面 | 无 |
| 异面 | **不共面** | 无 |

### 2.2.1 异面直线

**判定定理**：过平面外一点与平面内一点的直线，和平面内**不经过该点**的直线是异面直线。

**异面直线所成的角 $\theta$**：
- 范围：$\theta \in (0°, 90°]$
- 求法：
  1. **平移法**：将一条直线平移至与另一条相交
  2. **补形法**：将几何体补形成长方体/正方体
  3. **向量法**：$\cos\theta = \dfrac{|\vec{a} \cdot \vec{b}|}{|\vec{a}||\vec{b}|}$

> ⚠️ 向量法求异面直线角必须取**绝对值**（保证锐角/直角）。

### 2.2.2 等角定理

空间中两个角的两边分别对应平行 → 这两个角相等或互补。

---

## 2.3 直线与平面的位置关系

### 2.3.1 线面平行

**判定定理**：
> 平面**外**一直线平行于平面**内**一直线 → 该直线平行于平面。

$$a \not\subset \alpha,\ b \subset \alpha,\ a \parallel b \implies a \parallel \alpha$$

**性质定理**：
> 线平行于面，过该线的平面与已知平面相交 → 该线平行于交线。

**证明线面平行的套路**：
1. 在平面内找（或作）一条直线
2. 证明这条直线与已知直线平行（中位线/平行四边形）
3. 确认已知直线在平面外

---

### 2.3.2 线面垂直

**判定定理**：
> 一直线垂直于平面内**两条相交直线** → 该直线垂直于平面。

$$m \subset \alpha,\ n \subset \alpha,\ m \cap n = P,\ l \perp m,\ l \perp n \implies l \perp \alpha$$

**性质定理**：
1. 两条直线都垂直于同一平面 → 两直线平行
2. 线垂直于面 → 线垂直于面内任意直线

**证明线面垂直的套路**：
1. 在平面内选两条相交直线
2. 证明已知直线与两条直线都垂直（勾股定理逆定理/线面垂直链）
3. 运用判定定理

---

### 2.3.3 三垂线定理（经典工具）

> **定理**：平面内一条直线 ⊥ 斜线的射影 → 该直线 ⊥ 斜线
> **逆定理**：平面内一条直线 ⊥ 斜线 → 该直线 ⊥ 斜线的射影

**简记**："射影垂直 ↔ 斜线垂直"（平面内的那根线转起来）。

应用前提：$PA \perp \text{平面} \alpha$，$AO$ 为 $PO$ 在 $\alpha$ 内的射影。

---

### 2.3.4 直线与平面所成的角

**定义**：斜线与它在平面内的射影所成的锐角。

- 范围：$\theta \in [0°, 90°]$
- 线⊥面 → $\theta = 90°$，线∥面或线⊂面 → $\theta = 0°$

**向量公式**（注意是 $\sin$！）：
$$\boxed{\sin\theta = \frac{|\vec{s} \cdot \vec{n}|}{|\vec{s}||\vec{n}|}}$$
$\vec{s}$ 为直线方向向量，$\vec{n}$ 为平面法向量。

---

## 2.4 平面与平面的位置关系

### 2.4.1 面面平行

**判定定理**：
> 一个平面内的**两条相交直线**都平行于另一个平面 → 面面平行。

**性质定理**：
> 两平行平面同时与第三平面相交 → 两条交线平行。

---

### 2.4.2 面面垂直

**判定定理**：
> 一个平面**过**另一个平面的垂线 → 面面垂直。

$$l \perp \beta,\ l \subset \alpha \implies \alpha \perp \beta$$

**性质定理**：
> 两平面垂直，一平面内垂直于它们交线的直线 → 垂直于另一平面。

---

### 2.4.3 二面角

**平面角**：在棱上任取一点，分别在两个半平面内作棱的垂线，这两条垂线所成的角。

- 范围：$\theta \in [0°, 180°]$

**向量公式**（用法向量算）：
$$\boxed{|\cos\theta| = \frac{|\vec{n}_1 \cdot \vec{n}_2|}{|\vec{n}_1||\vec{n}_2|}}$$

> ⚠️ 法向量法算出来的是 $|\cos\theta|$，需根据图形判断二面角是锐角还是钝角。

---

## 2.5 空间距离

### 2.5.1 点到平面的距离

**等体积法**（几何法）：
$$h = \frac{3V}{S_{\text{底}}}$$

**向量公式**：
$$\boxed{d = \frac{|\vec{AP} \cdot \vec{n}|}{|\vec{n}|}}$$
$\vec{n}$ 为平面法向量，$P$ 为平面上任意一点。

---

### 2.5.2 直线到平行平面的距离

转化为直线上任一点到平面的距离。

### 2.5.3 两平行平面间的距离

转化为一平面内任一点到另一平面的距离。

### 2.5.4 异面直线的距离（扩展）

**向量公式**：
$$d = \frac{|\vec{AB} \cdot (\vec{a} \times \vec{b})|}{|\vec{a} \times \vec{b}|}$$

其中 $\vec{a}, \vec{b}$ 分别为两异面直线的方向向量，$A, B$ 为两直线上各一点。

---

## 2.6 平行与垂直关系总结

### 2.6.1 平行关系的转化链

$$\boxed{\text{线线平行} \rightleftarrows \text{线面平行} \rightleftarrows \text{面面平行}}$$

- 线线平行 → 线面平行（判定）
- 线面平行 → 线线平行（性质，需借助辅助面）
- 线面平行 → 面面平行（判定，需两条相交线）
- 面面平行 → 线面平行（性质）
- 面面平行 → 线线平行（性质，需借助辅助面）

### 2.6.2 垂直关系的转化链

$$\boxed{\text{线线垂直} \rightleftarrows \text{线面垂直} \rightleftarrows \text{面面垂直}}$$

- 线线垂直 → 线面垂直（判定，需两条相交线）
- 线面垂直 → 线线垂直（性质，任意一条）
- 线面垂直 → 面面垂直（判定）
- 面面垂直 → 线面垂直（性质，需垂直于交线）

# 3. 空间向量与立体几何

## 3.1 空间直角坐标系

**建系三要素**：原点 + 三条互相垂直的坐标轴 + 单位长度。

**右手系**：拇指→x，食指→y，中指→z。

**点的对称**（$P(x, y, z)$ 的对称点）：

| 对称关系 | 对称点坐标 |
|---------|-----------|
| 关于 $xOy$ 平面 | $(x, y, -z)$ |
| 关于 $x$ 轴 | $(x, -y, -z)$ |
| 关于原点 | $(-x, -y, -z)$ |

**两点距离**：
$$|P_1P_2| = \sqrt{(x_1-x_2)^2 + (y_1-y_2)^2 + (z_1-z_2)^2}$$

**中点**：
$$M = \left(\frac{x_1+x_2}{2}, \frac{y_1+y_2}{2}, \frac{z_1+z_2}{2}\right)$$

**三角形重心**（三个顶点的平均）：
$$G = \left(\frac{x_1+x_2+x_3}{3}, \frac{y_1+y_2+y_3}{3}, \frac{z_1+z_2+z_3}{3}\right)$$

---

## 3.2 空间向量的坐标表示与线性运算

**空间向量** $\vec{a} = (x, y, z)$ 的模：
$$|\vec{a}| = \sqrt{x^2 + y^2 + z^2}$$

**加法**：$(x_1, y_1, z_1) + (x_2, y_2, z_2) = (x_1+x_2, y_1+y_2, z_1+z_2)$

**数乘**：$\lambda(x, y, z) = (\lambda x, \lambda y, \lambda z)$

**共线条件**：$\vec{a} \parallel \vec{b} \iff \vec{a} = \lambda \vec{b} \iff \frac{x_1}{x_2} = \frac{y_1}{y_2} = \frac{z_1}{z_2}$（分母非零）

**共面条件**：$\vec{a}, \vec{b}, \vec{c}$ 共面 $\iff$ 其中一个是另外两个的线性组合，即混合积为 0。

---

## 3.3 空间向量的数量积

$$\boxed{\vec{a} \cdot \vec{b} = x_1x_2 + y_1y_2 + z_1z_2 = |\vec{a}||\vec{b}|\cos\theta}$$

**性质**：
- 垂直 ⟺ $\vec{a} \cdot \vec{b} = 0$
- $\vec{a} \cdot \vec{a} = |\vec{a}|^2$
- 交换律成立，结合律对数量乘法也成立
- **分配律**成立：$\vec{a} \cdot (\vec{b} + \vec{c}) = \vec{a} \cdot \vec{b} + \vec{a} \cdot \vec{c}$

---

## 3.4 空间向量的向量积（扩展）

$$\vec{a} \times \vec{b} = \begin{pmatrix} y_1z_2 - z_1y_2 \\ z_1x_2 - x_1z_2 \\ x_1y_2 - y_1x_2 \end{pmatrix}$$

- $|\vec{a} \times \vec{b}| = |\vec{a}||\vec{b}|\sin\theta$（等于以 $\vec{a}, \vec{b}$ 为边的平行四边形面积）
- 方向：右手法则（垂直于 $\vec{a}$ 和 $\vec{b}$ 所在平面）
- 用法：快速求法向量（用两条边向量叉乘）

---

## 3.5 向量法解题核心操作

### 3.5.1 求平面的法向量

> 两条不共线的方向向量的向量积 → 法向量。

步骤：
1. 取平面内两条不共线向量 $\vec{u}, \vec{v}$
2. 设法向量 $\vec{n} = (x, y, z)$
3. 解方程组 $\begin{cases} \vec{n} \cdot \vec{u} = 0 \\ \vec{n} \cdot \vec{v} = 0 \end{cases}$
4. 任取一组非零整数解（方便计算）

> **技巧**：方程只有两个，但有三个未知数 → 自由度为 1。任取 $z = 1$（或其他方便的值）解 $x, y$。

### 3.5.2 空间角的向量求法

| 所求角            | 公式                    | 注意                  |           |                         |     |           |     |              |     |                 |
| -------------- | --------------------- | ------------------- | --------- | ----------------------- | --- | --------- | --- | ------------ | --- | --------------- |
| 异面直线角 $\theta$ | $\cos\theta = \dfrac{ | \vec{a}\cdot\vec{b} | }{        | \vec{a}                 |     | \vec{b}   | }$  | 取**绝对值**     |     |                 |
| 线面角 $\theta$   | $\sin\theta = \dfrac{ | \vec{s}\cdot\vec{n} | }{        | \vec{s}                 |     | \vec{n}   | }$  | 用 **$\sin$** |     |                 |
| 二面角 $\varphi$  | $                     | \cos\varphi         | = \dfrac{ | \vec{n}_1\cdot\vec{n}_2 | }{  | \vec{n}_1 |     | \vec{n}_2    | }$  | $\cos$ 取绝对值后判锐钝 |

### 3.5.3 空间距离的向量求法

| 所求距离                     | 公式           |                                     |     |                      |                |
| ------------------------ | ------------ | ----------------------------------- | --- | -------------------- | -------------- |
| 点 $A$ 到平面（法向量 $\vec{n}$） | $d = \dfrac{ | \vec{AP}\cdot\vec{n}                | }{  | \vec{n}              | }$（$P$ 为面上任一点） |
| 异面直线间                    | $d = \dfrac{ | \vec{AB}\cdot(\vec{a}\times\vec{b}) | }{  | \vec{a}\times\vec{b} | }$             |

### 3.5.4 向量法证明平行与垂直

| 要证 | 向量条件 |
|------|----------|
| 线线平行 | $\vec{a} = \lambda \vec{b}$ |
| 线线垂直 | $\vec{a} \cdot \vec{b} = 0$ |
| 线面平行 | $\vec{s} \cdot \vec{n} = 0$（方向向量 ⊥ 法向量） |
| 线面垂直 | $\vec{s} \parallel \vec{n}$（方向向量 ∥ 法向量） |
| 面面平行 | $\vec{n}_1 \parallel \vec{n}_2$（两法向量平行） |
| 面面垂直 | $\vec{n}_1 \cdot \vec{n}_2 = 0$（两法向量垂直） |

## 3.6 建系原则与技巧

| 原则 | 说明 |
|------|------|
| **找三线两两垂直的交点** | 作为原点（正方体/长方体的顶点、矩形一角 + 高） |
| **正棱锥/正棱柱** | 原点在底面中心，$z$ 轴沿高方向 |
| **有等腰三角形** | 原点在底边中点 |
| **尽量使坐标简单** | 能取 $(1,0,0)$ 就不取 $(\sqrt{3},0,0)$ |

**常用建系模板**：

| 几何体 | 坐标系取法 |
|--------|-----------|
| 正方体/长方体 | 一顶点为原点，三条共点棱为 $x,y,z$ 轴 |
| 正四棱锥（底面中心为原点） | $x,y$ 轴沿底面对角线方向，$z$ 轴沿高 |
| 正三棱柱（底面中心为原点） | $z$ 轴沿高，底面在 $xOy$ 平面 |
| 底面有垂线 | 垂足为原点，垂线为 $z$ 轴 |

# 5. 二级结论大全

## 5.1 特殊几何体的常数

| 正四面体（棱长 $a$） | 值 |
|---------------------|-----|
| 高 | $h = \dfrac{\sqrt{6}}{3}a$ |
| 外接球半径 | $R = \dfrac{\sqrt{6}}{4}a$ |
| 内切球半径 | $r = \dfrac{\sqrt{6}}{12}a$ |
| 体积 | $V = \dfrac{\sqrt{2}}{12}a^3$ |
| 表面积 | $S = \sqrt{3}a^2$ |
| 对棱距离 | $\dfrac{\sqrt{2}}{2}a$ |

| 正方体（棱长 $a$） | 值 |
|--------------------|-----|
| 面对角线 | $\sqrt{2}a$ |
| 体对角线 | $\sqrt{3}a$ |
| 外接球半径 | $R = \dfrac{\sqrt{3}}{2}a$ |
| 内切球半径 | $r = \dfrac{a}{2}$ |
| 与各棱相切的球半径 | $\dfrac{\sqrt{2}}{2}a$ |

## 5.2 体积与面积关系

- **三棱锥换底**：$V_{A-BCD} = V_{B-ACD} = V_{C-ABD} = V_{D-ABC}$
- **相似比与体积比**：线度比 $k:1$ → 面积比 $k^2:1$ → 体积比 $k^3:1$
- **等底等高**的任何几何体体积相等

## 5.3 球与多面体

| 模型 | 特征 | 球心位置 |
|------|------|----------|
| 墙角模型 | 三条棱两两垂直 | 补形为长方体，体对角线中点为球心 |
| 正四面体 | 所有棱等长 | 高上的某一特殊点（距顶点 $\frac{3}{4}h$） |
| 正棱锥/正棱柱 | 有对称中心 | 对称中心即为球心 |

> **外接球问题万能思路**：找球心到各顶点的距离相等的点 → 用对称性。

## 5.4 角度与位置关系

- 异面直线所成角 → 平移后相交的角（范围 $(0°, 90°]$）
- 线面角 $\sin$ = 方向向量与法向量夹角的 $\cos$
- **余弦定理求角**：若已知三角形三边，$\cos A = \frac{b^2 + c^2 - a^2}{2bc}$

## 5.5 特殊结论

- **四面体对棱垂直的传递性**：两组对棱互相垂直 → 第三组也互相垂直
- **等腰三角形底边中线** ⊥ **底边**（纯几何求垂直的利器）
- **勾股定理逆定理**：$a^2 + b^2 = c^2$ ⟺ 直角（证明线线垂直）

## 5.6 最值模型（新增）

| 模型 | 最值点 | 结论 |
|------|--------|------|
| 球面上两点最短距离 | 大圆劣弧 | 两点 + 球心确定的平面与球面交线 |
| 点到平面最短距离 | 垂线段 | 唯一 |
| 异面直线上各取一点最小距离 | 公垂线段 | 唯一 |
| 动点到两定点距离和最小 | 反射法（展开） | 展开到同一平面后用直线段 |
| 截面面积最大 | 过球心的截面 | 大圆（半径为 $R$） |

# 6. 经典例题精讲（25 例）

## 例题 1-5：基础篇

### 例 1 · 正方体中异面直线所成角

正方体 $ABCD\text{-}A_1B_1C_1D_1$ 棱长 $1$，求 $A_1C_1$ 与 $BC$ 所成角。

**解（平移法）**：
- $A_1C_1 \parallel AC$（正方体上底面对角线平行于下底面对角线）
- 求 $AC$ 与 $BC$ 的夹角：在 $\triangle ACB$ 中，$AB \perp BC$，$\angle ACB = 45°$
- **答案**：$45°$

> 关键：通过平移把异面转化为共面。

---

### 例 2 · 证明面面垂直

正方体 $ABCD\text{-}A_1B_1C_1D_1$ 中，求证：平面 $A_1BD \perp$ 平面 $ACC_1A_1$。

**分析**：面面垂直 → 在一个面内找一条直线⊥另一个面。

**证明**：
- $BD \perp AC$（正方形对角线）
- $AA_1 \perp BD$（$AA_1 \perp$ 底面）
- ∴ $BD \perp$ 平面 $ACC_1A_1$（$BD$ ⊥ 面内两条相交直线 $AC$ 和 $AA_1$）
- 又 $BD \subset$ 平面 $A_1BD$
- ∴ 平面 $A_1BD \perp$ 平面 $ACC_1A_1$

---

### 例 3 · 等体积法求点面距

三棱锥 $P\text{-}ABC$：$PA \perp$ 底面 $ABC$，$AB \perp AC$，$PA=3, AB=4, AC=5$。求 $A$ 到平面 $PBC$ 的距离。

**解法一（等体积法）**：

$$V_{P\text{-}ABC} = \frac{1}{3} \times \frac{1}{2} \times 4 \times 5 \times 3 = 10$$

建系：$A(0,0,0), B(4,0,0), C(0,5,0), P(0,0,3)$

$\triangle PBC$ 面积用向量积：
$\overrightarrow{PB} = (4, 0, -3), \overrightarrow{PC} = (0, 5, -3)$
$S = \frac{1}{2}|(15, 12, 20)| = \frac{\sqrt{769}}{2}$

$$h = \frac{3V}{S} = \frac{30}{\sqrt{769}/2} = \frac{60}{\sqrt{769}}$$

**解法二（向量法）**：
平面 $PBC$ 法向量 $\vec{n} = (15, 12, 20)$，$|\vec{n}| = \sqrt{769}$

$d = \frac{|\vec{PA} \cdot \vec{n}|}{|\vec{n}|} = \frac{|(0,0,-3)\cdot(15,12,20)|}{\sqrt{769}} = \frac{60}{\sqrt{769}}$

两种方法一致 ✅。

### 例 4 · 外接球（墙角模型）

三棱锥 $S\text{-}ABC$：$SA \perp$ 平面 $ABC$，$AB \perp AC$，$SA=AB=2, AC=3$。求外接球表面积。

**解（补形法）**：
三条棱两两垂直 → 补形为长方体：长 $3$, 宽 $2$, 高 $2$

体对角线 = 外接球直径：
$$2R = \sqrt{3^2 + 2^2 + 2^2} = \sqrt{17} \implies R = \frac{\sqrt{17}}{2}$$

$$S = 4\pi R^2 = 4\pi \times \frac{17}{4} = 17\pi$$

> **套路**：三条棱两两垂直 → **立即补形**！

---

### 例 5 · 内切球半径

正四面体棱长 $a$，求内切球半径。

**解**：
$$V = \frac{\sqrt{2}}{12}a^3,\quad S_{\text{表}} = \sqrt{3}a^2$$

内切球半径（体积法）：$V = \frac{1}{3}S_{\text{表}} \cdot r$

$$r = \frac{3V}{S_{\text{表}}} = \frac{3 \times \frac{\sqrt{2}}{12}a^3}{\sqrt{3}a^2} = \frac{\sqrt{6}}{12}a$$

> 正四面体 $R : r = 3 : 1$（外接球半径是内切球半径的 3 倍）。

## 例题 6-10：进阶篇

### 例 6 · 翻折问题

矩形 $ABCD$，$AB=2, BC=1$，$E$ 为 $CD$ 中点。将 $\triangle ADE$ 沿 $AE$ 折起，使平面 $ADE \perp$ 平面 $ABCE$。求二面角 $D\text{-}AE\text{-}C$。

**解（抓不变 + 面面垂直性质）**：
- 折前：$AD = DE = 1$，$\angle ADE = 90°$，$AE = \sqrt{2}$
- $D$ 到 $AE$ 的距离：$DF = \frac{1 \times 1}{\sqrt{2}} = \frac{\sqrt{2}}{2}$（$F$ 为 $AE$ 中点）
- 折后：$DF \perp AE$（在面 $ADE$ 内）
- 面 $ADE \perp$ 面 $ABCE$ → $DF \perp$ 面 $ABCE$
- 在底面作 $FG \perp AE$，则 $\angle DFG$ 为二面角平面角
- $DF \perp FG$（$DF \perp$ 底面，$FG \subset$ 底面）
- **答案**：二面角 $= 90°$

### 例 7 · 探求存在性

长方体 $ABCD\text{-}A_1B_1C_1D_1$：$AB=2, AD=1, AA_1=2$。$AA_1$ 上是否存在 $P$ 使 $DP \perp$ 平面 $B_1PC$？

**解（坐标法设参数）**：
建系：$D(0,0,0), A(1,0,0), B(1,2,0), C(0,2,0), B_1(1,2,2)$

设 $P(1,0,h), h \in [0,2]$

$DP \perp PB_1$：$(1,0,h) \cdot (0,2,2-h) = h(2-h) = 0 \implies h = 0$ 或 $2$

$DP \perp PC$：$(1,0,h) \cdot (-1,2,-h) = -1 - h^2 = 0 \implies h^2 = -1$

无解！**答案**：不存在。

### 例 8 · 动点最值

正四棱柱 $ABCD\text{-}A_1B_1C_1D_1$：$AB=2, AA_1=4$，$E$ 为 $CC_1$ 中点。动点 $P$ 在侧面 $BCC_1B_1$ 内满足 $AP \perp BD_1$。求 $EP$ 与平面 $ABB_1A_1$ 所成角正弦值范围。

**解**：
建系 → 设 $P(x, 2, z)$ → $AP \perp BD_1$ 推得 $x = 2z$，$z \in [0, 1]$

$EP$ 方向向量：$(2z, 0, z-2)$，平面 $ABB_1A_1$ 法向量 $(1,0,0)$

$$\sin\theta = \frac{2z}{\sqrt{5z^2 - 4z + 4}}$$

求导得单调递增 → **答案**：$\left[0, \dfrac{2\sqrt{5}}{5}\right]$

### 例 9 · 四面体对棱垂直传递

四面体 $ABCD$ 中，$AB \perp CD$，$AC \perp BD$。求证：$AD \perp BC$。

**证明（向量法）**：
设 $\vec{AB} = \vec{b}, \vec{AC} = \vec{c}, \vec{AD} = \vec{d}$

$AB \perp CD$ → $\vec{b} \cdot (\vec{d} - \vec{c}) = 0$ → $\vec{b} \cdot \vec{d} = \vec{b} \cdot \vec{c}$

$AC \perp BD$ → $\vec{c} \cdot (\vec{d} - \vec{b}) = 0$ → $\vec{c} \cdot \vec{d} = \vec{c} \cdot \vec{b}$

得 $\vec{b} \cdot \vec{d} = \vec{b} \cdot \vec{c} = \vec{c} \cdot \vec{d}$

$AD \perp BC$：$\vec{d} \cdot (\vec{c} - \vec{b}) = \vec{d} \cdot \vec{c} - \vec{d} \cdot \vec{b} = 0$ ✅

### 例 10 · 截面面积最小值

正方体棱长 $2$，过体对角线作截面，求截面积的最小值。

**解**：
截面为平行四边形。当过体对角线的截面与另外四条棱的交点均在中点位置时，截面为菱形。计算可得最小面积为 $2\sqrt{2}$。

> 正方体过体对角线的最小截面是菱形（四个交点都是棱中点）。

**答案**：$2\sqrt{2}$

### 例 12 · 圆锥侧面展开

圆锥底面半径 $r = 3$，母线 $l = 6$。沿母线剪开展成扇形，求扇形圆心角。

**解**：
$$\theta = \frac{r}{l} \times 360° = \frac{3}{6} \times 360° = 180°$$

**答案**：$180°$（半圆）

### 例 13 · 圆柱面上最短路径

圆柱底面半径 $r$，高 $h$。从下底面边缘一点 $A$ 沿圆柱面到上底面正对面一点 $B$（$A, B$ 相差半圆），求最短路径长。

**解（展开法）**：
将圆柱侧面展开为矩形（宽 $2\pi r$，高 $h$）。$A$ 在左下角，$B$ 在顶部中点（距左边缘 $\pi r$）。

最短距离 = $\sqrt{(\pi r)^2 + h^2}$

**答案**：$\sqrt{\pi^2 r^2 + h^2}$

---

### 例 14 · 长方体表面最短路径

长方体 $3 \times 4 \times 5$。从一顶点沿表面对角顶点，求最短路径。

**解（展开到同一平面）**：
三种展开方式，分别计算距离：
- 展开 $3\times4$ 面 + $4\times5$ 面：$\sqrt{(3+5)^2 + 4^2} = \sqrt{64+16} = \sqrt{80}$
- 展开 $3\times4$ 面 + $3\times5$ 面：$\sqrt{(4+5)^2 + 3^2} = \sqrt{81+9} = \sqrt{90}$
- 展开 $4\times5$ 面 + $3\times5$ 面：$\sqrt{(3+4)^2 + 5^2} = \sqrt{49+25} = \sqrt{74}$

**取最小值**：$\sqrt{74}$

---

### 例 15 · 球的截面

球半径 $R = 5$，球心到截面距离 $d = 3$。求截面圆的面积。

**解**：
截面圆半径 $r = \sqrt{R^2 - d^2} = \sqrt{25 - 9} = 4$

面积 $= \pi r^2 = 16\pi$

**答案**：$16\pi$

## 例题 16-20：高考真题型

### 例 16 · 二面角计算（全国卷型）

四棱锥 $P\text{-}ABCD$ 底面为矩形，$PA \perp$ 底面，$PA = AB = 2, BC = 1$。求二面角 $B\text{-}PC\text{-}D$ 的余弦值。

**解（向量法）**：
建系 $A(0,0,0), B(2,0,0), C(2,1,0), D(0,1,0), P(0,0,2)$

面 $PBC$ 法向量：$\overrightarrow{PB} = (2,0,-2), \overrightarrow{PC} = (2,1,-2)$
$\vec{n}_1 = (2, 0, 2)$（化简得 $(1,0,1)$）

面 $PCD$ 法向量：$\overrightarrow{PC} = (2,1,-2), \overrightarrow{PD} = (0,1,-2)$
$\vec{n}_2 = (0, 2, 1)$

$\cos\theta = \frac{|\vec{n}_1 \cdot \vec{n}_2|}{|\vec{n}_1||\vec{n}_2|} = \frac{|1 \times 0 + 0 \times 2 + 1 \times 1|}{\sqrt{2} \cdot \sqrt{5}} = \frac{1}{\sqrt{10}} = \frac{\sqrt{10}}{10}$

**答案**：$\frac{\sqrt{10}}{10}$

---

### 例 17 · 线面角（全国卷型）

同例 16，求 $AB$ 与平面 $PCD$ 所成角的正弦值。

**解**：
$\overrightarrow{AB} = (2, 0, 0)$，平面 $PCD$ 法向量 $\vec{n} = (0, 2, 1)$

$$\sin\theta = \frac{|\overrightarrow{AB} \cdot \vec{n}|}{|\overrightarrow{AB}||\vec{n}|} = \frac{|2 \times 0 + 0 \times 2 + 0 \times 1|}{2 \cdot \sqrt{5}} = 0$$

**答案**：$0$（$AB \parallel$ 平面 $PCD$）。

---

### 例 18 · 异面直线距离（扩展题）

正方体棱长 $1$，求 $AB$ 与 $CC_1$ 的距离。

**解**：
$AB$ 方向 $(1,0,0)$，$CC_1$ 方向 $(0,0,1)$。取 $A(0,0,0), C(1,1,0)$

$\vec{a} \times \vec{b} = (1,0,0) \times (0,0,1) = (0, -1, 0)$，$|\vec{a} \times \vec{b}| = 1$

$d = \frac{|\vec{AC} \cdot (\vec{a} \times \vec{b})|}{|\vec{a} \times \vec{b}|} = \frac{|(1,1,0) \cdot (0,-1,0)|}{1} = 1$

**答案**：$1$（即正方体的棱长）。

---

### 例 19 · 球与正三棱柱（高考型）

正三棱柱底面边长 $2$，高 $3$。求其外接球的半径。

**解**：
球心在上下底面中心的连线的中点。底面正三角形外接圆半径 $r_0 = \frac{2}{\sqrt{3}}$。

$$R = \sqrt{r_0^2 + \left(\frac{h}{2}\right)^2} = \sqrt{\left(\frac{2}{\sqrt{3}}\right)^2 + \left(\frac{3}{2}\right)^2} = \sqrt{\frac{4}{3} + \frac{9}{4}} = \sqrt{\frac{16+27}{12}} = \frac{\sqrt{43}}{2\sqrt{3}}$$

**答案**：$\frac{\sqrt{129}}{6}$

---

### 例 20 · 旋转体体积（高考型）

直角三角形两直角边分别为 $3$ 和 $4$。分别以两条直角边为轴旋转一周，求两个圆锥的体积比。

**解**：
- 以 $3$ 为轴：$r=4, h=3, V_1 = \frac{1}{3}\pi \times 16 \times 3 = 16\pi$
- 以 $4$ 为轴：$r=3, h=4, V_2 = \frac{1}{3}\pi \times 9 \times 4 = 12\pi$

体积比：$V_1 : V_2 = 16\pi : 12\pi = 4 : 3$

**答案**：$4:3$

## 例题 21-25：压轴难题

### 例 21 · 正四棱锥内外接球半径比

正四棱锥底面边长 $a$，侧棱长也是 $a$。求外接球与内切球半径比。

**解**：
底面正方形对角线 $\sqrt{2}a$，半对角线 $\frac{\sqrt{2}}{2}a$

高 $h = \sqrt{a^2 - \left(\frac{\sqrt{2}}{2}a\right)^2} = \sqrt{a^2 - \frac{a^2}{2}} = \frac{\sqrt{2}}{2}a$

正四棱锥可补形为正方体（棱长为 $\frac{a}{\sqrt{2}}$）的一部分。

外接球半径 $R = \frac{\sqrt{3}}{2} \cdot \frac{a}{\sqrt{2}} = \frac{\sqrt{6}}{4}a$

内切球半径（体积法）：$V = \frac{1}{3}a^2 \cdot h = \frac{\sqrt{2}}{6}a^3$，侧面积 $S_{\text{侧}}$（4 个全等正三角形）= $4 \times \frac{\sqrt{3}}{4}a^2 = \sqrt{3}a^2$

$S_{\text{表}} = a^2 + \sqrt{3}a^2 = (1+\sqrt{3})a^2$

$$r = \frac{3V}{S_{\text{表}}} = \frac{3 \times \frac{\sqrt{2}}{6}a^3}{(1+\sqrt{3})a^2} = \frac{\sqrt{2}}{2(1+\sqrt{3})}a$$

$R : r = \frac{\sqrt{6}}{4} : \frac{\sqrt{2}}{2(1+\sqrt{3})} = \frac{\sqrt{6}}{4} \times \frac{2(1+\sqrt{3})}{\sqrt{2}} = \frac{\sqrt{3}(1+\sqrt{3})}{2} = \frac{3+\sqrt{3}}{2}$

---

### 例 22 · 折叠后体积最值

矩形纸片 $ABCD$，$AB = 6, BC = 4$。沿 $AC$ 折叠使平面 $ABC \perp$ 平面 $ACD$。求四面体 $ABCD$ 的体积。

**解**：
$AC = \sqrt{6^2 + 4^2} = \sqrt{52} = 2\sqrt{13}$

$B$ 到 $AC$ 的距离：$d_B = \frac{6 \times 4}{2\sqrt{13}} = \frac{12}{\sqrt{13}}$

$D$ 到 $AC$ 的距离：同上（对称），$d_D = \frac{12}{\sqrt{13}}$

折叠后，$B$ 和 $D$ 在垂直于 $AC$ 的两个互相垂直的平面内。

$$V = \frac{1}{3} \times \left(\frac{1}{2} \times AC \times d_B\right) \times d_D \times \sin 90° = \frac{1}{6} \times 2\sqrt{13} \times \frac{12}{\sqrt{13}} \times \frac{12}{\sqrt{13}}$$

$$= \frac{1}{6} \times 2 \times 12 \times \frac{12}{\sqrt{13}} = \frac{48}{\sqrt{13}}$$

### 例 23 · 球面最短距离

地球视为球 $R = 6400$ km。北京（北纬 $40°$，东经 $116°$）和纽约（北纬 $40°$，西经 $74°$）近似同纬度。求球面最短距离。

**解**：
纬度圈半径 $r = R\cos 40°$

两城市经度差 $= 116° + 74° = 190°$（注意东经西经方向相反）

球面最短距离走大圆（不是同纬度圈的小圆）：
$$\cos\alpha = \sin^2 40° + \cos^2 40° \times \cos 190°$$

（此处为示意，具体数值略。重点在方法：大圆劣弧用球面三角公式。）

---

### 例 24 · 三棱锥体积最大

底面 $\triangle ABC$ 中 $AB = AC = a$，$\angle BAC = 60°$。$P$ 在底面所在平面的同侧，且 $PA = PB = PC = a$。求 $V_{P\text{-}ABC}$ 的最大值。

**解**：
底面为正三角形（等腰 + $60°$ 顶角），边长 $a$。

$P$ 到底面距离最大时，$P$ 在底面中心的正上方。

底面中心到顶点距离 $= \frac{a}{\sqrt{3}}$。高 $h_{\max} = \sqrt{a^2 - \left(\frac{a}{\sqrt{3}}\right)^2} = \sqrt{\frac{2}{3}}a$

$$V_{\max} = \frac{1}{3} \times \frac{\sqrt{3}}{4}a^2 \times \sqrt{\frac{2}{3}}a = \frac{\sqrt{2}}{12}a^3$$

（恰好是正四面体体积）

---

### 例 25 · 特殊截面面积（综合）

正方体棱长 $1$，$E, F$ 分别为棱 $AA_1, CC_1$ 的中点。求截面 $BED_1F$ 的面积。

**解**：
$B(1,1,0), E(0,0,\frac{1}{2}), D_1(1,0,1), F(1,1,\frac{1}{2})$

截面为菱形。对角线：
- $BD_1 = \sqrt{1^2 + 1^2 + 1^2} = \sqrt{3}$
- $EF = \sqrt{1^2 + 1^2 + 0^2} = \sqrt{2}$

菱形面积 $= \frac{1}{2} \times BD_1 \times EF = \frac{\sqrt{6}}{2}$

**答案**：$\frac{\sqrt{6}}{2}$

# 7. 易错陷阱清单

| # | 陷阱 | 正确做法 |
|---|------|----------|
| 1 | 异面直线角取 $\cos$ 时不加绝对值 | 必须 $|\cos|$，保证锐角 |
| 2 | 线面角用 $\cos$ 公式 | 线面角用 **$\sin$** = $\frac{|\vec{s}\cdot\vec{n}|}{|\vec{s}||\vec{n}|}$ |
| 3 | 二面角直接用 $\cos$ 不判断锐钝 | 先算 $\mid\cos\mid$，再根据图形判断正负 |
| 4 | 直方图面积当频率 | 面积 = 频率；纵轴 = 频率/组距（统计相关） |
| 5 | 圆柱与圆锥侧面展开混淆 | 圆柱→矩形；圆锥→扇形 |
| 6 | 棱锥体积忘记除 3 | $V = \frac{1}{3}S_{\text{底}}h$ |
| 7 | 补形法忘记还原 | 补形为长方体后，求完球半径记得确认原几何体在长方体中的位置 |
| 8 | 斜二测面积关系记反 | $S_{\text{原}} = 2\sqrt{2} S_{\text{直观}}$ |
| 9 | 翻折问题误以为所有量不变 | 只有折线相关的长度不变，空间位置和新形成的角改变了 |
| 10 | 建系后忘记检查坐标 | 先验证关键点距离是否符合已知条件 |
