## 使用方法

### 在 Obsidian 中打开（推荐）

1. 安装 [Obsidian](https://obsidian.md/)（免费，Windows/Mac/Linux/手机全平台）
2. 克隆本仓库：

```bash
git clone https://github.com/linjie20091027-a11y/Obsidian-Vault.git
```

3. 打开 Obsidian → 点击"打开本地仓库"→ 选择克隆的文件夹
4. 效果：所有 `[[双向链接]]`、`![[嵌入]]`、Dataview 查询、Canvas 画布将完整渲染，和作者看到的完全一致

### 直接在 GitHub 上浏览

可直接点击浏览 `.md` 文件。GitHub 支持标准 Markdown（标题、加粗、列表、表格、代码块、LaTeX 公式等），但以下 Obsidian 专有语法**无法在 GitHub 上渲染**：

| 语法 | 效果 | GitHub 显示 |
|------|------|------------|
| `[[页面对应]]` | 双向链接 | 纯文字，不可点击 |
| `![[文件]]` | 嵌入其他笔记 | 不生效 |
| `> [!note]` | Callout 标注 | GitHub 部分支持 |
| Dataview 查询块 | 动态查询 | 显示源码 |
| Canvas (.canvas) | 无限画布 | 不支持 |

> 强烈建议 clone 后用 Obsidian 打开。

## 特点

- **[[双向链接]]**：知识节点互相连接，形成网络而非线性目录
- **Layer 化目录结构**：顶层为知识领域，逐层深入
- **公式支持**：LaTeX 数学公式，用 Obsidian 或 VS Code 打开均可渲染
- **持续更新**：知识库随学习进度不断完善
