# 线段树

**线段树**是维护**区间信息**的分治二叉树。每个节点代表一个区间，左右子节点各代表一半区间。

---

## 1. 基本结构

- 根节点表示区间 `[1, n]`
- 每个节点 `[l, r]` 的左子为 `[l, mid]`，右子为 `[mid+1, r]`
- 叶子节点 `l == r`
- 数组存储需 **4 倍空间**（最坏情况 `4n`）

```cpp
const int MAXN = 100005;
int tree[MAXN << 2];  // 4 倍空间
int a[MAXN];          // 原数组
```

---

## 2. 建树 O(n)

```cpp
// 宏定义加速（可选）
#define lc (u << 1)
#define rc (u << 1 | 1)
#define mid ((l + r) >> 1)

void build(int u, int l, int r) {
    if (l == r) {
        tree[u] = a[l];
        return;
    }
    build(lc, l, mid);
    build(rc, mid + 1, r);
    pushUp(u);  // tree[u] = tree[lc] + tree[rc];
}
void pushUp(int u) { tree[u] = tree[lc] + tree[rc]; }
```

---

## 3. 单点修改 O(log n)

```cpp
void update(int u, int l, int r, int pos, int val) {
    if (l == r) {
        tree[u] += val;  // 或 = val
        return;
    }
    if (pos <= mid) update(lc, l, mid, pos, val);
    else update(rc, mid + 1, r, pos, val);
    pushUp(u);
}
```

---

## 4. 区间查询 O(log n)

查询 `[ql, qr]` 的和（或最大值/min/gcd）：

```cpp
int query(int u, int l, int r, int ql, int qr) {
    if (ql <= l && r <= qr) return tree[u];  // 完全覆盖
    int res = 0;
    if (ql <= mid) res += query(lc, l, mid, ql, qr);
    if (qr > mid)  res += query(rc, mid + 1, r, ql, qr);
    return res;
}
```

---

## 5. 区间修改 + lazy tag ⭐⭐⭐

```cpp
int lazy[MAXN << 2];  // 懒标记

void pushDown(int u, int l, int r) {
    if (lazy[u]) {
        tree[lc] += lazy[u] * (mid - l + 1);
        tree[rc] += lazy[u] * (r - mid);
        lazy[lc] += lazy[u];
        lazy[rc] += lazy[u];
        lazy[u] = 0;
    }
}

void rangeAdd(int u, int l, int r, int ql, int qr, int val) {
    if (ql <= l && r <= qr) {
        tree[u] += val * (r - l + 1);
        lazy[u] += val;
        return;
    }
    pushDown(u, l, r);
    if (ql <= mid) rangeAdd(lc, l, mid, ql, qr, val);
    if (qr > mid)  rangeAdd(rc, mid + 1, r, ql, qr, val);
    pushUp(u);
}

int query(int u, int l, int r, int ql, int qr) {
    if (ql <= l && r <= qr) return tree[u];
    pushDown(u, l, r);
    int res = 0;
    if (ql <= mid) res += query(lc, l, mid, ql, qr);
    if (qr > mid)  res += query(rc, mid + 1, r, ql, qr);
    return res;
}
```

### lazy 标记原理

修改时，如果当前区间完全被 `[ql, qr]` 包含，直接更新 `tree[u]` 并打上 `lazy[u] += val`，**不再递归子节点**。下次访问该节点时再 `pushDown` 传给子节点。这叫**延迟传播**。

---

## 6. 维护不同信息的 pushUp

```cpp
// 维护区间和
void pushUp(int u) { tree[u] = tree[lc] + tree[rc]; }

// 维护区间最大值
void pushUp(int u) { tree[u] = max(tree[lc], tree[rc]); }

// 维护区间最小值
void pushUp(int u) { tree[u] = min(tree[lc], tree[rc]); }

// 维护区间 GCD
void pushUp(int u) { tree[u] = __gcd(tree[lc], tree[rc]); }
```

## 7. 动态开点线段树

当值域非常大（如 1e9）但操作次数少时，不用预分配 4n，而是**用到时 new 节点**。

```cpp
struct Node {
    int lc, rc, sum;
} tree[MAXN * 40];  // 开大，比如 40n
int root = 0, tot = 0;

void update(int& u, int l, int r, int pos, int val) {
    if (!u) u = ++tot;
    if (l == r) { tree[u].sum += val; return; }
    int mid = (l + r) >> 1;
    if (pos <= mid) update(tree[u].lc, l, mid, pos, val);
    else update(tree[u].rc, mid + 1, r, pos, val);
    tree[u].sum = tree[tree[u].lc].sum + tree[tree[u].rc].sum;
}
```

## 8. 线段树合并

将两棵线段树对应位置合并。复杂度 O(重叠节点数)。

```cpp
int merge(int u, int v, int l, int r) {
    if (!u || !v) return u + v;  // 其中一个为空，返回非空
    if (l == r) {
        tree[u].sum += tree[v].sum;
        return u;
    }
    int mid = (l + r) >> 1;
    tree[u].lc = merge(tree[u].lc, tree[v].lc, l, mid);
    tree[u].rc = merge(tree[u].rc, tree[v].rc, mid + 1, r);
    pushUp(u);  // 合并后记得更新
    return u;
}
```

常用于树上问题：每个节点挂一棵线段树，自底向上合并（如统计子树颜色数）。

## 9. 经典应用

| 问题 | 做法 |
|------|------|
| 逆序对 | 线段树（比 BIT 稍慢但同样可以做） |
| 区间加 & 区间和 | lazy tag |
| 区间乘法 & 区间加法 & 区间和 | 双 lazy（加懒标记和乘懒标记） |
| 区间最值 | 修改 pushUp 即可 |
| 扫描线求矩形面积 | 线段树维护扫描线 |
| 权值线段树求第 k 小 | 线段树上二分 |

## ⚠️ 易错点

1. **4 倍空间不是 2 倍**：`MAXN << 2` 而非 `MAXN << 1`。极端情况会越界。
2. **lazy 要清除**：`pushDown` 后 `lazy[u] = 0`，否则重复传播。
3. **mid 宏的括号**：`#define mid ((l+r)>>1)` 必须加括号，防止运算符优先级错误。
4. **区间修改时完全覆盖即返回**：不用再递归子节点，这是 lazy 的核心。
5. **pushUp 时机**：建树、区间修改的递归返回后都要 `pushUp`。
6. **查询先 pushDown**：查询一个区间前要先把当前节点的 lazy 推下去。
