# 并查集

**并查集（DSU / Union-Find）** 用于维护**不相交集合**的合并与查询。核心：每个集合用一棵树表示，根节点代表集合。

---

## 1. 基础实现

```cpp
int fa[MAXN];

void init(int n) {
    for (int i = 1; i <= n; i++) fa[i] = i;
}

// 查找（含路径压缩）
int find(int x) {
    return fa[x] == x ? x : fa[x] = find(fa[x]);
}

// 合并（按大小合并）
int sz[MAXN];  // 初始化全为 1
void unite(int x, int y) {
    x = find(x), y = find(y);
    if (x == y) return;
    if (sz[x] < sz[y]) swap(x, y);
    fa[y] = x;
    sz[x] += sz[y];
}
```

### 复杂度分析

仅路径压缩：均摊 O(log n)。路径压缩 + 按秩/按大小合并：均摊 **O(α(n))**，α 是阿克曼函数的反函数，实际中 α(n) ≤ 5。

## 2. 带权并查集

维护节点到根的**距离**或**关系**。经典：判断两个节点关系（同类/不同类）。

```cpp
int fa[MAXN], d[MAXN];  // d[x] = x 到 fa[x] 的距离（或关系值）

int find(int x) {
    if (fa[x] == x) return x;
    int root = find(fa[x]);
    d[x] += d[fa[x]];   // 路径压缩时维护距离
    return fa[x] = root;
}

// 合并时设定关系
void unite(int x, int y, int relation) {
    // relation: 从 x 到 y 的关系值
    int rx = find(x), ry = find(y);
    if (rx == ry) return;
    fa[rx] = ry;
    d[rx] = d[y] - d[x] + relation;  // 向量计算
}
```

### 向量理解法

画箭头：x → rx（距离 d[x]），y → ry（距离 d[y]），需要建立 rx → ry 的距离。由三角关系得出：

```
d[rx] = d[y] + relation - d[x]
```

## 3. 扩展域并查集（种类并查集）

将每个节点拆分成多个"域"，代表不同状态/种类。

**经典题：关押罪犯 (NOIP 2010)**

> 罪犯 i 和 j 不能在同一监狱。判断能否分配使得冲突值不超过某阈值。

将 `1..n` 表示"和自己在同一监狱"，`n+1..2n` 表示"和自己在不同监狱"。

```cpp
int fa[MAXN * 2];  // 二倍空间

void unite(int x, int y) { fa[find(x)] = find(y); }

// 若 a 和 b 不能在同一组（互为敌人）
unite(a, b + n);    // a 的朋友域 连 b 的敌人域
unite(b, a + n);    // b 的朋友域 连 a 的敌人域

// 判断冲突
if (find(a) == find(b)) /* a 和 b 被迫同组 → 冲突 */;
```

### 扩展域模板（二域版）

```cpp
int find(int x) { return fa[x] == x ? x : fa[x] = find(fa[x]); }

void merge(int x, int y) { fa[find(x)] = find(y); }

// 声明 x 和 y 是朋友（同域）
void setFriend(int x, int y) {
    merge(x, y);
    merge(x + n, y + n);
}

// 声明 x 和 y 是敌人（异域）
void setEnemy(int x, int y) {
    merge(x, y + n);
    merge(y, x + n);
}
```

## 4. 经典应用

| 应用 | 做法 |
|------|------|
| 连通性判断 | 合并不在同一集合的点，查询 find(x)==find(y) |
| Kruskal 最小生成树 | 从小到大加边，判断两端是否已连通（见下） |
| 食物链 (NOI 2001) | 三域扩展并查集 / 带权并查集（模 3 关系） |
| 关押罪犯 (NOIP 2010) | 二域扩展 + 二分答案 |

### Kruskal 最简模板

```cpp
struct Edge { int u, v, w; };
bool cmp(Edge a, Edge b) { return a.w < b.w; }

int kruskal(int n, vector<Edge>& edges) {
    sort(edges.begin(), edges.end(), cmp);
    init(n);
    int ans = 0, cnt = 0;
    for (auto& e : edges) {
        if (find(e.u) != find(e.v)) {
            unite(e.u, e.v);
            ans += e.w;
            cnt++;
        }
        if (cnt == n - 1) break;
    }
    return ans;  // cnt < n - 1 则图不连通
}
```

## 5. 食物链 (NOI 2001) — 带权并查集经典

三类动物 A/B/C 循环吃。用 `d[x] % 3` 表示 x 和 fa[x] 的关系：0=同类, 1=吃父, 2=被父吃。

```cpp
int find(int x) {
    if (fa[x] == x) return x;
    int root = find(fa[x]);
    d[x] = (d[x] + d[fa[x]]) % 3;
    return fa[x] = root;
}

bool check(int op, int x, int y) {
    if (x > n || y > n) return false;   // 假话规则 2
    int rx = find(x), ry = find(y);
    if (op == 1) {  // x 和 y 是同类
        if (rx == ry) return d[x] == d[y];
        fa[rx] = ry;
        d[rx] = (d[y] - d[x] + 3) % 3;  // 关系传递
    } else {         // x 吃 y
        if (rx == ry) return (d[x] - d[y] + 3) % 3 == 1;
        fa[rx] = ry;
        d[rx] = (d[y] - d[x] + 1 + 3) % 3;
    }
    return true;
}
```

---

## ⚠️ 易错点

1. **find 中忘记 `d[fa[x]]` 之前的更新**：必须先递归更新父节点再计算。
2. **带权并查集的关系方向**：`d[x]` 表示 x → fa[x]，合并时要画向量图确认公式。
3. **扩展域数组大小**：k 类就是 `k * n` 大小，遗漏会越界。
4. **按秩合并与路径压缩同时用时**：`fa[find(x)]` 而不是 `fa[x]`。
5. **Kruskal 边排序前未读入边**：基本错误但常见。
