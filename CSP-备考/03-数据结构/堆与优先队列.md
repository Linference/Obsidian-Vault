# 堆与优先队列

**堆**是一棵**完全二叉树**，满足：每个父节点的值 ≤/≥ 子节点的值。小根堆父节点最小，大根堆父节点最大。

---

## 1. 堆的数组存储（1-indexed）

- `i` 的左儿子：`2i`
- `i` 的右儿子：`2i + 1`
- `i` 的父节点：`i / 2`
- 根节点：`1`

**完全二叉树**确保数组紧凑无空洞。

---

## 2. 堆的基本操作

### 插入（上浮 push / O(log n)）

新元素放在数组末尾，然后**向上调整**（与父节点比较、交换）。

```cpp
int heap[MAXN], sz = 0;  // 小根堆

void push(int x) {
    heap[++sz] = x;
    int i = sz;
    while (i > 1 && heap[i] < heap[i / 2]) {
        swap(heap[i], heap[i / 2]);
        i /= 2;
    }
}
```

### 删除堆顶（下沉 pop / O(log n)）

用最后一个元素替换堆顶，然后**向下调整**（与较小/较大的子节点交换）。

```cpp
void pop() {
    heap[1] = heap[sz--];
    int i = 1;
    while (i * 2 <= sz) {
        int son = i * 2;
        // 小根堆：选较小的儿子
        if (son + 1 <= sz && heap[son + 1] < heap[son]) son++;
        if (heap[i] <= heap[son]) break;
        swap(heap[i], heap[son]);
        i = son;
    }
}
```

### 建堆 O(n)

```cpp
// 从最后一个非叶子节点（sz/2）开始逐个下沉
void build() {
    for (int i = sz / 2; i >= 1; i--) {
        int p = i;
        while (p * 2 <= sz) {
            int son = p * 2;
            if (son + 1 <= sz && heap[son + 1] < heap[son]) son++;
            if (heap[p] <= heap[son]) break;
            swap(heap[p], heap[son]);
            p = son;
        }
    }
}
```

## 3. STL priority_queue

```cpp
#include <queue>

// 默认大根堆
priority_queue<int> pq1;            // 队首最大
priority_queue<int, vector<int>, less<int>> pq2;  // 等价

// 小根堆
priority_queue<int, vector<int>, greater<int>> pq_min;

pq.push(x);  pq.pop();  pq.top();  pq.empty();  pq.size();
```

### 自定义比较（结构体）

```cpp
struct Node {
    int val, idx;
};
struct Cmp {
    bool operator()(const Node& a, const Node& b) {
        return a.val > b.val;  // 注意：> 表示小顶堆
    }
};
priority_queue<Node, vector<Node>, Cmp> pq;
```

## 4. 堆排序 O(n log n)

```cpp
void heapSort(int a[], int n) {
    // 把数组塞进堆
    sz = n;
    for (int i = 1; i <= n; i++) heap[i] = a[i];
    // 建堆 O(n)
    for (int i = n / 2; i >= 1; i--) /* 下沉 */;
    // 依次取堆顶 O(n log n)
    for (int i = 1; i <= n; i++) {
        a[i] = heap[1];
        pop();  // O(log n)
    }
}
```

## 5. 对顶堆求中位数 ⭐

维护一个大根堆（左半）和一个小根堆（右半），保证 `left.size() == right.size() 或 left.size() == right.size() + 1`。

中位数 = 大根堆堆顶。

```cpp
priority_queue<int> left;                             // 大根堆
priority_queue<int, vector<int>, greater<int>> right; // 小根堆

void addNum(int x) {
    if (left.empty() || x <= left.top())
        left.push(x);
    else
        right.push(x);
    // 平衡
    if (left.size() > right.size() + 1) {
        right.push(left.top()); left.pop();
    } else if (right.size() > left.size()) {
        left.push(right.top()); right.pop();
    }
}

double findMedian() {
    if (left.size() == right.size())
        return (left.top() + right.top()) / 2.0;
    return left.top();
}
```

## 6. 真题：合并果子（NOIP 2004）

> 有 n 堆果子，每次合并两堆消耗体力 = 两堆重量之和。求最小总消耗。

本质是**哈夫曼树**：每次取最小的两堆合并，用小根堆维护。

```cpp
int minCost(vector<int>& a) {
    priority_queue<int, vector<int>, greater<int>> pq;
    for (int x : a) pq.push(x);
    int ans = 0;
    while (pq.size() > 1) {
        int a = pq.top(); pq.pop();
        int b = pq.top(); pq.pop();
        ans += a + b;
        pq.push(a + b);
    }
    return ans;
}
```

复杂度 `O(n log n)`。每次合并都从堆中取出最小的两堆即可保证最优。

---

## ⚠️ 易错点

1. **STL priority_queue 默认大根堆**：需要小根堆必须加 `greater<int>`。
2. **自定义比较函数的方向**：`return a.val > b.val` 表示小的优先（反直觉！因为 priority_queue 用 `<` 比较，你要反过来理解）。
3. **对顶堆平衡条件**：左右数量差不能超过 1，否则中位数取错。
4. **堆排序不是稳定排序**。
5. **建堆 O(n) 从 sz/2 开始**：叶子节点不需要下沉。
