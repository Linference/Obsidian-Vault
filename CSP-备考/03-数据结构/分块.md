# 分块

**分块** 是暴力 + 预处理的思想：将序列分成 `sqrt(n)` 个块，整块打标记，零碎暴力。时间复杂度 `O(sqrt(n))` 每次操作。

---

## 1. 分块基本框架

```cpp
int n, block_size, block_cnt;
int a[MAXN];                 // 原数组
int belong[MAXN];            // belong[i] = i 属于哪个块
int sum[MAXB];               // 每块的和
int lazy[MAXB];              // 每块的懒标记（区间加）

void init() {
    block_size = sqrt(n);
    block_cnt = (n + block_size - 1) / block_size;  // 向上取整
    for (int i = 1; i <= n; i++) {
        belong[i] = (i - 1) / block_size + 1;
        sum[belong[i]] += a[i];
    }
}
```

## 2. 区间加法

```cpp
void rangeAdd(int l, int r, int val) {
    int L = belong[l], R = belong[r];
    if (L == R) {
        // 同一块内，暴力修改
        for (int i = l; i <= r; i++) {
            a[i] += val;
            sum[L] += val;
        }
        return;
    }
    // 左边零散部分
    for (int i = l; belong[i] == L; i++) {
        a[i] += val;
        sum[L] += val;
    }
    // 右边零散部分
    for (int i = r; belong[i] == R; i--) {
        a[i] += val;
        sum[R] += val;
    }
    // 中间整块打标记
    for (int i = L + 1; i < R; i++) {
        lazy[i] += val;
        sum[i] += val * block_size;
    }
}
```

## 3. 区间求和

```cpp
int rangeQuery(int l, int r) {
    int L = belong[l], R = belong[r], ans = 0;
    if (L == R) {
        for (int i = l; i <= r; i++)
            ans += a[i] + lazy[L];
        return ans;
    }
    // 左边零散
    for (int i = l; belong[i] == L; i++)
        ans += a[i] + lazy[L];
    // 右边零散
    for (int i = r; belong[i] == R; i--)
        ans += a[i] + lazy[R];
    // 中间整块
    for (int i = L + 1; i < R; i++)
        ans += sum[i];
    return ans;
}
```

时间复杂度：零散部分 O(sqrt(n))，整块 O(sqrt(n))，总 O(sqrt(n))。

## 4. 分块 vs 线段树

| 维度 | 分块 | 线段树 |
|------|------|--------|
| 代码复杂度 | 低（纯暴力 + 标记） | 高（递归 + lazy） |
| 时间复杂度 | O(sqrt(n)) | O(log n) |
| 空间复杂度 | O(n) | O(4n) |
| 灵活性 | 极高，任何操作都能套 | 需满足区间可合并 |
| 常数 | 小（sqrt n 不大时） | 中等 |

分块的优势在于**灵活**——对于无法用线段树维护的复杂操作（如区间排序、区间众数），分块更容易实现。

## 5. 莫队算法

莫队是**离线**处理区间询问的算法。将询问排序后，移动左右指针，O(n sqrt(n))。

### 适用范围

区间询问 `[l, r]` 已知答案后，可以 O(1) 扩展到 `[l-1, r]` / `[l+1, r]` / `[l, r-1]` / `[l, r+1]`。

### 排序方法

```cpp
struct Query {
    int l, r, id;
} q[MAXN];

bool cmp(Query a, Query b) {
    if (belong[a.l] != belong[b.l])
        return belong[a.l] < belong[b.l];
    // 奇偶优化：奇数块 r 降序，偶数块 r 升序
    return (belong[a.l] & 1) ? a.r < b.r : a.r > b.r;
}
```

### 莫队模板

```cpp
int cnt[MAXV];    // 值域计数数组
int curAns = 0;   // 当前答案

void add(int pos) {
    int x = a[pos];
    curAns += /* 加入 x 的贡献，根据题目计算 */;
    cnt[x]++;
}

void remove(int pos) {
    int x = a[pos];
    cnt[x]--;
    curAns -= /* 删除 x 的贡献 */;
}

void moAlgorithm() {
    sort(q + 1, q + 1 + m, cmp);
    int l = 1, r = 0;
    for (int i = 1; i <= m; i++) {
        while (l > q[i].l) add(--l);
        while (r < q[i].r) add(++r);
        while (l < q[i].l) remove(l++);
        while (r > q[i].r) remove(r--);
        ans[q[i].id] = curAns;
    }
}
```

### 复杂度分析

l 每次移动 O(sqrt(n))，总移动 O(m * sqrt(n))。r 在同一块内单调移动，总 O(n * sqrt(n))。

## 6. 经典应用

| 问题 | 做法 |
|------|------|
| 区间加 + 区间求和 | 基本分块 |
| 区间加 + 区间第 k 小 | 块内排序 + 二分（O(sqrt(n) log n)） |
| 区间众数 | 莫队（O(n sqrt(n))） |
| 区间不同颜色数 | 莫队（O(n sqrt(n))） |
| 带修改的区间询问 | 带修莫队（O(n^(5/3))） |
| 区间开方（每次开方后值减小） | 分块维护区间最大值，全为 1 时跳过 |

## 8. 带修莫队简述

在普通莫队基础上加一个**时间维度 t**。询问 `(l, r, t)` 表示在前 t 个修改后的状态。

块大小取 `n^(2/3)`，复杂度 O(n^(5/3))。

```cpp
struct Modify { int pos, oldVal, newVal; } mod[MAXN];
// 时间指针移动时应用/撤销修改
void apply(int t, int l, int r) {
    if (l <= mod[t].pos && mod[t].pos <= r) {
        remove(mod[t].pos);
        a[mod[t].pos] = mod[t].newVal;
        add(mod[t].pos);
    } else {
        a[mod[t].pos] = mod[t].newVal;
    }
}
```

## 易错点

1. **block_size 取值**：`sqrt(n)` 是基础值，莫队中可以调参（如 `n / sqrt(m)`）。
2. **belong 的计算**：`(i - 1) / block_size + 1`，注意 1-indexed。
3. **零散部分处理**：块内暴力时必须加上 `lazy[belong[i]]`。
4. **莫队的 add/remove 顺序**：先 add 再 remove，或保持统一方向。
5. **奇偶排序优化**记得加，能快 30%+。
