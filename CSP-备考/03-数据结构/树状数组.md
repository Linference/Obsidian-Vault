# 树状数组

**树状数组 (Fenwick Tree / BIT)** 维护**前缀和**的数据结构，支持单点修改 O(log n)、前缀查询 O(log n)。常数极小，代码极短。

---

## 1. 核心：lowbit

`lowbit(x)` 返回 x 二进制中最低位的 1 所代表的值。

```
lowbit(6) = lowbit(110₂) = 10₂ = 2
lowbit(8) = lowbit(1000₂) = 8
```

```cpp
int lowbit(int x) { return x & -x; }
```

### 树状数组的结构

`tree[x]` 维护区间 `[x - lowbit(x) + 1, x]` 的和。每个位置覆盖的区间长度恰好为 `lowbit(x)`。

---

## 2. 单点修改 + 前缀和查询

```cpp
int tree[MAXN], n;

// pos 位置增加 delta
void add(int pos, int delta) {
    while (pos <= n) {
        tree[pos] += delta;
        pos += lowbit(pos);  // 向上跳
    }
}

// 查询前缀和 [1..pos]
int query(int pos) {
    int sum = 0;
    while (pos > 0) {
        sum += tree[pos];
        pos -= lowbit(pos);  // 向下跳
    }
    return sum;
}

// 区间和 [l, r]
int rangeQuery(int l, int r) {
    return query(r) - query(l - 1);
}
```

复杂度：修改和查询均为 O(log n)。

---

## 3. 区间修改 + 单点查询

用**差分**思想。`tree[i]` 维护差分数组 `d[i] = a[i] - a[i-1]` 的前缀和。

- 区间 [l, r] 加 v：`add(l, v); add(r+1, -v);`
- 单点查询 a[x]：`query(x)`（即差分数组前缀和）

```cpp
void rangeAdd(int l, int r, int v) {
    add(l, v);
    add(r + 1, -v);
}
int pointQuery(int x) {
    return query(x);
}
```

---

## 4. 区间修改 + 区间查询

需要维护**两个 BIT**。推导：

区间 [1, n] 的和 = Σ a[i] = Σ Σ d[j] (j=1..i) = n·Σ d[i] - Σ (i-1)d[i]

这个推导比较数学，可以直接记模板：

```cpp
// tree1 维护 d[i]        tree2 维护 d[i] * (i - 1)
void rangeAddTwo(int l, int r, int v) {
    add(tree1, l, v);     add(tree1, r + 1, -v);
    add(tree2, l, v * (l - 1));  add(tree2, r + 1, -v * r);
}

int prefixSum(int x) {
    return query(tree1, x) * x - query(tree2, x);
}

int rangeSum(int l, int r) {
    return prefixSum(r) - prefixSum(l - 1);
}
```

## 5. 求逆序对

```cpp
// a 是原数组，值域需离散化
int reversePairs(vector<int>& a) {
    // 1. 离散化
    vector<int> b = a;
    sort(b.begin(), b.end());
    b.erase(unique(b.begin(), b.end()), b.end());
    // 2. 倒序扫描
    int ans = 0;
    for (int i = a.size() - 1; i >= 0; i--) {
        int idx = lower_bound(b.begin(), b.end(), a[i]) - b.begin() + 1;
        ans += query(idx - 1);  // 比 a[i] 小且出现在后面的个数
        add(idx, 1);
    }
    return ans;
}
```

## 6. 二维 BIT

```cpp
int tree[N][M];  // N, M 是行列最大值

void add(int x, int y, int delta) {
    for (int i = x; i <= n; i += lowbit(i))
        for (int j = y; j <= m; j += lowbit(j))
            tree[i][j] += delta;
}

int query(int x, int y) {
    int sum = 0;
    for (int i = x; i > 0; i -= lowbit(i))
        for (int j = y; j > 0; j -= lowbit(j))
            sum += tree[i][j];
    return sum;
}

// 二维区域和 [(x1,y1), (x2,y2)]
int rangeQuery(int x1, int y1, int x2, int y2) {
    return query(x2, y2) - query(x1-1, y2) - query(x2, y1-1) + query(x1-1, y1-1);
}
```

## 7. BIT vs 线段树

| 维度 | 树状数组 | 线段树 |
|------|----------|--------|
| 代码长度 | 极短 | 较长 |
| 常数因子 | 极小 | 较大 |
| 维护信息 | 前缀和 / 前缀最大值 | 任意区间合并信息 |
| 区间修改 | 差分技巧（有限制） | lazy tag（通用） |
| 适用场景 | 前缀类问题 | 复杂区间操作 |

**铁律**：能 BIT 不用线段树。BIT 解决不了（如区间 gcd、区间众数）再上线段树。

---

## ⚠️ 易错点

1. **下标从 1 开始**：BIT 的 `pos` 必须 ≥ 1，0 会死循环（`lowbit(0)=0`）。
2. **离散化后下标 +1**：确保从 1 开始。
3. **区间修改后的 query 偏移**：`add(r+1, -v)`，r+1 可能超出 n，数组要开大。
4. **求逆序对方向**：正序扫描是求顺序对（前面比自己小的个数），倒序扫描才是逆序对。
5. **树状数组不能维护区间最大值（减操作不支持）**：因为 max 没有逆运算。要维护用线段树。
