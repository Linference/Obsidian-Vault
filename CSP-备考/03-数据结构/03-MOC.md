# 03-数据结构 MOC

CSP-J/S 数据结构知识体系导航。数据结构 = **存储方式 + 操作方式 + 算法配合**。

## 复习路径建议

```
线性表 → 栈与队列 → 哈希表 → 堆 → 树基础
                                      ↓
并查集 ← 树状数组 ← 线段树 ← 分块 ← 树进阶（LCA/直径）
    ↓
平衡树 + 单调栈/队列（综合提高）
```

---

## J 级必会（普及组）

| 笔记 | 难度 | 核心考点 |
|------|------|----------|
| [[线性表]] ⭐ | J+S | 顺序表、单向/双向链表、链表反转、快慢指针 |
| [[栈与队列]] ⭐⭐ | J+S | 括号匹配、表达式求值、BFS、单调栈/队列入门 |
| [[堆与优先队列]] ⭐ | J+S | priority_queue、堆排序、合并果子 |
| [[哈希表]] ⭐ | J+S | 字符串哈希、unordered_map、冲突处理 |
| [[树基础]] ⭐⭐ | J+S | 二叉树遍历（前中后层序）、BST、树的存储 |

## S 级进阶（提高组）

| 笔记 | 难度 | 核心考点 |
|------|------|----------|

| [[平衡树]] | S | Treap、Splay 入门、STL set/multiset、pb_ds |
| [[单调栈与单调队列]] | S | 滑动窗口最大值、next greater element |
| [[分块]] | S | 分块思想、莫队算法 |

## 难度速查

| 标记 | 含义 |
|------|------|
| J | 普及组难度，入门必学 |
| J+S | 普及到提高过渡，两阶段都会考 |
| S | 提高组难度，含省选入门 |

## 数据结构选型速查

| 需求 | 选什么 | 时间复杂度 |
|------|--------|-----------|
| 前缀和查询 + 单点修改 | 树状数组 | O(log n) |
| 区间最值/和 + 区间修改 | 线段树 + lazy | O(log n) |
| 滑动窗口最值 | 单调队列 | O(n) |
| 集合合并 + 连通性 | 并查集 | 近似 O(1) |
| 中位数动态维护 | 对顶堆 | O(log n) |
| 有序序列动态插入删除 | 平衡树 / set | O(log n) |
| 子串快速判重 | 字符串哈希 | O(1) 查询 |
| 区间众数（离线） | 莫队 | O(n√n) |
| 最近公共祖先 | 倍增法 | O(n log n) 预处理 |
