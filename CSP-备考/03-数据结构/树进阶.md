# 树进阶

## 1. 二叉搜索树 (BST)

**性质**：左子树所有值 < 根 < 右子树所有值。中序遍历得到有序序列。

### 基本操作

```cpp
struct BSTNode {
    int val;
    BSTNode *left, *right;
    BSTNode(int v) : val(v), left(nullptr), right(nullptr) {}
};

// 查找 O(h)
BSTNode* search(BSTNode* root, int key) {
    if (!root || root->val == key) return root;
    return key < root->val ? search(root->left, key) : search(root->right, key);
}

// 插入 O(h)
BSTNode* insert(BSTNode* root, int val) {
    if (!root) return new BSTNode(val);
    if (val < root->val) root->left = insert(root->left, val);
    else root->right = insert(root->right, val);
    return root;
}
```

> ⚠️ BST 可能退化成链（如插入 1,2,3,4），此时查找 O(n)。解法 → [[平衡树]]。

## 2. AVL 树（平衡二叉搜索树）

**平衡因子** = 左子树高度 - 右子树高度。AVL 要求 |平衡因子| ≤ 1。

四种旋转情况（祖孙三代 z-y-x 路径）：

| 情况 | 形状 | 操作 |
|------|------|------|
| LL | z.left = y, y.left = x | 右旋 z |
| RR | z.right = y, y.right = x | 左旋 z |
| LR | z.left = y, y.right = x | 先左旋 y，再右旋 z |
| RL | z.right = y, y.left = x | 先右旋 y，再左旋 z |

```cpp
int height(AVLNode* n) { return n ? n->h : 0; }
int balance(AVLNode* n) { return height(n->left) - height(n->right); }

AVLNode* rotateRight(AVLNode* y) {
    AVLNode* x = y->left;
    AVLNode* T = x->right;
    x->right = y;
    y->left = T;
    y->h = max(height(y->left), height(y->right)) + 1;
    x->h = max(height(x->left), height(x->right)) + 1;
    return x;  // 新根
}

AVLNode* rotateLeft(AVLNode* x) {
    AVLNode* y = x->right;
    AVLNode* T = y->left;
    y->left = x;
    x->right = T;
    x->h = max(height(x->left), height(x->right)) + 1;
    y->h = max(height(y->left), height(y->right)) + 1;
    return y;
}

AVLNode* insert(AVLNode* root, int val) {
    if (!root) return new AVLNode(val);
    if (val < root->val) root->left = insert(root->left, val);
    else root->right = insert(root->right, val);
    root->h = max(height(root->left), height(root->right)) + 1;
    int bal = balance(root);
    // LL
    if (bal > 1 && val < root->left->val) return rotateRight(root);
    // RR
    if (bal < -1 && val > root->right->val) return rotateLeft(root);
    // LR
    if (bal > 1 && val > root->left->val) {
        root->left = rotateLeft(root->left);
        return rotateRight(root);
    }
    // RL
    if (bal < -1 && val < root->right->val) {
        root->right = rotateRight(root->right);
        return rotateLeft(root);
    }
    return root;
}
```

> CSP 考场一般不会要求手写 AVL，理解旋转思想即可。实际用 **STL set** 或 [[平衡树]] 中的 Treap。

## 3. 最近公共祖先 (LCA)

### 3.1 倍增法（最常用）O(n log n) 预处理，O(log n) 查询

```cpp
const int LOG = 20;
int fa[MAXN][LOG], depth[MAXN];
vector<int> g[MAXN];

void dfs(int u, int p) {
    fa[u][0] = p;
    depth[u] = depth[p] + 1;
    for (int k = 1; k < LOG; k++)
        fa[u][k] = fa[fa[u][k-1]][k-1];
    for (int v : g[u])
        if (v != p) dfs(v, u);
}

int lca(int u, int v) {
    if (depth[u] < depth[v]) swap(u, v);
    // 提到同一深度
    for (int k = LOG - 1; k >= 0; k--)
        if (depth[fa[u][k]] >= depth[v])
            u = fa[u][k];
    if (u == v) return u;
    // 一起向上跳
    for (int k = LOG - 1; k >= 0; k--)
        if (fa[u][k] != fa[v][k])
            u = fa[u][k], v = fa[v][k];
    return fa[u][0];
}
```

### 3.2 Tarjan 离线 LCA（并查集）

把所有询问离线保存，DFS 时用并查集合并子树。**O(n + q)**，但不能在线。

### 3.3 树剖求 LCA

见树链剖分（超出 CSP 范围但思想相通：重链划分后跳链头）。

## 4. 树的直径

最长简单路径。

- **两次 BFS/DFS**：从任意点出发找最远点 u，再从 u 找最远点 v，u-v 即为直径。适用于边权非负。
- **树形 DP**：`d1[u]` 和 `d2[u]` 分别记录 u 向下最长和次长路径长度。直径 = max(d1[u] + d2[u])。

```cpp
int d1[MAXN], d2[MAXN], diameter;

void dfs(int u, int p) {
    for (int v : g[u]) {
        if (v == p) continue;
        dfs(v, u);
        int d = d1[v] + 1;  // +1 是边权（带权则 +w）
        if (d > d1[u]) { d2[u] = d1[u]; d1[u] = d; }
        else if (d > d2[u]) { d2[u] = d; }
    }
    diameter = max(diameter, d1[u] + d2[u]);
}
```

## 5. 树的重心

删去重心后，剩余连通块大小的最大值最小。

```cpp
int sz[MAXN], centroid, minMaxPart = INF;

void dfs(int u, int p) {
    sz[u] = 1;
    int maxPart = 0;
    for (int v : g[u]) {
        if (v == p) continue;
        dfs(v, u);
        sz[u] += sz[v];
        maxPart = max(maxPart, sz[v]);
    }
    maxPart = max(maxPart, n - sz[u]);  // 父节点方向的连通块
    if (maxPart < minMaxPart) {
        minMaxPart = maxPart;
        centroid = u;
    }
}
```

## 6. 树上差分

类比一维差分，树上差分用于**区间修改 + 单点查询**。

### 点差分

对 u→v 路径上所有节点 +1：
```cpp
diff[u]++; diff[v]++; diff[lca]--; diff[fa[lca][0]]--;
```
DFS 子节点累加到父节点得到最终值。

### 边差分

将边权对应到深度较大的节点上：
```cpp
diff[u]++; diff[v]++; diff[lca] -= 2;
```

---

## ⚠️ 易错点

1. **倍增 LCA 的 LOG 大小**：2^k 需覆盖 n，LOG = ceil(log2(n)) + 1。
2. **depth 数组**：根 depth 设为 1（0 亦可，但要和 `fa[root][0] = 0` 配合）。
3. **树上差分注意方向**：点差分要减 `fa[lca]`，边差分只减 lca 两次。
4. **直径 DP 理解**：`d1 + d2` 是经过该节点的最长路径，全局取 max。
