# 栈与队列

**栈**是后进先出（LIFO），**队列**是先进先出（FIFO）。两者都是限定操作的线性表。

---

## 1. 栈 (Stack)

### STL stack 基本操作

```cpp
#include <stack>
stack<int> stk;
stk.push(x);    // 入栈
stk.pop();      // 出栈（无返回值）
stk.top();      // 栈顶元素
stk.empty();    // 判空
stk.size();     // 大小
// 注意：pop() 不返回值，要先 top() 再 pop()
```

### 数组模拟栈（考场推荐，更快）

```cpp
int stk[MAXN], top = 0;  // top 指向栈顶

void push(int x) { stk[++top] = x; }
void pop() { top--; }
int peek() { return stk[top]; }
bool empty() { return top == 0; }
```

## 2. 栈的经典应用

### 2.1 括号匹配

```cpp
bool isValid(string s) {
    stack<char> stk;
    for (char c : s) {
        if (c == '(' || c == '[' || c == '{')
            stk.push(c);
        else {
            if (stk.empty()) return false;
            char t = stk.top(); stk.pop();
            if ((c == ')' && t != '(') ||
                (c == ']' && t != '[') ||
                (c == '}' && t != '{')) return false;
        }
    }
    return stk.empty();
}
```

### 2.2 表达式求值（中缀转后缀）

两个栈：操作数栈 + 运算符栈。核心规则：**当前运算符优先级 ≤ 栈顶时，弹出栈顶计算**。

```cpp
// 简易版：只有 + - * / 和括号，无负数
int priority(char op) {
    if (op == '+' || op == '-') return 1;
    if (op == '*' || op == '/') return 2;
    return 0;
}

int calc(string s) {
    stack<int> nums;
    stack<char> ops;
    for (int i = 0; i < s.size(); i++) {
        char c = s[i];
        if (isdigit(c)) {
            int x = 0;
            while (i < s.size() && isdigit(s[i]))
                x = x * 10 + (s[i++] - '0');
            i--; nums.push(x);
        } else if (c == '(') {
            ops.push(c);
        } else if (c == ')') {
            while (ops.top() != '(') {
                // eval: pop 2 nums and 1 op, push result
                int b = nums.top(); nums.pop();
                int a = nums.top(); nums.pop();
                char op = ops.top(); ops.pop();
                if (op == '+') nums.push(a + b);
                else if (op == '-') nums.push(a - b);
                else if (op == '*') nums.push(a * b);
                else nums.push(a / b);
            }
            ops.pop(); // 弹出 '('
        } else {
            while (!ops.empty() && priority(ops.top()) >= priority(c)) {
                int b = nums.top(); nums.pop();
                int a = nums.top(); nums.pop();
                char op = ops.top(); ops.pop();
                if (op == '+') nums.push(a + b);
                else if (op == '-') nums.push(a - b);
                else if (op == '*') nums.push(a * b);
                else nums.push(a / b);
            }
            ops.push(c);
        }
    }
    while (!ops.empty()) {
        int b = nums.top(); nums.pop();
        int a = nums.top(); nums.pop();
        char op = ops.top(); ops.pop();
        if (op == '+') nums.push(a + b);
        else if (op == '-') nums.push(a - b);
        else if (op == '*') nums.push(a * b);
        else nums.push(a / b);
    }
    return nums.top();
}
```

## 3. 队列 (Queue)

### STL queue

```cpp
#include <queue>
queue<int> q;
q.push(x); q.pop(); q.front(); q.back(); q.empty();
// 同样，pop() 无返回值
```

### 数组模拟循环队列

```cpp
int q[MAXN], head = 0, tail = -1;

void push(int x) { q[++tail] = x; }
void pop() { head++; }
int front() { return q[head]; }
bool empty() { return head > tail; }

// 循环队列版本（节约空间）
void pushCircle(int x) {
    tail = (tail + 1) % MAXN;
    q[tail] = x;
}
```

### 队列的经典应用：BFS

```cpp
void bfs(int start) {
    queue<int> q;
    bool vis[MAXN] = {false};
    q.push(start); vis[start] = true;
    while (!q.empty()) {
        int u = q.front(); q.pop();
        for (int v : g[u]) {
            if (!vis[v]) {
                vis[v] = true;
                q.push(v);
            }
        }
    }
}
```

## 4. 双端队列 deque

```cpp
#include <deque>
deque<int> dq;
dq.push_front(x); dq.push_back(x);
dq.pop_front();    dq.pop_back();
dq.front();        dq.back();
```

CSP 常用于**0-1 BFS**：边权为 0 时 push_front，为 1 时 push_back，保证队列单调性。

## 5. 单调队列（详见 [[单调栈与单调队列]]）

**滑动窗口最大值**是 CSP 高频考点：维护一个**递减双端队列**，队头始终是窗口最大值。

```cpp
vector<int> maxSlidingWindow(vector<int>& nums, int k) {
    deque<int> dq;  // 存储下标
    vector<int> res;
    for (int i = 0; i < nums.size(); i++) {
        // 移除超出窗口的元素
        if (!dq.empty() && dq.front() == i - k) dq.pop_front();
        // 保持递减：弹出比当前小的
        while (!dq.empty() && nums[dq.back()] <= nums[i]) dq.pop_back();
        dq.push_back(i);
        if (i >= k - 1) res.push_back(nums[dq.front()]);
    }
    return res;
}
```

---

## ⚠️ 易错点

1. **STL pop() 无返回值**：必须 `x = top()/front(); pop();`。
2. **数组模拟栈的 top 初始值**：用 `top = 0` 表示空，入栈时 `stk[++top]` 而非 `stk[top++]`。
3. **表达式求值注意运算符优先级**：`* /` > `+ -`，左结合。
4. **单调队列存的是下标不是值**：方便判断窗口范围。
5. **BFS 入队时立即标记 vis**：防止同一个节点重复入队导致爆队列。
