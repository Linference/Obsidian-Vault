# 单调栈与单调队列

单调栈/队列的核心思想：**维护元素单调性，及时弹出无用元素**，将 O(n^2) 暴力优化到 O(n)。

---

## 1. 单调栈

栈内元素**单调递增或单调递减**。新元素入栈时，弹出所有破坏单调性的栈顶元素，再入栈。

### 经典问题：寻找每个元素下一个更大元素 (NGE)

```cpp
vector<int> nextGreaterElement(vector<int>& a) {
    int n = a.size();
    vector<int> ans(n, -1);
    stack<int> stk;
    for (int i = 0; i < n; i++) {
        while (!stk.empty() && a[stk.top()] < a[i]) {
            ans[stk.top()] = i;
            stk.pop();
        }
        stk.push(i);
    }
    return ans;
}
```

### 变种速查表

| 需求 | 单调方向 | pop 条件 |
|------|----------|----------|
| 右边第一个更大的 | 递减栈 | a[top] < a[i] |
| 右边第一个更小的 | 递增栈 | a[top] > a[i] |
| 左边第一个更大的 | 递减栈 | 弹完后栈顶即为 |
| 左边第一个更小的 | 递增栈 | 弹完后栈顶即为 |

### 左边第一个更小的（模板）

```cpp
vector<int> leftSmaller(vector<int>& a) {
    int n = a.size();
    vector<int> ans(n, -1);
    stack<int> stk;
    for (int i = 0; i < n; i++) {
        while (!stk.empty() && a[stk.top()] >= a[i]) stk.pop();
        ans[i] = stk.empty() ? -1 : stk.top();
        stk.push(i);
    }
    return ans;
}
```

## 2. 最大矩形面积

直方图中最大矩形面积：给定高度数组 h，求能画出的最大矩形。核心是对每个柱子，找左右第一个比它矮的柱子。

```cpp
int largestRectangleArea(vector<int>& h) {
    int n = h.size(), ans = 0;
    h.push_back(0);
    stack<int> stk;
    for (int i = 0; i <= n; i++) {
        while (!stk.empty() && h[stk.top()] > h[i]) {
            int height = h[stk.top()]; stk.pop();
            int left = stk.empty() ? -1 : stk.top();
            int width = i - left - 1;
            ans = max(ans, height * width);
        }
        stk.push(i);
    }
    h.pop_back();
    return ans;
}
```

末尾加 0 作为哨兵，保证所有柱子最终都被弹出计算。

## 3. 单调队列

双端队列，队内元素单调。队头是极值，队尾维护单调性。

### 滑动窗口最大值

```cpp
vector<int> maxSlidingWindow(vector<int>& a, int k) {
    int n = a.size();
    deque<int> dq;
    vector<int> ans;
    for (int i = 0; i < n; i++) {
        if (!dq.empty() && dq.front() <= i - k)
            dq.pop_front();
        while (!dq.empty() && a[dq.back()] <= a[i])
            dq.pop_back();
        dq.push_back(i);
        if (i >= k - 1)
            ans.push_back(a[dq.front()]);
    }
    return ans;
}
```

三步：移除过期 → 维护单调 → 入队。

### 滑动窗口最小值

把 `a[dq.back()] <= a[i]` 改为 `>=` 即可（维护递增队列）。

---

## 4. 单调队列优化 DP

对于形如 `dp[i] = max(dp[j]) + cost`（j 在某个滑动窗口内）的 DP，用单调队列维护 `dp[j]` 的极值，将 O(n^2) 优化到 O(n)。

经典例题：多重背包单调队列优化。按 mod wi 分组后，每组是独立的滑动窗口问题。

---

## 5. 单调栈 vs 单调队列

| 对比维度 | 单调栈 | 单调队列 |
|----------|--------|----------|
| 数据结构 | stack | deque |
| 适用范围 | 左右第一个更大/更小 | 滑动窗口极值 |
| 过期机制 | 无 | 有（窗口外过期） |
| 典型题 | NGE、最大矩形 | 滑动窗口极值、DP 优化 |

---

## 6. 直觉理解

**单调栈（递增栈为例）**：一群人站队，比你矮的人挡住比你高的人。高于你的被弹掉，剩下递增序列。

**单调队列（递减为例）**：窗口滑动，新来的又"年轻"又"高"→老又矮的再也用不上，弹掉。队头始终是窗口内最高。

## 易错点

1. 栈/队列存的是**下标**不是**值**，下标能同时拿到值和位置信息。
2. 比较符号别写反：递减栈用 `<`，递增栈用 `>`。
3. 弹出条件加等号 `>=` 或 `<=` 确保严格单调。
4. 滑动窗口过期判断：`dq.front() <= i - k`。
5. 最大矩形末尾加 0 做哨兵强制清空栈。
