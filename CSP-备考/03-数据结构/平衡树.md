# 平衡树

> 为什么需要平衡树？二叉搜索树可能退化成链表 O(n)，平衡树保证高度 O(log n)。

---

## 1. Treap = BST + Heap

**Treap** 的核心思想：每个节点额外带一个**随机优先级**，保持 BST 性质（键值）+ 堆性质（优先级），从而期望平衡。

```cpp
struct TreapNode {
    int val, pri, cnt, sz;
    TreapNode *l, *r;
    TreapNode(int v) : val(v), pri(rand()), cnt(1), sz(1), l(nullptr), r(nullptr) {}
};
```

### 旋转操作

```
右旋(y):       y           x
              / \   →     / \
             x   C       A   y
            / \             / \
           A   B           B   C

左旋(x):       x           y
              / \   →     / \
             A   y       x   C
                / \     / \
               B   C   A   B
```

```cpp
void pushUp(TreapNode* u) {
    u->sz = u->cnt + (u->l ? u->l->sz : 0) + (u->r ? u->r->sz : 0);
}

TreapNode* rotateRight(TreapNode* y) {
    TreapNode* x = y->l;
    y->l = x->r;
    x->r = y;
    pushUp(y); pushUp(x);
    return x;
}

TreapNode* rotateLeft(TreapNode* x) {
    TreapNode* y = x->r;
    x->r = y->l;
    y->l = x;
    pushUp(x); pushUp(y);
    return y;
}
```

### 插入

```cpp
TreapNode* insert(TreapNode* u, int val) {
    if (!u) return new TreapNode(val);
    if (val == u->val) { u->cnt++; pushUp(u); return u; }
    if (val < u->val) {
        u->l = insert(u->l, val);
        if (u->l->pri < u->pri) u = rotateRight(u);
    } else {
        u->r = insert(u->r, val);
        if (u->r->pri < u->pri) u = rotateLeft(u);
    }
    pushUp(u);
    return u;
}
```

### 删除

```cpp
TreapNode* remove(TreapNode* u, int val) {
    if (!u) return nullptr;
    if (val < u->val) u->l = remove(u->l, val);
    else if (val > u->val) u->r = remove(u->r, val);
    else {
        if (u->cnt > 1) { u->cnt--; pushUp(u); return u; }
        if (!u->l && !u->r) { delete u; return nullptr; }
        if (!u->l || (u->r && u->r->pri < u->l->pri)) {
            u = rotateLeft(u);
            u->l = remove(u->l, val);
        } else {
            u = rotateRight(u);
            u->r = remove(u->r, val);
        }
    }
    pushUp(u);
    return u;
}
```

### 查询操作

```cpp
// 查找值是否存在
bool find(TreapNode* u, int val) {
    if (!u) return false;
    if (u->val == val) return true;
    return val < u->val ? find(u->l, val) : find(u->r, val);
}

// 前驱：小于 val 的最大值
int getPrev(TreapNode* u, int val) {
    int res = INT_MIN;
    while (u) {
        if (u->val < val) { res = u->val; u = u->r; }
        else u = u->l;
    }
    return res;
}

// 后继：大于 val 的最小值
int getNext(TreapNode* u, int val) {
    int res = INT_MAX;
    while (u) {
        if (u->val > val) { res = u->val; u = u->l; }
        else u = u->r;
    }
    return res;
}

// 排名：小于 val 的数个数 + 1
int getRank(TreapNode* u, int val) {
    int rank = 1;
    while (u) {
        if (val <= u->val) u = u->l;
        else {
            rank += (u->l ? u->l->sz : 0) + u->cnt;
            u = u->r;
        }
    }
    return rank;
}

// 第 k 小
int getKth(TreapNode* u, int k) {
    while (u) {
        int leftSz = u->l ? u->l->sz : 0;
        if (k <= leftSz) u = u->l;
        else if (k <= leftSz + u->cnt) return u->val;
        else { k -= leftSz + u->cnt; u = u->r; }
    }
    return -1;
}
```

## 2. Splay 简介

**Splay** 每次操作后通过**旋转**将访问节点移到根。利用了访问的时间局部性。实现比 Treap 复杂，CSP 考场通常 Treap 够用。

核心操作 `splay(x, goal)`：将 x 旋转到 goal 的儿子位置。分 zig（父是根）、zig-zig（父和自身同向）、zig-zag（不同向）三种情况。

## 3. STL 替代方案

```cpp
#include <set>
set<int> st;
st.insert(x); st.erase(x);
auto it = st.find(x);  // O(log n)
st.lower_bound(x);     // >= x 的最小值
st.upper_bound(x);     // > x 的最小值
*st.begin();           // 最小值
*st.rbegin();          // 最大值

// multiset 允许重复元素
multiset<int> mst;
mst.insert(x);
mst.erase(mst.find(x));  // 仅删除一个
// mst.erase(x);          // 删除所有 x
```

**set 基于红黑树**，支持有序遍历，但**无法 O(log n) 查第 k 小**和**排名**（C++ 标准库没有这两个接口）。

## ⚠️ 易错点

1. **Treap 旋转后忘记 pushUp**：旋转改变子树结构，必须更新 sz。
2. **随机种子**：`srand(time(0))` 确保每次运行优先级不同。
3. **删除时三种情况**：cnt>1 减计数、叶子直接删、非叶子旋转下沉后递归删。
4. **multiset erase 问题**：`erase(x)` 删除所有 x，`erase(find(x))` 只删一个。
5. **pb_ds 只在 GCC 可用**：NOI Linux 2.0 支持，Windows 的 MinGW 也支持。
