# 树基础

> n 个节点、n-1 条边、无环连通图 = 树。任意两点间有且仅有一条简单路径。

---

## 1. 术语速查

| 术语 | 含义 |
|------|------|
| 根 root | 无父节点的节点（人为指定） |
| 叶子 leaf | 无子节点的节点 |
| 深度 depth | 从根到该节点的边数（根 depth=0） |
| 高度 height | 该节点到最深叶子的边数 |
| 子树 | 某节点及其所有后代 |
| 森林 | 多棵树的集合 |

---

## 2. 二叉树

每个节点**最多两个子节点**（左/右）。

### 特殊二叉树

| 类型 | 条件 |
|------|------|
| **满二叉树** | 每层节点全满（第 k 层有 2^k 个） |
| **完全二叉树** | 除最后一层外全满，最后一层左对齐 |
| **二叉搜索树 BST** | 左子树 < 根 < 右子树 |

---

## 3. 二叉树的存储

### 链式存储（指针）

```cpp
struct TreeNode {
    int val;
    TreeNode *left, *right;
    TreeNode(int v) : val(v), left(nullptr), right(nullptr) {}
};
```

### 数组存储（完全二叉树专用）

```cpp
// 1-indexed
int tree[MAXN];  // tree[i] 表示节点 i 的值
// 左儿子: 2*i   右儿子: 2*i+1   父: i/2
// 适用于堆、线段树等完全二叉树结构
```

## 4. 二叉树的遍历（重中之重）

```
        1
       / \
      2   3
     / \
    4   5

前序(根左右): 1 2 4 5 3
中序(左根右): 4 2 5 1 3
后序(左右根): 4 5 2 3 1
层序(BFS):    1 2 3 4 5
```

### 递归实现

```cpp
void preorder(TreeNode* root) {
    if (!root) return;
    cout << root->val << " ";
    preorder(root->left);
    preorder(root->right);
}

void inorder(TreeNode* root) {
    if (!root) return;
    inorder(root->left);
    cout << root->val << " ";
    inorder(root->right);
}

void postorder(TreeNode* root) {
    if (!root) return;
    postorder(root->left);
    postorder(root->right);
    cout << root->val << " ";
}
```

### 层序遍历（BFS）

```cpp
void levelorder(TreeNode* root) {
    if (!root) return;
    queue<TreeNode*> q;
    q.push(root);
    while (!q.empty()) {
        TreeNode* u = q.front(); q.pop();
        cout << u->val << " ";
        if (u->left) q.push(u->left);
        if (u->right) q.push(u->right);
    }
}
```

### 非递归前序/中序（用栈模拟）

```cpp
// 非递归前序
void preorderIter(TreeNode* root) {
    stack<TreeNode*> stk;
    TreeNode* cur = root;
    while (cur || !stk.empty()) {
        while (cur) {
            cout << cur->val << " ";  // 先访问
            stk.push(cur);
            cur = cur->left;
        }
        cur = stk.top(); stk.pop();
        cur = cur->right;
    }
}

// 非递归中序 — 仅访问时机不同
void inorderIter(TreeNode* root) {
    stack<TreeNode*> stk;
    TreeNode* cur = root;
    while (cur || !stk.empty()) {
        while (cur) {
            stk.push(cur);
            cur = cur->left;
        }
        cur = stk.top(); stk.pop();
        cout << cur->val << " ";  // 弹出时访问
        cur = cur->right;
    }
}
```

## 5. 已知遍历序列求树

**已知前序 + 中序 → 唯一确定二叉树**

```cpp
// pre[l1..r1] 和 in[l2..r2] 分别表示前序和中序区间
TreeNode* build(vector<int>& pre, vector<int>& in,
                int l1, int r1, int l2, int r2) {
    if (l1 > r1) return nullptr;
    int rootVal = pre[l1];
    TreeNode* root = new TreeNode(rootVal);
    int pos = l2;
    while (in[pos] != rootVal) pos++;  // 在中序中找根
    int leftSize = pos - l2;
    root->left  = build(pre, in, l1 + 1, l1 + leftSize, l2, pos - 1);
    root->right = build(pre, in, l1 + leftSize + 1, r1, pos + 1, r2);
    return root;
}
```

> 已知后序 + 中序同样可唯一确定。但**前序 + 后序**不能唯一确定（除非是真二叉树）。

## 6. 一般树的存储

### 邻接表（最常用）

```cpp
vector<int> g[MAXN];  // g[u] 存储 u 的所有邻接节点
g[u].push_back(v);
g[v].push_back(u);    // 无根树双向
```

### 孩子兄弟表示法

```cpp
struct Node {
    int val;
    Node *firstChild, *nextSibling;
};
// 将一般树转为二叉树
```

## 7. 并查集 vs 树

| 并查集 | 树 |
|--------|-----|
| 每个节点只知道父节点 | 知道全部子节点 |
| 向上查找 O(α(n)) | 向下遍历 O(n) |
| 用于集合合并判定 | 用于层次关系、路径 |

---

## ⚠️ 易错点

1. **二叉树遍历顺序记混**：前=根左右，中=左根右，后=左右根。口诀：前中后指**根的位置**。
2. **建树函数边界计算**：`leftSize = pos - l2`（不是 `pos - l2 + 1`）。
3. **数组存二叉树 1-indexed**：0 号位不用，否则 2i/2i+1 公式错误。
4. **邻接表 DFS 注意父节点回退**：加 `int parent` 参数防止往回走。
5. **完全二叉树 ≠ 满二叉树**：完全二叉树最后一层可不全满。
