# 线性表

**定义**：n 个相同类型元素的有限序列，元素之间有线性（一对一）关系。两种实现：**顺序表**（数组）和**链表**（指针）。

---

## 1. 顺序表（数组实现）

### 特点
- 内存**连续**，随机访问 `O(1)`
- 中间插入/删除需要移动元素，`O(n)`
- 适合**读多写少**、需要二分查找的场景

```cpp
const int MAXN = 100005;
int a[MAXN], n;

void insert(int pos, int val) {
    for (int i = n; i >= pos; i--) a[i + 1] = a[i];
    a[pos] = val;
    n++;
}

void erase(int pos) {
    for (int i = pos; i < n; i++) a[i] = a[i + 1];
    n--;
}
```

> CSP 考场数组绝对够用，`vector` 在不需要频繁头插时也可。

## 2. 单向链表

节点包含 `val` 和 `next` 指针。查找 `O(n)`，已知位置的插入/删除 `O(1)`。

```cpp
struct Node {
    int val;
    Node* next;
    Node(int v, Node* n = nullptr) : val(v), next(n) {}
};

// 头插法（常用）
Node* insertHead(Node* head, int val) {
    return new Node(val, head);
}

// 删除：需要前驱节点
void eraseAfter(Node* prev) {
    Node* tmp = prev->next;
    prev->next = tmp->next;
    delete tmp;
}
```

### 静态链表（数组模拟，考场推荐）

```cpp
int val[MAXN], nxt[MAXN], head = -1, tot = 0;

void insert(int v) {  // 头插
    val[++tot] = v;
    nxt[tot] = head;
    head = tot;
}

void erase(int prev_idx) {
    nxt[prev_idx] = nxt[nxt[prev_idx]];
}

// 遍历
for (int i = head; i != -1; i = nxt[i])
    cout << val[i] << " ";
```

## 3. 双向链表

```cpp
struct DNode {
    int val;
    DNode *prev, *next;
    DNode(int v) : val(v), prev(nullptr), next(nullptr) {}
};

// 在 p 之后插入
void insertAfter(DNode* p, int v) {
    DNode* q = new DNode(v);
    q->next = p->next; q->prev = p;
    if (p->next) p->next->prev = q;
    p->next = q;
}

// 删除节点 p 自身
void erase(DNode* p) {
    if (p->prev) p->prev->next = p->next;
    if (p->next) p->next->prev = p->prev;
    delete p;
}
```

## 4. 链表反转

### 迭代法

```cpp
Node* reverse(Node* head) {
    Node *prev = nullptr, *cur = head;
    while (cur) {
        Node* nxt = cur->next;
        cur->next = prev;
        prev = cur;
        cur = nxt;
    }
    return prev;
}
```

### 递归法

```cpp
Node* reverseRec(Node* head) {
    if (!head || !head->next) return head;
    Node* newHead = reverseRec(head->next);
    head->next->next = head;
    head->next = nullptr;
    return newHead;
}
```

## 5. 快慢指针

两个指针以不同速度遍历链表，是链表题的核心技巧。

| 技巧 | 应用 |
|------|------|
| 快指针走 2 步，慢指针走 1 步 | 找**中点**（快指针到末尾时慢指针在中点） |
| 同上 | 判断**是否有环**（快慢相遇则有环） |
| 快指针先走 k 步，然后一起走 | 找**倒数第 k 个节点** |

```cpp
bool hasCycle(Node* head) {
    Node *slow = head, *fast = head;
    while (fast && fast->next) {
        slow = slow->next;
        fast = fast->next->next;
        if (slow == fast) return true;
    }
    return false;
}
```

## 6. 数组 vs 链表选择

| 维度 | 数组 | 链表 |
|------|------|------|
| 随机访问 | O(1) ✅ | O(n) |
| 头/尾插入删除 | O(1)（尾部） | O(1) ✅ |
| 中间插入删除 | O(n) | O(n)（需先查找） |
| 内存连续性 | 连续（缓存友好） | 分散 |
| 动态扩容 | 需预分配或 vector | 天然动态 |

**CSP 铁律**：能用数组（含 vector）就用数组，链表只在**需要频繁在非尾部插入删除**且已持有指针时才用。

---

## ⚠️ 易错点

1. **链表删除后忘记 `delete`**：虽然 OJ 不判内存泄漏，但好的习惯是要释放。
2. **空指针解引用**：插入/删除前务必判空 `if (!head)`。
3. **数组模拟链表时索引从 1 开始**：`head` 初始为 0 或 -1，`nxt[0]` 不用。
4. **反转后忘记更新 head**：迭代法返回 `prev`，调用处要 `head = reverse(head)`。
5. **快慢指针的终止条件**：`while (fast && fast->next)` 两个条件缺一不可。
