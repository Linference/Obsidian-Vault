# 矩阵快速幂

## 核心思想

**矩阵快速幂 = 矩阵乘法 + 快速幂**，用于加速线性递推，将 O(n) 优化为 O(log n)。核心：将递推式写成矩阵形式，通过矩阵的 n 次幂一次性算出第 n 项。

## 矩阵乘法

定义：A(n×m) × B(m×p) = C(n×p)，`C[i][j] = Σ(k=0..m-1) A[i][k] × B[k][j]`。

```cpp
using ll = long long;

struct Matrix {
    int n, m;
    vector<vector<ll>> a;
    Matrix(int n, int m) : n(n), m(m), a(n, vector<ll>(m, 0)) {}

    Matrix operator*(const Matrix& other) const {
        Matrix res(n, other.m);
        for (int i = 0; i < n; i++)
            for (int k = 0; k < m; k++)
                if (a[i][k])  // 常数优化：跳过零
                    for (int j = 0; j < other.m; j++)
                        res.a[i][j] = (res.a[i][j] + a[i][k] * other.a[k][j]) % MOD;
        return res;
    }
};
```

> **三重循环顺序很关键**：i → k → j，利用 Cache 局部性原理，比 i → j → k 快不少。

## 矩阵快速幂

```cpp
Matrix matPow(Matrix a, ll b) {
    Matrix res(a.n, a.n);
    for (int i = 0; i < a.n; i++) res.a[i][i] = 1;  // 单位矩阵
    while (b) {
        if (b & 1) res = res * a;
        a = a * a;
        b >>= 1;
    }
    return res;
}
```

## 应用：斐波那契数列

递推：`F[n] = F[n-1] + F[n-2]`，`F[0] = 0, F[1] = 1`。

矩阵形式：
```
[F[n]  ]   [1 1]^(n-1)   [F[1]]
[F[n-1]] = [1 0]        × [F[0]]
```

```cpp
ll fib(ll n) {
    if (n == 0) return 0;
    if (n == 1) return 1;
    Matrix A(2, 2);
    A.a[0][0] = 1; A.a[0][1] = 1;
    A.a[1][0] = 1; A.a[1][1] = 0;
    A = matPow(A, n - 1);
    return (A.a[0][0] * 1 + A.a[0][1] * 0) % MOD;
}
```

## 一般线性递推

对于 `F[n] = c_1·F[n-1] + c_2·F[n-2] + ... + c_k·F[n-k]`：

k×k 矩阵形如（以 k=3 为例）：

```
[c_1  c_2  c_3]
[ 1    0    0 ]
[ 0    1    0 ]
```

第一行是系数，下面的 1 形成"移位"效果。

## 图上路径计数

**定理**：邻接矩阵 A 的 k 次幂中，`A^k[i][j]` 表示从 i 到 j 且长度恰好为 k 的路径条数。

```cpp
// 邻接矩阵 g[i][j] = 1 表示有边, 0 表示无边
Matrix path = matPow(adjMatrix, k);
// path.a[i][j] = i到j的k步路径数
```

应用：判断图中是否存在长度为 L 的路径、计数路径等。

## 矩阵乘法结合律 × 无交换律

- 结合律：(A×B)×C = A×(B×C) → 可用快速幂
- 无交换律：A×B ≠ B×A（一般情况）→ 注意乘法顺序

## ⚠️ 易错点

1. **循环顺序 i-k-j**：写成 i-j-k 会导致大量 Cache miss 和额外判断
2. **单位矩阵初始化**：对角线为 1，不是全 1 矩阵
3. **递推式对应关系**：矩阵乘法结果向量第一维 = 第 n 项，仔细验证
4. **取模遗漏**：所有加法乘法和中间结果都要取模
5. **n 值特判**：如斐波那契 n=0,1 时无需矩阵快速幂
6. **矩阵行列大小**：乘法时 A 的列数 = B 的行数，注意构造函数参数顺序
