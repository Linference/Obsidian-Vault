# 组合数学

## 核心思想

组合数学研究计数问题。CSP 中常见：排列组合、容斥原理、卡特兰数等。核心技术：**将计数问题转化为已知公式**。

## 加法原理与乘法原理

- **加法原理**：做一件事有 n 类互斥方法 → 总方案数 = 各类之和
- **乘法原理**：做一件事分 n 个独立步骤 → 总方案数 = 各步之积

## 排列 (P) 与组合 (C)

- 排列数 P(n,k) = n! / (n-k)!（有序选取 k 个）
- 组合数 C(n,k) = n! / (k! × (n-k)!)（无序选取 k 个）

```cpp
long long A(int n, int k, long long mod) {
    return fact[n] * invFact[n - k] % mod;  // 排列
}
long long C(int n, int k, long long mod) {
    return fact[n] * invFact[k] % mod * invFact[n - k] % mod;  // 组合
}
```

## 二项式定理

$$(a + b)^n = \sum_{k=0}^{n} C(n,k) \cdot a^{n-k} \cdot b^k$$

常用推论：
- `2^n = Σ C(n,k)`（a=b=1）
- `0 = Σ (-1)^k × C(n,k)`（a=1, b=-1）

## 可重排列与可重组合

- **可重排列**：n 种元素，取 r 个（可重复）→ n^r
- **可重组合**：n 种元素，取 r 个（可重复，顺序无关）→ C(n+r-1, r)

## 隔板法（挡板法）

把 n 个相同物品分成 m 份（每份至少 1 个）：C(n-1, m-1)

每份至少 0 个（允许空）：C(n+m-1, m-1)

**应用**：`x_1 + x_2 + ... + x_m = n` 的正整数解个数为 C(n-1, m-1)。

## 容斥原理

$$|A ∪ B ∪ C| = Σ|A| - Σ|A ∩ B| + Σ|A ∩ B ∩ C| - ...$$

通用公式：`|∪Ai| = Σ(-1)^(k+1) Σ|∩Aij|`

```cpp
// 例题：求 [1, n] 中能被 a, b, c 之一整除的数的个数
int ans = 0;
ans += n / a + n / b + n / c;                   // 一层
ans -= n / lcm(a,b) + n / lcm(b,c) + n / lcm(a,c); // 二层
ans += n / lcm(lcm(a,b), c);                    // 三层
```

## 错排公式

n 个元素的排列，没有元素在原来位置上的排列数：

$$D_n = (n-1) \times (D_{n-1} + D_{n-2})$$

初值：`D[0] = 1, D[1] = 0`

```cpp
const int MAXN = 1000;
long long D[MAXN];
D[0] = 1; D[1] = 0;
for (int i = 2; i < MAXN; i++)
    D[i] = (i - 1) * (D[i-1] + D[i-2]) % MOD;
```

## 卡特兰数

$$C_n = \frac{1}{n+1} C(2n, n) = \frac{C(2n, n)}{n+1}$$

递推形式：`C_n = Σ(C_i × C_{n-1-i})` for i=0..n-1，或 `C_n = C(2n,n) - C(2n,n+1)`。

前几项：`1, 1, 2, 5, 14, 42, 132, 429, 1430, 4862`

```cpp
long long catalan(int n) {
    return C(2*n, n) * inv(n + 1, MOD) % MOD;
}
```

**经典应用**：
- n对括号的合法序列数
- n个元素的出栈序列数
- n个节点的不同二叉树数
- n×n网格不越过对角线的路径数
- 凸n+2边形的三角剖分数

## 斯特林数（简介）

### 第二类斯特林数 S(n,k)

将 n 个不同元素分成 k 个非空集合的方案数。

递推式：`S(n,k) = S(n-1,k-1) + k × S(n-1,k)`

### 第一类斯特林数

n 个不同元素形成 k 个无向轮换的方案数。

递推式：`s(n,k) = s(n-1,k-1) + (n-1) × s(n-1,k)`

## next_permutation 枚举全排列

```cpp
vector<int> v = {1, 2, 3};
do {
    for (int x : v) cout << x << " ";
    cout << endl;
} while (next_permutation(v.begin(), v.end()));
// 输出顺序：字典序递增
// prev_permutation 是字典序递减
```

时间复杂度：O(n! × n)。仅适用于 n ≤ 10。

## ⚠️ 易错点

1. **排列 vs 组合**：排列有序，组合无序，看清题目要求
2. **取模除法**：组合数公式中除法必须用逆元，不能直接除
3. **卡特兰数下标**：第 n 项 = C(2n,n)/(n+1)，注意 n 从 0 开始
4. **容斥符号**：加一层，减二层，加三层... 交替进行
5. **隔板法前提**：物品相同，份数可以区分
6. **组合数溢出**：n>mod 时用卢卡斯定理处理
