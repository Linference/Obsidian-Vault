# 数论

## 核心思想

数论研究整数的性质。在CSP中，数论主要用于取模运算、求逆元、解同余方程等。

## 素数判定

试除法，O(√n)：

```cpp
bool isPrime(long long n) {
    if (n < 2) return false;
    for (long long i = 2; i * i <= n; i++)
        if (n % i == 0) return false;
    return true;
}
```

## 素数筛

### 埃氏筛 O(n log log n)

```cpp
vector<int> sieve(int n) {
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (!isPrime[i]) continue;
        for (int j = i * i; j <= n; j += i)
            isPrime[j] = false;
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++)
        if (isPrime[i]) primes.push_back(i);
    return primes;
}
```

### 欧拉线性筛 O(n)

每个合数只被其最小质因数筛掉：

```cpp
vector<int> linearSieve(int n) {
    vector<int> primes;
    vector<int> minPrime(n + 1, 0);
    for (int i = 2; i <= n; i++) {
        if (minPrime[i] == 0) {
            minPrime[i] = i;
            primes.push_back(i);
        }
        for (int p : primes) {
            if (p > minPrime[i] || i * p > n) break;
            minPrime[i * p] = p;
        }
    }
    return primes;
}
```

## 质因数分解

```cpp
vector<pair<long long, int>> factorize(long long n) {
    vector<pair<long long, int>> factors;
    for (long long i = 2; i * i <= n; i++) {
        if (n % i == 0) {
            int cnt = 0;
            while (n % i == 0) { n /= i; cnt++; }
            factors.push_back({i, cnt});
        }
    }
    if (n > 1) factors.push_back({n, 1});
    return factors;
}
```

## 最大公约数 (GCD)

辗转相除法（欧几里得算法），O(log min(a,b))：

```cpp
long long gcd(long long a, long long b) {
    return b == 0 ? a : gcd(b, a % b);
}

long long lcm(long long a, long long b) {
    return a / gcd(a, b) * b;  // 先除再乘，防溢出
}
```

C++17 起可用 `__gcd` 和 `__lcm`（GCC 内置）。

## 扩展欧几里得 (exgcd)

求 `ax + by = gcd(a, b)` 的整数解 (x, y)：

```cpp
long long exgcd(long long a, long long b, long long &x, long long &y) {
    if (b == 0) { x = 1; y = 0; return a; }
    long long d = exgcd(b, a % b, y, x);
    y -= a / b * x;
    return d;
}
```

应用：求模意义下的**乘法逆元**。

## 快速幂 O(log n)

```cpp
long long qpow(long long a, long long b, long long mod) {
    long long res = 1;
    while (b) {
        if (b & 1) res = res * a % mod;
        a = a * a % mod;
        b >>= 1;
    }
    return res;
}
```

## 乘法逆元

**法一：费马小定理**（mod为质数时）：`inv(a) = a^(mod-2) mod MOD`

```cpp
long long inv(long long a, long long mod) {
    return qpow(a, mod - 2, mod);
}
```

**法二：exgcd**：求 `a * x ≡ 1 (mod m)`，即解 `ax + my = 1`。

```cpp
long long inv_exgcd(long long a, long long mod) {
    long long x, y;
    exgcd(a, mod, x, y);
    return (x % mod + mod) % mod;
}
```

**法三：递推求 1~n 的逆元**：`inv[i] = (mod - mod/i) * inv[mod%i] % mod`。

```cpp
inv[1] = 1;
for (int i = 2; i <= n; i++)
    inv[i] = (mod - mod / i) * inv[mod % i] % mod;
```

## 欧拉函数

φ(n) = [1, n] 中与 n 互质的整数个数。

性质：
- 若 p 为质数，φ(p) = p - 1
- 积性：若 gcd(a,b) = 1，则 φ(ab) = φ(a)φ(b)
- 通项公式：φ(n) = n × Π(1 - 1/p_i)

```cpp
long long phi(long long n) {
    long long ans = n;
    for (long long i = 2; i * i <= n; i++) {
        if (n % i == 0) {
            ans = ans / i * (i - 1);
            while (n % i == 0) n /= i;
        }
    }
    if (n > 1) ans = ans / n * (n - 1);
    return ans;
}
```

## 中国剩余定理 (CRT)

解同余方程组 `x ≡ a_i (mod m_i)`，其中诸 `m_i` 两两互质。

```cpp
long long CRT(vector<long long>& a, vector<long long>& m) {
    long long M = 1, ans = 0;
    for (auto mi : m) M *= mi;
    for (int i = 0; i < a.size(); i++) {
        long long Mi = M / m[i];
        long long t = inv(Mi % m[i], m[i]);  // Mi在模m[i]下的逆元
        ans = (ans + a[i] * Mi % M * t % M) % M;
    }
    return ans;
}
```

## 组合数

```cpp
const int MAXN = 1e6;
long long fact[MAXN], invFact[MAXN];

// 预计算阶乘和阶乘逆元
void precompute(long long mod) {
    fact[0] = invFact[0] = 1;
    for (int i = 1; i < MAXN; i++) fact[i] = fact[i-1] * i % mod;
    invFact[MAXN-1] = qpow(fact[MAXN-1], mod - 2, mod);
    for (int i = MAXN - 2; i >= 1; i--)
        invFact[i] = invFact[i+1] * (i + 1) % mod;
}

long long C(int n, int k, long long mod) {
    if (k < 0 || k > n) return 0;
    return fact[n] * invFact[k] % mod * invFact[n-k] % mod;
}
```

## 卢卡斯定理

当 n 很大但 mod 是较小质数时：

```cpp
long long lucas(long long n, long long m, long long mod) {
    if (m == 0) return 1;
    return C(n % mod, m % mod, mod) * lucas(n / mod, m / mod, mod) % mod;
}
```

## ⚠️ 易错点

1. **LCM溢出**：`a * b / gcd(a,b)` 中间积可能溢出，应写成 `a / gcd(a,b) * b`
2. **逆元不存在**：gcd(a, mod) != 1 时 a 没有模 mod 的逆元
3. **负数取模**：C++中 `-3 % 5 == -3`，应写成 `((x % mod) + mod) % mod`
4. **快速幂取模遗漏**：a*a 时也要取模
5. **CRT要求模数互质**：不互质要用扩展CRT（exCRT）
6. **组合数预处理范围**：fact数组必须够大，否则越界
