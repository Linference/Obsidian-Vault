# 06-搜索 MOC

## 模块概览

搜索是 CSP-J/S 的**必考模块**，J 组侧重 DFS/BFS 基本功，S 组要求剪枝、记忆化、启发式搜索。

## 知识图谱

```
搜索
├── 基础搜索
│   ├── [[BFS与DFS]]
│   │   ├── DFS（深度优先）—— 递归 + 回溯
│   │   ├── BFS（广度优先）—— 队列 + 逐层
│   │   ├── 连通块 / 洪水填充
│   │   └── 迷宫问题
│   └── 状态去重（vis 数组 / unordered_set）
│
├── 优化技巧
│   ├── [[剪枝与记忆化搜索]]
│   │   ├── 可行性剪枝
│   │   ├── 最优性剪枝
│   │   ├── 顺序剪枝
│   │   ├── 记忆化搜索 = DFS + DP
│   │   └── 经典题：小木棍 (sticks)
│   └── 双向 BFS（从起点和终点同时搜）
│
└── 启发式搜索
    └── [[A星与IDA星]]
        ├── A* = BFS + 估价函数 f = g + h
        ├── IDA* = 迭代加深 + A* 剪枝
        └── 经典题：八数码 / 十五数码
```

## 学习路线

| 阶段 | 内容 | 目标 |
|------|------|------|
| 1 | DFS / BFS 基础模板 | 能写出框架，解决迷宫/连通块 |
| 2 | 回溯 + 八皇后 | 掌握"做选择→递归→撤销" |
| 3 | BFS 最短路径 | 理解层序 = 最短 |
| 4 | 剪枝策略 | 小木棍/数的划分 |
| 5 | 记忆化搜索 | 数位 DP / 棋盘路径 |
| 6 | A* / IDA* | 八数码 / 骑士精神 |

## 核心对比

| 算法 | 数据结构 | 空间 | 最优解 | 适用 |
|------|----------|------|--------|------|
| DFS | 栈（递归） | O(深度) | 否 | 所有解、回溯 |
| BFS | 队列 | O(宽度) | 是(等权) | 最短路径 |
| 记忆化 | 哈希表/数组 | O(状态数) | 是 | 重叠子问题 |
| A* | 优先队列 | O(分支) | 是 | 有启发函数 |
| IDA* | 栈（递归） | O(深度) | 是 | 空间受限 |

## 典型题型

- **迷宫最短路径** → BFS
- **八皇后 / 全排列** → DFS 回溯
- **小木棍拼凑** → DFS + 多重剪枝
- **八数码** → BFS / A* / IDA*
- **数位 DP** → 记忆化搜索
- **连通块计数** → DFS / BFS flood fill
