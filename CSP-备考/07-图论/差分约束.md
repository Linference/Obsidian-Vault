# 差分约束系统

---

## 1. 问题描述

给定 n 个变量 x1...xn 和 m 个不等式约束，每个约束形如：

x_i - x_j <= c_k

求一组可行解（或判断无解）。

注意不等号方向。不同题目可能是 >=、<=、>、<。统一化成 <= 或 >= 再用图论。

---

## 2. 核心转化：不等式 -> 最短路

### 关键观察

最短路中的三角不等式：

dist[v] <= dist[u] + w(u, v)

移项得：

dist[v] - dist[u] <= w(u, v)

这与差分约束 x_i - x_j <= c 形式完全一致。

### 建图规则

对约束 x_i - x_j <= c：
建边：j -> i, w = c

记忆口诀：**变号移到右边，从减数连到被减数**。

---

## 3. 求解步骤

### step 1：统一不等号方向

| 原始 | 化成 <= | 建边 |
|------|---------|------|
| xi - xj <= c | 不变 | j -> i, w = c |
| xi - xj >= c | xj - xi <= -c | i -> j, w = -c |
| xi - xj < c | xi - xj <= c-1（整数） | j -> i, w = c-1 |
| xi = xj + c | xi - xj <= c 且 xj - xi <= -c | j->i(w=c), i->j(w=-c) |

### step 2：添加超级源点

建一个超级源点 S = 0（或 n+1），向所有顶点连权值为 0 的边：

S -> i，w = 0（i = 1..n）

意义：x_i - x_S <= 0，即把 x_S 作为基准参考。

### step 3：跑最短路（SPFA）

从超级源点出发跑 SPFA / Bellman-Ford，得到 dist[i] 即为 x_i 的一组可行解。

### step 4：判无解

若存在**负环** -> 不等式组无解（矛盾约束）。

## 4. 完整模板

```cpp
const int N = 5005;
const int INF = 0x3f3f3f3f;
vector<pair<int,int>> g[N];
int dist[N], cnt[N];
bool inq[N];
int n, m;

bool solve() {
    // 超级源点 0 -> 所有点
    for (int i = 1; i <= n; i++)
        g[0].push_back({i, 0});

    memset(dist, 0x3f, sizeof(dist));
    memset(cnt, 0, sizeof(cnt));
    dist[0] = 0;
    queue<int> q;
    q.push(0);
    inq[0] = true;

    while (!q.empty()) {
        int u = q.front(); q.pop();
        inq[u] = false;

        for (auto [v, w] : g[u]) {
            if (dist[v] > dist[u] + w) {
                dist[v] = dist[u] + w;
                cnt[v] = cnt[u] + 1;
                if (cnt[v] > n) return false;   // 负环 -> 无解
                if (!inq[v]) {
                    q.push(v);
                    inq[v] = true;
                }
            }
        }
    }
    return true;   // dist[1..n] 即为一组解
}
```

## 5. 变式：求最大 / 最小值

| 目标 | 不等号 | 算法 | 结果 |
|------|:------:|------|------|
| 求 xi 的**最大值** | 全部化成 <= | 最短路 | dist[i] |
| 求 xi 的**最小值** | 全部化成 >= | 最长路（dist 初始 -INF） | dist[i] |

## 6. 典型例题

### 例题 1：区间选点（Luogu P1250 / POJ 1201）

在数轴 [0, 50000] 上选最少的点，使得每个区间 [a, b] 内至少选 c 个。

**建模**：
- 设 s[i] 为 [0, i] 已选点数
- 约束：
  - s[b] - s[a-1] >= c -> s[a-1] - s[b] <= -c -> 建边 b -> a-1, w = -c
  - 0 <= s[i] - s[i-1] <= 1 -> s[i-1] - s[i] <= 0 和 s[i] - s[i-1] <= 1
- 求 s[50000] 的最小值 -> 跑最长路

### 例题 2：糖果分配（Luogu P3275）

有 n 个小朋友分糖果，m 个约束（=、>=、>、<=、<），求最少糖果总数。

**建模**：每个约束转为差分约束，设超级源点，跑最长路。若出现正环则无解。

## 7. 差分约束 vs 普通最短路

| | 最短路 | 差分约束 |
|------|--------|-----------|
| 已知 | 图的边权和起点 | 不等式组 |
| 需求 | 最短距离 | 一组可行解 / 极值 |
| 负权 | 存在时需特殊算法 | 就是对着来的 |
| 负环 | 不可达/无意义 | 有环 = 无解 |
| 超级源点 | 不需要 | **必须加** |

## 易错点

1. **忘记超级源点**：必须添加 S->i (w=0)，否则图不连通，SPFA 无法遍历全局
2. **不等号方向**：化 <= 时 a - b <= c -> for b->a w=c；化 >= 时 a - b >= c -> for a->b w=c（跑最长路）
3. **严格不等式**：x_i - x_j < c 在整数域等同于 x_i - x_j <= c-1
4. **SPFA 判环**：cnt 从 0 开始，如果超级源点也算一个点，判 cnt > n（n+1 个点）
5. **负环 vs 正环**：最短路判负环，最长路判正环，别搞混
