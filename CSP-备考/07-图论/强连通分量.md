# 强连通分量 (SCC)

---

## 定义

- **强连通**：有向图中，若 u 可达 v 且 v 可达 u，则 u 和 v 强连通
- **强连通分量 (SCC)**：极大的强连通子图（加入任何其他顶点都会破坏强连通性）
- **缩点**：将每个 SCC 收缩为一个点，得到 DAG

---

## 1. Tarjan 算法

### 核心思想

一次 DFS 同时求出所有 SCC，O(n + m)。

维护两个数组：

| 数组 | 含义 |
|------|------|
| `dfn[u]` | u 在 DFS 中被发现的**时间戳**（不变） |
| `low[u]` | u 及其后代**能追溯到的**最早祖先的 dfn（动态更新） |

**判定**：当 `dfn[u] == low[u]` 时，u 是当前 SCC 的"根"，栈中 u 及以上的点构成一个 SCC。

### low 的更新规则

```cpp
// 设当前节点为 u，邻边为 u → v

// 情形 1：v 尚未访问（树边）
dfs(v);
low[u] = min(low[u], low[v]);

// 情形 2：v 已访问且 v 在栈中（后向边 / 横叉边）
low[u] = min(low[u], dfn[v]);
```

> ⚠️ **关键**：情形 2 必须用 `dfn[v]` 而非 `low[v]`！用 `low[v]` 在某些横叉边场景会出错。

### 模板

```cpp
const int N = 100005;
vector<int> g[N];
int dfn[N], low[N], scc[N], sccSize[N];
bool instk[N];
stack<int> stk;
int timer = 0, sccCnt = 0;

void tarjan(int u) {
    dfn[u] = low[u] = ++timer;
    stk.push(u);
    instk[u] = true;

    for (auto v : g[u]) {
        if (!dfn[v]) {
            tarjan(v);
            low[u] = min(low[u], low[v]);
        } else if (instk[v]) {         // v 在栈中 → 后向边/横叉边
            low[u] = min(low[u], dfn[v]);
        }
    }

    if (dfn[u] == low[u]) {            // u 是 SCC 的根
        sccCnt++;
        int v;
        do {
            v = stk.top(); stk.pop();
            instk[v] = false;
            scc[v] = sccCnt;
            sccSize[sccCnt]++;
        } while (v != u);
    }
}

// 主函数调用
for (int i = 1; i <= n; i++)
    if (!dfn[i]) tarjan(i);
```

### 复杂度

O(n + m)，每个顶点和每条边各被访问一次。

## 2. Kosaraju 算法

### 核心思想

1. **第一遍 DFS（正图）**：记录每个顶点**离开**的时间顺序（后序）
2. **第二遍 DFS（逆图）**：按后序的**逆序**在逆图上 DFS，每个连通块即一个 SCC

### 模板

```cpp
vector<int> g[N], rg[N];    // 正图、逆图
vector<int> order;           // 后序遍历顺序
bool vis[N];
int scc[N], sccCnt;

void dfs1(int u) {
    vis[u] = true;
    for (auto v : g[u])
        if (!vis[v]) dfs1(v);
    order.push_back(u);      // 后序入队
}

void dfs2(int u, int id) {
    scc[u] = id;
    for (auto v : rg[u])
        if (!scc[v]) dfs2(v, id);
}

void kosaraju() {
    for (int i = 1; i <= n; i++)
        if (!vis[i]) dfs1(i);

    for (int i = order.size() - 1; i >= 0; i--) {
        int u = order[i];
        if (!scc[u]) dfs2(u, ++sccCnt);
    }
}
```

| 对比 | Tarjan | Kosaraju |
|------|:------:|:--------:|
| 时间 | O(n+m) | O(n+m)，但两次 DFS |
| 空间 | 需维护栈 | 需存逆图 |
| 代码 | 稍复杂，low 更新易错 | 直观简单 |
| 考场地 | ✅ 首选（一次遍历） | ⚠️ 备选 |

## 3. SCC 缩点

将 SCC 缩成 DAG 后，可以在 DAG 上 DP / 拓扑排序。

```cpp
// 缩点建 DAG
vector<int> dag[N];
int indeg[N];

for (int u = 1; u <= n; u++) {
    for (auto v : g[u]) {
        if (scc[u] != scc[v]) {
            dag[scc[u]].push_back(scc[v]);
            indeg[scc[v]]++;
        }
    }
}

// DAG 上拓扑 DP（示例：求从某个 SCC 出发的最长路）
void topoDP() {
    queue<int> q;
    vector<int> dp(sccCnt + 1, 0);
    for (int i = 1; i <= sccCnt; i++)
        if (indeg[i] == 0) q.push(i);

    while (!q.empty()) {
        int u = q.front(); q.pop();
        dp[u] += sccSize[u];            // 加上自身 SCC 的大小
        for (auto v : dag[u]) {
            dp[v] = max(dp[v], dp[u]);
            if (--indeg[v] == 0) q.push(v);
        }
    }
}
```

## 4. 应用场景

### 4.1 2-SAT

布尔变量 x₁...xₙ，m 个形如 `(a ∨ b)` 的子句（a, b 为变量或否定），判断是否存在一组赋值使所有子句为真。

**转换为 SCC**：
- 每个变量建两个点：`x`（真）和 `¬x`（假）
- `a ∨ b` 等价于 `¬a → b` 和 `¬b → a`
- 若某变量的真和假在同一 SCC → 无解
- 否则按 SCC 的拓扑序**逆序**赋值（越后面的 SCC 越先为真）

```cpp
// i: 为真, i+n: 为假
void addImpl(int u, int v) { g[u].push_back(v); }

// a or b:
addImpl(a + n, b);   // ¬a → b
addImpl(b + n, a);   // ¬b → a
```

### 4.2 传递闭包 + 可达性

求有向图中某点可达哪些点 / 哪些点可达该点 → 先缩点再拓扑求可达性。

### 4.3 半连通分量

缩点后 DAG 上的最长链。

## ⚠️ 易错点

1. **low 更新**：后向边必须用 `dfn[v]` 而不是 `low[v]`！这是最常见的错误
2. **栈判定**：`instk[v]` 必须检查，只对栈内节点更新 low。用 `dfn[v] != 0` 代替会错
3. **未访问的分量**：`for (i=1..n) if(!dfn[i]) tarjan(i)` ——图不一定全连通
4. **缩点后重边**：缩点建 DAG 时可能有重边，需要的话用 `set<pair>` 去重
5. **2-SAT 建边**：`a ∨ b` 建两条边 `¬a→b` 和 `¬b→a`，不要漏
