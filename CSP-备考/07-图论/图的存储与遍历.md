# 图的存储与遍历

---

## 1. 基本概念

| 术语 | 定义 |
|------|------|
| **顶点 (Vertex)** | 图的基本单元，编号通常从 1 开始 |
| **边 (Edge)** | 连接两个顶点的关系，分有向 / 无向 |
| **权值 (Weight)** | 边上的数值，表示距离、代价等 |
| **度 (Degree)** | 无向图中与顶点相连的边数 |
| **入度 / 出度** | 有向图中进入 / 离开顶点的边数 |
| **路径** | 从 u 到 v 经过的顶点序列 |
| **连通图** | 无向图中任意两点均可达；有向图分强/弱连通 |

> **约定**：n = 顶点数，m = 边数。下文中 `g[][]`、`head[]` 等均以 1 为起始下标。

---

## 2. 三种存储方式

### 2.1 邻接矩阵

**思想**：用二维数组 `g[i][j]` 表示 i→j 的边权（无边则为 INF，自身为 0）。

```cpp
const int N = 1005;
const int INF = 0x3f3f3f3f;
int g[N][N]; // g[i][j] 表示 i->j 的边权

void add(int u, int v, int w) {
    g[u][v] = w;           // 有向边
    // g[v][u] = w;        // 无向图加这行
}
```

| 优点 | 缺点 |
|------|------|
| O(1) 查边是否存在 | O(n²) 空间，n > 5000 直接 MLE |
| 适合 Floyd 多源最短路 | 遍历邻边需 O(n)，稀疏图浪费严重 |

**适用**：稠密图（m ≈ n²），如 Floyd 算法、Prim 朴素版。

---

### 2.2 邻接表 (vector)

**思想**：每个顶点维护一个 `vector<pair<int,int>>`，存放 (邻点, 边权)。

```cpp
const int N = 100005;
struct Edge {
    int to, w;
};
vector<Edge> g[N];   // g[u] 存储 u 的所有出边

void add(int u, int v, int w) {
    g[u].push_back({v, w});
    // g[v].push_back({u, w}); // 无向图
}

// 遍历 u 的邻边
for (auto [v, w] : g[u]) {
    // ...
}
```

| 优点 | 缺点 |
|------|------|
| O(n + m) 空间，足够大 | vector 动态扩容有微小开销 |
| 写法直观 | 反向边查找慢 |

**适用**：99% 的稀疏图场景。

---

### 2.3 链式前向星

**思想**：用数组模拟链表，每条边记录 (to, w, nxt)，head[u] 指向 u 的第一条边。

```cpp
const int N = 100005, M = 200005;  // M 为边数×2（无向图）
int head[N], to[M], nxt[M], w[M];
int cnt = 0;

void add(int u, int v, int wt) {
    to[++cnt] = v;
    w[cnt] = wt;
    nxt[cnt] = head[u];
    head[u] = cnt;
}

// 遍历 u 的邻边
for (int i = head[u]; i; i = nxt[i]) {
    int v = to[i], wt = w[i];
    // ...
}
```

| 优点 | 缺点 |
|------|------|
| 数组连续，常数最小 | 代码比 vector 繁琐 |
| 无动态扩容，极限压常数 | 删边麻烦 |

**适用**：需要极致性能的题目或含多组数据的题目。

> ⚡ **小技巧**：用 `cnt = 1` 开始 + `i ^ 1` 可快速找反向边（网络流必备）。

## 3. DFS 遍历图

### 3.1 连通分量计数

```cpp
int n, vis[N];
vector<int> g[N];

void dfs(int u) {
    vis[u] = 1;
    for (auto v : g[u]) {
        if (!vis[v]) dfs(v);
    }
}

int countComponents() {
    int cnt = 0;
    for (int i = 1; i <= n; i++) {
        if (!vis[i]) {
            dfs(i);
            cnt++;               // 每次 dfs 发现一个新连通块
        }
    }
    return cnt;
}
```

### 3.2 无向图环检测

```cpp
bool dfs(int u, int fa) {
    vis[u] = 1;
    for (auto v : g[u]) {
        if (v == fa) continue;   // 跳过父节点
        if (vis[v]) return true; // 遇到已访问节点 → 有环
        if (dfs(v, u)) return true;
    }
    return false;
}
```

> 有向图环检测：维护 0=未访问 / 1=访问中 / 2=已完成 三种状态。

## 4. BFS 遍历图

```cpp
void bfs(int s) {
    queue<int> q;
    vector<int> dist(n + 1, -1);
    q.push(s);
    dist[s] = 0;

    while (!q.empty()) {
        int u = q.front(); q.pop();
        for (auto v : g[u]) {
            if (dist[v] == -1) {     // 未访问
                dist[v] = dist[u] + 1;
                q.push(v);
            }
        }
    }
}
```

**应用**：无权图最短路径、分层、二分图染色。

## 5. 拓扑排序

**定义**：有向无环图 (DAG) 的线性序列，所有边从前指向后。

### Kahn 算法 (BFS)

```cpp
int indeg[N];
vector<int> g[N];

vector<int> topoSort() {
    queue<int> q;
    vector<int> res;
    for (int i = 1; i <= n; i++)
        if (indeg[i] == 0) q.push(i);

    while (!q.empty()) {
        int u = q.front(); q.pop();
        res.push_back(u);
        for (auto v : g[u]) {
            if (--indeg[v] == 0) q.push(v);
        }
    }

    if (res.size() != n) return {};  // 有环，无法拓扑排序
    return res;
}
```

> **判环**：如果输出序列长度 != n，则图存在环。

## ⚠️ 易错点

1. **无向图开数组**：链式前向星中 M 要开 `2 * maxM`，不然 RE
2. **vis 状态**：有向图环检测必须用三色标记（0/1/2），用两色会误判
3. **拓扑排序**：多组数据时忘记清空 `indeg[]` 和 `g[]`
4. **邻接矩阵的 INF**：用 `0x3f3f3f3f`，不能用 `INT_MAX`（加法会溢出）
5. **0-index vs 1-index**：题目顶点编号通常从 1 开始
