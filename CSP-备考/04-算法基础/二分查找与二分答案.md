# 二分查找与二分答案

## 核心思想

**二分查找**：在有序数据中，每次把搜索范围缩小一半，从 O(n) 降到 O(log n)。

**二分答案**：当问题难以直接求解，但容易判断"某个值是否可行"时，对答案值域二分，将**最优化问题转化为判定问题**。

---

## 二分查找

### 前提条件

- 数据**有序**（升序或降序）
- 支持**随机访问**（数组、vector）

### 模板一：左闭右闭 `[l, r]`

```cpp
// 查找 target 的下标，不存在返回 -1
int binarySearch(int a[], int n, int target) {
    int l = 0, r = n - 1;
    while (l <= r) {
        int mid = l + (r - l) / 2;    // 防溢出写法
        if (a[mid] == target) return mid;
        else if (a[mid] < target) l = mid + 1;
        else r = mid - 1;
    }
    return -1;
}
```

### 模板二：左闭右开 `[l, r)`

```cpp
int binarySearch2(int a[], int n, int target) {
    int l = 0, r = n;
    while (l < r) {
        int mid = l + (r - l) / 2;
        if (a[mid] >= target) r = mid;       // 找第一个 ≥ target 的位置
        else l = mid + 1;
    }
    return l;  // l 是第一个 ≥ target 的位置（即 lower_bound）
}
```

### lower_bound 和 upper_bound

| STL 函数 | 含义 | 示例 (a=[1,2,2,3,4,4,5]) |
|----------|------|--------------------------|
| `lower_bound(a, a+n, x)` | 第一个 ≥ x 的位置 | `lower_bound(3)` → 下标 3 |
| `upper_bound(a, a+n, x)` | 第一个 > x 的位置 | `upper_bound(3)` → 下标 4 |

```cpp
#include <algorithm>
int *p = lower_bound(a, a + n, x);   // 返回指针
int idx = p - a;                      // 转下标
int count = upper_bound(a, a+n, x) - lower_bound(a, a+n, x); // x 出现的次数

// vector 用法
auto it = lower_bound(v.begin(), v.end(), x);
```

**手写 lower_bound（左闭右开）**：

```cpp
int lowerBound(int a[], int n, int x) {
    int l = 0, r = n;
    while (l < r) {
        int mid = l + (r - l) / 2;
        if (a[mid] >= x) r = mid;
        else l = mid + 1;
    }
    return l;   // 返回第一个 ≥ x 的下标
}
```

## 二分答案（核心考点）

### 三段式模板

```cpp
int l = 最小可能答案, r = 最大可能答案, ans = -1;
while (l <= r) {
    int mid = l + (r - l) / 2;
    if (check(mid)) {
        ans = mid;
        l = mid + 1;   // 求最大值，扩大范围
        // r = mid - 1; // 求最小值，缩小范围
    } else {
        r = mid - 1;   // 求最大值，缩小范围
        // l = mid + 1; // 求最小值，扩大范围
    }
}
return ans;
```

> 口诀：**求最大值 → check 成立时 `l=mid+1`；求最小值 → check 成立时 `r=mid-1`**

### check 函数设计

check(mid) 回答：**如果答案是 mid，能不能满足题目条件？**

```cpp
// 例：能否在不超过 M 次操作内，把所有数降到 ≤ limit
bool check(int limit) {
    int ops = 0;
    for (int i = 0; i < n; i++)
        if (a[i] > limit)
            ops += (a[i] - limit + k - 1) / k;  // 上取整
    return ops <= M;
}
```

## 浮点数二分

```cpp
double l = 0, r = 1e9;
for (int i = 0; i < 100; i++) {       // 固定 100 次迭代保证精度
    double mid = (l + r) / 2;
    if (check(mid)) r = mid;          // 求最小值
    else l = mid;
}
// 或用精度控制：
while (r - l > 1e-9) {
    double mid = (l + r) / 2;
    if (check(mid)) r = mid;
    else l = mid;
}
```

## 典型例题

### 1. 洛谷 P1873 砍树

> N 棵树高度 h[i]，锯子高度为 H 时能获得 sum(max(0, h[i]-H)) 的木材。问至少获得 M 米木材时，H 最大是多少。

**思路**：H 越大，木材越少。二分 H，check 计算得到木材总量是否 ≥ M。求最大 H。

```cpp
long long l = 0, r = *max_element(h, h + n), ans = 0;
while (l <= r) {
    long long mid = (l + r) / 2;
    long long sum = 0;
    for (int i = 0; i < n; i++)
        if (h[i] > mid) sum += h[i] - mid;
    if (sum >= M) { ans = mid; l = mid + 1; }
    else r = mid - 1;
}
cout << ans;
```

### 2. 洛谷 P2678 跳石头

> 一条直线上 N 块石头，起点 0 终点 L。移除 M 块后，使最短跳跃距离最大。

**思路**：二分最短距离 mid，check 判断能否在移除 ≤M 块石头下实现。求最大值。

```cpp
bool check(int dist) {
    int removed = 0, last = 0;  // last = 上一个保留的石头位置
    for (int i = 0; i < N; i++) {
        if (rock[i] - last < dist) removed++;
        else last = rock[i];
    }
    if (L - last < dist) removed++;  // 检查终点
    return removed <= M;
}
```

### 3. 洛谷 P1577 切绳子

> N 条绳子，需要 K 段等长的绳子，求每段最大长度（浮点数）。

**思路**：浮点数二分长度，check 等于 `sum(floor(L[i] / mid)) >= K`。

---

## ⚠️ 易错点

1. **死循环**：`mid = (l+r)/2` 在 l+1=r 时，若 l=mid 会导致死循环。用 `l+(r-l)/2` 并确保区间收缩
2. **整数溢出**：`mid = (l + r) / 2` 当 l+r > 2^31 时溢出，用 `l + (r - l) / 2`
3. **check 函数写成 O(n²)**：check 必须是 O(n) 或 O(n log n)，否则总体退化为 O(n² log n)
4. **二分答案的单调性**：答案值必须随 mid 单调变化（砍树：H↑ → 木材↓），否则不能二分
5. **二分边界设错**：r 如果太小，可能错过正确答案
6. **浮点数二分用 `while(l<r)` 死循环**：浮点精度问题，用固定次数或 eps
