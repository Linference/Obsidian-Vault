# 算法基础 — MOC

## 模块概述

算法基础是 CSP-J/S 的核心模块，覆盖了 NOI 大纲要求的入门级和基础级算法。掌握这些内容后，可以解决 CSP-J 全部题目和 CSP-S 的部分题目。

## 知识地图

```
算法基础
├── 复杂度分析        → 为什么学算法？怎么判断算法好坏？
├── 枚举与模拟        → 暴力出奇迹，但要有章法
├── 排序算法全集      → 不只是调用 sort()，要懂原理
├── 二分查找与二分答案 → 从 O(n) 到 O(log n) 的质变
├── 前缀和与差分      → 空间换时间的典范
├── 双指针与滑动窗口  → O(n²) 降到 O(n) 的利器
├── 贪心算法          → 局部最优 → 全局最优（需要证明）
├── 递推与递归        → 程序设计的灵魂
└── 高精度计算        → 突破 long long 的限制
```

## 学习路线

### 第一阶段（J组必备）
1. [[复杂度分析]] — 先懂"好坏"
2. [[枚举与模拟]] — 暴力基本功
3. [[排序算法全集]] — 至少掌握冒泡、归并、sort()
4. [[递推与递归]] — 分治思想雏形

### 第二阶段（J组高分 + S组入门）
5. [[二分查找与二分答案]] — 考查频率极高
6. [[前缀和与差分]] — 区间问题的杀手锏
7. [[贪心算法]] — 需要证明能力

### 第三阶段（S组必备）
8. [[双指针与滑动窗口]]
9. [[高精度计算]]
10. 进入 [[05-数据结构]] 模块深造

## 核心能力目标

- 能够分析代码复杂度，说出大O记号
- 能够根据数据范围反推算法
- 能够手写二分、归并排序、前缀和
- 能够识别"贪心可行"的信号
- 能够将递归转递推（避免爆栈）
- 能够处理大整数运算
