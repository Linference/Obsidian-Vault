# 双指针与滑动窗口

## 核心思想

**双指针**：用两个指针在数组上移动，将 O(n²) 枚举降为 O(n)。两个指针可以：

- **同向移动** → 滑动窗口
- **对向移动** → 两数之和类问题

**滑动窗口**：维护一个左闭右闭/左闭右开的区间 [l, r]，右指针扩展窗口，左指针在条件不满足时收缩窗口。暴力 O(n²) 的枚举 → O(n)。

---

## 滑动窗口模板

### 通用框架

```cpp
int l = 0, ans = 0;
for (int r = 0; r < n; r++) {
    // 1. 加入 a[r]，更新窗口内状态
    add(a[r]);

    // 2. 当窗口不满足条件时，收缩左边界
    while (!check()) {
        remove(a[l]);
        l++;
    }

    // 3. 此时窗口 [l, r] 满足条件，更新答案
    ans = max(ans, r - l + 1);   // 最大/最长类
    // ans += r - l + 1;          // 计数类
}
```

> 核心：**右指针 r 只增不减 → O(2n) = O(n)**。

## 经典题型

### 1. 最长无重复字符子串

```cpp
int lengthOfLongestSubstring(string s) {
    int cnt[256] = {0}, l = 0, ans = 0;   // cnt 记录字符出现次数
    for (int r = 0; r < s.size(); r++) {
        cnt[s[r]]++;
        while (cnt[s[r]] > 1) {           // 有重复 → 收缩
            cnt[s[l]]--;
            l++;
        }
        ans = max(ans, r - l + 1);
    }
    return ans;
}
```

### 2. 长度最小的子数组

> 求满足 sum ≥ target 的最短连续子数组长度。

```cpp
int minSubArrayLen(int target, vector<int>& nums) {
    int l = 0, sum = 0, ans = INT_MAX;
    for (int r = 0; r < nums.size(); r++) {
        sum += nums[r];
        while (sum >= target) {
            ans = min(ans, r - l + 1);
            sum -= nums[l++];
        }
    }
    return ans == INT_MAX ? 0 : ans;
}
```

### 3. 和为 k 的子数组数量

```cpp
int subarraySum(vector<int>& nums, int k) {
    unordered_map<int, int> mp;   // 前缀和 → 出现次数
    mp[0] = 1;
    int sum = 0, cnt = 0;
    for (int r = 0; r < nums.size(); r++) {
        sum += nums[r];
        cnt += mp[sum - k];        // 若之前有前缀和为 sum-k
        mp[sum]++;
    }
    return cnt;
}
```

> ⚠️ 本题不能用标准滑动窗口，因为数组有负数，sum 不单调！

## 对向双指针

### 两数之和（有序数组）

```cpp
vector<int> twoSum(vector<int>& nums, int target) {
    int l = 0, r = nums.size() - 1;
    while (l < r) {
        int sum = nums[l] + nums[r];
        if (sum == target) return {l, r};
        else if (sum < target) l++;
        else r--;
    }
    return {-1, -1};
}
```

### 三数之和

```cpp
vector<vector<int>> threeSum(vector<int>& nums, int target) {
    sort(nums.begin(), nums.end());
    vector<vector<int>> res;
    int n = nums.size();
    for (int i = 0; i < n - 2; i++) {
        if (i > 0 && nums[i] == nums[i-1]) continue;  // 去重
        int l = i + 1, r = n - 1;
        while (l < r) {
            int sum = nums[i] + nums[l] + nums[r];
            if (sum == target) {
                res.push_back({nums[i], nums[l], nums[r]});
                while (l < r && nums[l] == nums[l+1]) l++;  // 去重
                while (l < r && nums[r] == nums[r-1]) r--;
                l++; r--;
            } else if (sum < target) l++;
            else r--;
        }
    }
    return res;
}
```

### 接雨水

```cpp
int trap(vector<int>& height) {
    int l = 0, r = height.size() - 1;
    int leftMax = 0, rightMax = 0, ans = 0;
    while (l < r) {
        leftMax = max(leftMax, height[l]);
        rightMax = max(rightMax, height[r]);
        if (leftMax < rightMax) {
            ans += leftMax - height[l];
            l++;
        } else {
            ans += rightMax - height[r];
            r--;
        }
    }
    return ans;
}
```

## 滑动窗口的"收缩条件"

判断窗口是否合法的函数，必须满足**单调性**：

> 如果窗口 [l, r] 不合法，那么 [l-1, r] 更不合法；如果窗口 [l, r] 合法，那么 [l, r-1] 也合法。

这样才能保证 while 收缩左指针不会错过最优解。

---

## 双指针 vs 二分

| 双指针（对向） | 二分查找 |
|---------------|---------|
| O(n) | O(log n) |
| 不需要随机访问 | 需要随机访问 |
| 适合有序数组的配对查找 | 适合有序数组的单值查找 |
| 三数之和外层 O(n)+内层 O(n)=O(n²) | 三数之和外层 O(n)+内层 O(log n)=O(n log n) |

## 典型真题

- **洛谷 P1638 逛画展**：求包含所有画家的最短区间 → 滑动窗口
- **洛谷 P1886 滑动窗口**：求滑动窗口中的最大/最小值 → 单调队列 + 滑动窗口
- **CSP-J 2020 表达式**：模拟即可，不直接用滑动窗口

## ⚠️ 易错点

1. **非单调情况不能用标准滑动窗口**：数组有负数时 sum 不单调（如"和为 k 的子数组"），改用前缀和 + 哈希表
2. **对向双指针的排重**：三数之和中 `while(l<r && nums[l]==nums[l+1])` 的位置要在找到一个解之后，而不是之前
3. **窗口更新顺序**：必须先加入再判断（`add` → `while(check)`），还是先判断再加入？取决于窗口定义
4. **l 不能超过 r**：收缩时确保 `while(l <= r && !check())`
5. **循环退出条件**：对向双指针用 `while(l < r)` 还是 `while(l <= r)` 取决于是否允许 l==r 处理单元素
6. **滑动窗口不一定只用一个循环**：复杂的可能需要 for r 内套 while l
