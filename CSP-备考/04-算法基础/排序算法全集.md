# 排序算法全集

## 核心思想

排序是将无序数据按某种规则（升序/降序）重新排列。CSP 中直接调 `sort()` 即可，但**手写归并排序、理解各排序原理**是必考内容。

---

## 算法对比总表

| 算法 | 平均时间 | 最坏时间 | 空间 | 稳定 | 适用场景 |
|------|---------|---------|------|------|---------|
| 冒泡排序 | O(n²) | O(n²) | O(1) | 是 | 教学、n 极小时 |
| 选择排序 | O(n²) | O(n²) | O(1) | 否 | 交换次数最少 |
| 插入排序 | O(n²) | O(n²) | O(1) | 是 | 接近有序时接近 O(n) |
| 归并排序 | O(n log n) | O(n log n) | O(n) | 是 | **求逆序对** |
| 快速排序 | O(n log n) | O(n²) | O(log n) | 否 | 实际最快的内排序 |
| 堆排序 | O(n log n) | O(n log n) | O(1) | 否 | 不需递归 |
| 计数排序 | O(n+k) | O(n+k) | O(n+k) | 是 | 值域 k 较小时 |
| 基数排序 | O(d(n+k)) | O(d(n+k)) | O(n+k) | 是 | 多位整数排序 |

## 1. 冒泡排序

```cpp
void bubbleSort(int a[], int n) {
    for (int i = 0; i < n - 1; i++)           // n-1 轮
        for (int j = 0; j < n - 1 - i; j++)   // 每轮比较未排好的部分
            if (a[j] > a[j + 1])
                swap(a[j], a[j + 1]);
}

void bubbleSortOptimized(int a[], int n) {     // 优化：提前结束
    for (int i = 0; i < n - 1; i++) {
        bool swapped = false;
        for (int j = 0; j < n - 1 - i; j++)
            if (a[j] > a[j + 1]) {
                swap(a[j], a[j + 1]);
                swapped = true;
            }
        if (!swapped) break;                   // 本轮无交换 → 已有序
    }
}
```

## 2. 选择排序

```cpp
void selectionSort(int a[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int minIdx = i;
        for (int j = i + 1; j < n; j++)
            if (a[j] < a[minIdx]) minIdx = j;
        swap(a[i], a[minIdx]);
    }
}
```

## 3. 插入排序

```cpp
void insertionSort(int a[], int n) {
    for (int i = 1; i < n; i++) {
        int key = a[i];
        int j = i - 1;
        while (j >= 0 && a[j] > key) {
            a[j + 1] = a[j];
            j--;
        }
        a[j + 1] = key;
    }
}
```

接近有序时，while 循环很少进入，时间接近 O(n)。

## 4. 归并排序（重点）

```cpp
const int N = 100005;
int a[N], tmp[N];

void mergeSort(int l, int r) {
    if (l >= r) return;
    int mid = (l + r) >> 1;
    mergeSort(l, mid);
    mergeSort(mid + 1, r);

    int i = l, j = mid + 1, k = l;
    while (i <= mid && j <= r) {
        if (a[i] <= a[j]) tmp[k++] = a[i++];
        else tmp[k++] = a[j++];
    }
    while (i <= mid) tmp[k++] = a[i++];
    while (j <= r)   tmp[k++] = a[j++];
    for (int i = l; i <= r; i++) a[i] = tmp[i];
}
```

### 归并排序求逆序对

```cpp
long long mergeSortCount(int l, int r) {
    if (l >= r) return 0;
    int mid = (l + r) >> 1;
    long long cnt = mergeSortCount(l, mid) + mergeSortCount(mid + 1, r);

    int i = l, j = mid + 1, k = l;
    while (i <= mid && j <= r) {
        if (a[i] <= a[j]) tmp[k++] = a[i++];
        else {
            cnt += mid - i + 1;    // 关键！a[i..mid] 都 > a[j]
            tmp[k++] = a[j++];
        }
    }
    while (i <= mid) tmp[k++] = a[i++];
    while (j <= r)   tmp[k++] = a[j++];
    for (int i = l; i <= r; i++) a[i] = tmp[i];
    return cnt;
}
```

## 5. 快速排序

```cpp
void quickSort(int a[], int l, int r) {
    if (l >= r) return;
    int pivot = a[(l + r) >> 1];     // 取中间值避免退化
    int i = l, j = r;
    while (i <= j) {
        while (a[i] < pivot) i++;
        while (a[j] > pivot) j--;
        if (i <= j) swap(a[i++], a[j--]);
    }
    quickSort(a, l, j);
    quickSort(a, i, r);
}
```

**最坏 O(n²)** 出现在每次 pivot 是最大/最小值（已有序数组+选首元素为pivot）。**随机化 pivot 或三数取中**可避免。

## 6. 堆排序

```cpp
#include <queue>
void heapSort(int a[], int n) {
    priority_queue<int, vector<int>, greater<int>> pq;  // 小根堆
    for (int i = 0; i < n; i++) pq.push(a[i]);
    for (int i = 0; i < n; i++) {
        a[i] = pq.top();
        pq.pop();
    }
}
```

纯手写堆排序考试不常见，了解原理即可。详见 [[05-数据结构]] 中的堆章节。

## 7. 计数排序

```cpp
void countingSort(int a[], int n, int maxVal) {
    vector<int> cnt(maxVal + 1, 0);
    for (int i = 0; i < n; i++) cnt[a[i]]++;
    int idx = 0;
    for (int v = 0; v <= maxVal; v++)
        while (cnt[v]--) a[idx++] = v;
}
```

值域 k 必须可控。`n=10⁶, k=10⁶` 能过；`n=10⁵, k=10⁹` 不行。

## 8. 基数排序

```cpp
void radixSort(int a[], int n) {
    int maxVal = *max_element(a, a + n);
    vector<int> bucket[10];
    for (int exp = 1; maxVal / exp > 0; exp *= 10) {
        for (int i = 0; i < 10; i++) bucket[i].clear();
        for (int i = 0; i < n; i++)
            bucket[(a[i] / exp) % 10].push_back(a[i]);
        int idx = 0;
        for (int i = 0; i < 10; i++)
            for (int x : bucket[i]) a[idx++] = x;
    }
}
```

## STL sort 用法

```cpp
#include <algorithm>

int a[] = {3, 1, 4, 1, 5};
int n = 5;

sort(a, a + n);                      // 升序，O(n log n)
sort(a, a + n, greater<int>());      // 降序

// 自定义比较
struct Node { int x, y; };
Node arr[n];
sort(arr, arr + n, [](Node &p, Node &q) {
    return p.x != q.x ? p.x < q.x : p.y < q.y;  // 先按 x，再按 y
});
```

## 稳定性

**稳定排序**：相等元素的相对顺序在排序后不变。

- 稳定：冒泡、插入、归并、计数、基数
- 不稳定：选择、快排、堆排

C++ 中 `stable_sort()` 保证稳定，复杂度 O(n log² n) 或 O(n log n)（有额外空间时）。

## 内排序 vs 外排序

| | 内排序 | 外排序 |
|------|--------|--------|
| 数据存储 | 全部在内存 | 部分在内存，部分在外存（硬盘） |
| 瓶颈 | CPU | I/O（读写磁盘） |
| 代表 | 上述全部 | 多路归并 |

---

## ⚠️ 易错点

1. **快排对已有序数组退化**：取中间元素为 pivot 或使用 `sort()`（STL 已优化）
2. **归并排序空间 O(n)**：需要辅助数组 tmp，不要每个递归层都 new
3. **自定义比较函数的严格弱序**：`cmp(a,a)` 必须返回 false，否则崩溃
4. **整数溢出**：归并求逆序对时 `cnt += mid - i + 1` 可能超 int，要用 `long long`
5. **计数排序忘记清空桶**：基数排序每轮要清空 bucket
6. **sort 左闭右开**：`sort(a, a+n)` 而非 `sort(a, a+n-1)`
