# 前缀和与差分

## 核心思想

**前缀和** 和 **差分** 是互逆的两个技巧，核心是 **空间换时间**：

- **前缀和**：预处理 O(n)，之后每次区间求和 O(1)
- **差分**：预处理 O(1) 完成一次区间修改，最后还原 O(n)

---

## 一维前缀和

### 定义

`pref[i]` = 数组 a 前 i 个元素的和 = `a[1] + a[2] + ... + a[i]`（1-indexed）

### 构建

```cpp
int a[N], pref[N];
for (int i = 1; i <= n; i++)
    pref[i] = pref[i - 1] + a[i];
```

### 区间查询

$$sum(l, r) = pref[r] - pref[l-1]$$

```cpp
int query(int l, int r) {
    return pref[r] - pref[l - 1];
}
```

> `[2, 5]` 的和 = `pref[5] - pref[1]`，O(1) 完成！

---

## 二维前缀和

### 定义

`pref[i][j]` = 以 (1,1) 为左上角，(i,j) 为右下角的子矩阵元素和。

### 图解

```
pref[i][j] = a[i][j] + pref[i-1][j] + pref[i][j-1] - pref[i-1][j-1]
                         ^^^^^^^^^^^^   ^^^^^^^^^^^^   ^^^^^^^^^^^^^^^^
                         上面矩形      左边矩形       重复减的部分
```

### 构建

```cpp
int a[N][N], pref[N][N];
for (int i = 1; i <= n; i++)
    for (int j = 1; j <= m; j++)
        pref[i][j] = a[i][j] + pref[i-1][j] + pref[i][j-1] - pref[i-1][j-1];
```

### 子矩阵查询

求以 (x1, y1) 为左上角，(x2, y2) 为右下角的子矩阵和：

```cpp
int query(int x1, int y1, int x2, int y2) {
    return pref[x2][y2] - pref[x1-1][y2] - pref[x2][y1-1] + pref[x1-1][y1-1];
}
```

图解：

```
完整大矩形[x2][y2]
- 上面部分[x1-1][y2]
- 左边部分[x2][y1-1]
+ 左上角重复减的部分[x1-1][y1-1]
```

## 一维差分

### 问题场景

> 对数组进行 M 次操作：每次将区间 [l, r] 内所有数加 val。求最终数组。

暴力 O(n×M) 可能超时。

### 核心思路

设差分数组 `diff[]`，满足 `a[i] = diff[1] + diff[2] + ... + diff[i]`（即 diff 是 a 的"相邻差"）。

区间操作：
```cpp
diff[l] += val;
diff[r + 1] -= val;
```

**原理**：对 diff[l] 加 val 后，之后所有前缀和都加 val。对 diff[r+1] 减 val 后，r+1 之后的前缀和抵消，**只有 [l, r] 受影响**。

### 完整代码

```cpp
int a[N], diff[N];

// 构建差分数组
for (int i = 1; i <= n; i++)
    diff[i] = a[i] - a[i-1];              // 也可：diff[i] += a[i]; diff[i+1] -= a[i];

// M 次区间加操作
while (M--) {
    int l, r, val;
    cin >> l >> r >> val;
    diff[l] += val;
    diff[r + 1] -= val;
}

// 还原：对 diff 求前缀和
for (int i = 1; i <= n; i++) {
    diff[i] += diff[i - 1];              // 此时 diff[i] 就是结果 a[i]
    cout << diff[i] << " ";
}
```

## 二维差分

### 问题场景

> 对矩阵进行 M 次操作：将子矩阵 (x1,y1)~(x2,y2) 内所有元素加 val。

### 核心操作

```cpp
diff[x1][y1]     += val;
diff[x2+1][y1]   -= val;
diff[x1][y2+1]   -= val;
diff[x2+1][y2+1] += val;
```

图解（四个角点）：

```
(x1,y1)-----+--------+
  |   +val  | -val   |
  |         |        |
  +-----------------(x2,y2)
  |   -val  | +val   |
  +---------+--------+
```

### 完整代码

```cpp
int diff[N][N];

// 构建差分（初始矩阵为 a）
for (int i = 1; i <= n; i++)
    for (int j = 1; j <= m; j++)
        diff[i][j] = a[i][j] - a[i-1][j] - a[i][j-1] + a[i-1][j-1];

// 或直接用"区间加"来构建
for (int i = 1; i <= n; i++)
    for (int j = 1; j <= m; j++) {
        diff[i][j] += a[i][j];
        diff[i+1][j] -= a[i][j];
        diff[i][j+1] -= a[i][j];
        diff[i+1][j+1] += a[i][j];
    }

// M 次子矩阵加
while (M--) {
    int x1, y1, x2, y2, val;
    cin >> x1 >> y1 >> x2 >> y2 >> val;
    diff[x1][y1] += val;
    diff[x2+1][y1] -= val;
    diff[x1][y2+1] -= val;
    diff[x2+1][y2+1] += val;
}

// 还原：二维前缀和
for (int i = 1; i <= n; i++)
    for (int j = 1; j <= m; j++) {
        diff[i][j] += diff[i-1][j] + diff[i][j-1] - diff[i-1][j-1];
        cout << diff[i][j] << " \n"[j == m];
    }
```

## 前缀和与差分的互逆关系

```
原始数组 a
    ↓ 前缀和
前缀和数组 pref    ←→    pref 的差分 = a
    ↓ 差分
差分数组 diff      ←→    diff 的前缀和 = a
```

本质上：**前缀和和差分是互逆操作**。理解这一点就不会忘了公式。

## 典型真题

### 1. 洛谷 P1719 最大子段和 / 最大子矩阵

一维：`pref[i] = pref[i-1] + a[i]`，最大子段和 = `max(pref[r] - min(pref[0..r-1]))`

二维：枚举上下边界，压成一维后求最大子段和 → O(n³)

### 2. 洛谷 P3406 海底高铁

> N 个城市，M 段行程，每段铁路有工本费和每次费用。求最优购卡策略。

**思路**：用差分统计每条铁路经过次数，然后 `min(cost1 * cnt, cost2 * cnt + cost_card)`。

### 3. CSP-J 2020 直播获奖

- 不能直接用前缀和，但差分+桶排序思想可以做到 O(600n)。

## ⚠️ 易错点

1. **下标从 0 开始时公式要调整**：pref[i] = a[0..i] 的和，区间 [l,r] = pref[r] - pref[l-1]（要判 l=0）
2. **一维差分还原忘记 r+1 不能越界**：diff 数组至少开 n+2
3. **二维差分四个角的符号记混**：+val 的补丁在 (x2+1, y2+1)，而不是 (x1+1, y1+1)
4. **差分还原顺序错误**：一维先累加 diff[i]，二维先算内部两维前缀和
5. **pref 数组不开 long long**：n=10⁵, a[i]=10⁹ → sum=10¹⁴ 超 int
6. **前缀和/差分与原始数组混用同一数组**：可以用同一数组但要注意更新顺序
