# 高精度计算

## 核心思想

C++ 的 `long long` 只能存到约 9×10^18，当数字超出这个范围（如 100!、2^100），就需要用**数组/字符串**存储每一位，手动模拟竖式计算。

---

## 数据存储方式

**约定：vector<int> 倒序存储**，即 `a[0]` 是个位，`a[1]` 是十位…… 便于进位处理。

```cpp
// 字符串 -> 高精度（倒序存）
vector<int> toBigInt(string s) {
    vector<int> a;
    for (int i = s.size() - 1; i >= 0; i--)
        a.push_back(s[i] - '0');
    return a;
}

// 高精度 -> 字符串（输出用）
string toString(vector<int> &a) {
    if (a.empty()) return "0";
    string s;
    for (int i = a.size() - 1; i >= 0; i--)
        s += (a[i] + '0');
    return s;
}
```

## 1. 高精度加法

```cpp
// A + B = C
vector<int> add(vector<int> &A, vector<int> &B) {
    vector<int> C;
    int t = 0;                          // 进位
    for (int i = 0; i < A.size() || i < B.size() || t; i++) {
        if (i < A.size()) t += A[i];
        if (i < B.size()) t += B[i];
        C.push_back(t % 10);            // 当前位
        t /= 10;                        // 进位传给下一位
    }
    return C;
}
```

复杂度：O(max(lenA, lenB))。

## 2. 高精度减法

**前提**：A >= B，否则交换并添加负号。

```cpp
// 比较 A 和 B，A >= B 返回 true
bool cmp(vector<int> &A, vector<int> &B) {
    if (A.size() != B.size()) return A.size() > B.size();
    for (int i = A.size() - 1; i >= 0; i--)
        if (A[i] != B[i]) return A[i] > B[i];
    return true;  // 相等
}

// A - B = C，保证 A >= B
vector<int> sub(vector<int> &A, vector<int> &B) {
    vector<int> C;
    int t = 0;  // 借位
    for (int i = 0; i < A.size(); i++) {
        t = A[i] - t;                   // 带上之前的借位
        if (i < B.size()) t -= B[i];
        C.push_back((t + 10) % 10);     // 取正数
        t = (t < 0) ? 1 : 0;            // 是否需要借位
    }
    // 去掉前导零
    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```

复杂度：O(max(lenA, lenB))。

## 3. 高精度乘法

### 3.1 高精度 × 低精度

```cpp
// A * b = C，b 在 int 范围内
vector<int> mul(vector<int> &A, int b) {
    vector<int> C;
    int t = 0;                          // 进位
    for (int i = 0; i < A.size() || t; i++) {
        if (i < A.size()) t += A[i] * b;
        C.push_back(t % 10);
        t /= 10;
    }
    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```

复杂度：O(lenA)。

### 3.2 高精度 × 高精度

```cpp
// A * B = C（逐位相乘）
vector<int> mul(vector<int> &A, vector<int> &B) {
    vector<int> C(A.size() + B.size() + 10, 0);
    for (int i = 0; i < A.size(); i++)
        for (int j = 0; j < B.size(); j++)
            C[i + j] += A[i] * B[j];

    // 统一处理进位
    for (int i = 0; i < C.size(); i++) {
        C[i + 1] += C[i] / 10;
        C[i] %= 10;
    }

    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```

复杂度：O(lenA × lenB)。可用 **FFT** 优化到 O(n log n)，但 CSP 不考。

## 4. 高精度除法（高精度 ÷ 低精度）

```cpp
// A / b = C，余数为 r
vector<int> div(vector<int> &A, int b, int &r) {
    vector<int> C;
    r = 0;
    for (int i = A.size() - 1; i >= 0; i--) {   // 从高位到低位
        r = r * 10 + A[i];
        C.push_back(r / b);
        r %= b;
    }
    reverse(C.begin(), C.end());          // C 现在是从高到低存，反转
    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```

复杂度：O(lenA)。

**高精度 ÷ 高精度**较复杂（需要用减法模拟试商），一般竞赛较少出现，了解思路即可。

## 5. 压位高精度

为减少数组长度，可以在每个数组元素中存多个十进制位。

```cpp
const int BASE = 10000;  // 每位存 4 位十进制

vector<int> toBigInt(string s) {
    vector<int> a;
    for (int i = s.size() - 1; i >= 0; i -= 4) {
        int val = 0;
        for (int j = max(0, i - 3); j <= i; j++)
            val = val * 10 + (s[j] - '0');
        a.push_back(val);
    }
    return a;
}

vector<int> add(vector<int> &A, vector<int> &B) {
    vector<int> C;
    int t = 0;
    for (int i = 0; i < A.size() || i < B.size() || t; i++) {
        if (i < A.size()) t += A[i];
        if (i < B.size()) t += B[i];
        C.push_back(t % BASE);
        t /= BASE;
    }
    return C;
}
```

压位后输出需注意补前导零（除最高位外）。

## 运算符重载写法（推荐）

将高精度写成结构体，重载运算符，使用时和普通整数几乎一样：

```cpp
struct BigInt {
    vector<int> digits;
    static const int BASE = 10;

    BigInt() {}
    BigInt(string s) {
        for (int i = s.size() - 1; i >= 0; i--)
            digits.push_back(s[i] - '0');
    }
    BigInt(long long x) {
        if (x == 0) digits.push_back(0);
        while (x) { digits.push_back(x % BASE); x /= BASE; }
    }

    BigInt operator+(BigInt &b) { /* 复用 add 函数 */ }
    BigInt operator*(BigInt &b) { /* 复用 mul 函数 */ }
    bool operator<(BigInt &b)  { return !cmp(*this, b); }
    bool operator==(BigInt &b) { /* 比较相等 */ }

    friend ostream& operator<<(ostream &out, BigInt &x) {
        for (int i = x.digits.size() - 1; i >= 0; i--)
            out << x.digits[i];
        return out;
    }
};
```

## 典型真题

### 1. 洛谷 P1009 阶乘之和
计算 1! + 2! + ... + n!（n ≤ 50）。结果远超 long long，必须高精度。

```cpp
vector<int> sum = {0}, fac = {1};    // fac 存当前阶乘
for (int i = 1; i <= n; i++) {
    fac = mul(fac, i);               // 高精度乘以低精度
    sum = add(sum, fac);
}
```

### 2. 洛谷 P1601 A+B Problem（高精）
两位大整数相加。直接用上面的 add 即可。

### 3. CSP-J 2020 优秀的拆分
不需高精度，但如果是输出 2^100 就需要了。

## ⚠️ 易错点

1. **存储顺序**：必须倒序（个位在前），否则进位无法处理
2. **去前导零**：减法、乘法、除法结果末尾可能有 0（如 "0005"），必须去掉但至少留一位 "0"
3. **进位最后仍有残留**：加法循环条件加 `|| t`（进位未处理完）
4. **乘法结果数组大小**：A×B 结果最多 A.size+B.size 位，数组要开够
5. **除法 vs 乘法的遍历方向不同**：除法从高位往低位，乘法从低位往高位
6. **减法必须 A >= B**：先判断大小，不够减就调换并加负号
7. **压位输出补零**：如 BASE=10000，中间值为 23 时输出 "0023"
