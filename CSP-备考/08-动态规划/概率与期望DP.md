# 概率与期望DP

## 核心思想

概率与期望 DP 解决"随机过程"中的最优决策或期望值计算。核心工具是**期望的线性性**。

## 期望的线性性

$$E(X + Y) = E(X) + E(Y)$$

无论 X 和 Y 是否独立，线性性恒成立。这允许我们将复杂期望分解为简单子期望之和。

## 期望 DP 的基本形式

`dp[i]` = 从状态 i 到达目标状态的期望步数/期望收益。

```cpp
// 经典形式：dp[i] = 1 + Σ(P[状态j] × dp[j])
// "1" 表示从当前状态到下一步已经走了一步
```

### 示例：掷骰子期望次数

掷一个公平骰子，第一次掷出 6 的期望掷骰次数。

$$E = \frac{1}{6} \times 1 + \frac{5}{6} \times (1 + E)$$
解得：$E = 6$。

DP 视角：`dp[i]` 表示已连续掷 `i` 次非 6 后的期望次数。推导即可。

## 收集卡牌问题（Coupon Collector）

N 种卡牌，每次随机获得一种，求集齐所有种类的期望次数。

```cpp
// dp[i] = 已经收集了 i 种，还需多少次集齐所有 N 种
// dp[N] = 0
// dp[i] = 1 + (i/N) * dp[i] + ((N-i)/N) * dp[i+1]
// 移项得：dp[i] = N/(N-i) + dp[i+1]
// dp[i] = N * Σ(1/k) for k=1..N-i

double expectedCollect(int N) {
    double ans = 0;
    for (int i = 0; i < N; i++) {
        ans += (double)N / (N - i);
    }
    return ans;
}
// ans = N * H_N  (H_N 为调和数)
```

## 随机游走 / 马尔可夫链

在一个图上随机游走，求从起点到终点的期望步数。

```cpp
// dp[u] = 从节点 u 到终点的期望步数
// dp[u] = 1 + (1/deg(u)) * Σ(dp[v])  for all neighbors v
// 终点: dp[t] = 0

// 如果图无环（DAG），按拓扑序 DP
// 如果有环，需要用高斯消元
```

### 一维随机游走

数轴上从 0 出发，每次 ±1，求首次到达 n 的期望步数。

```cpp
// dp[i] = 从 i 出发到 n 的期望步数
// dp[i] = 1 + 0.5 * dp[i-1] + 0.5 * dp[i+1]
// 差分方程求解：dp[i] - 2*dp[i-1] + dp[i-2] = -2
// 结果是 dp[i] = n^2 - i^2
```

## 概率 DP

某些问题只关心概率而非期望。

```cpp
// dp[i][j] 表示某种联合概率
// 经典题：掷 n 次硬币有 k 次正面
// dp[i][j] = P(前i次有j次正面)
// dp[i][j] = p * dp[i-1][j-1] + (1-p) * dp[i-1][j]
```

## 高斯消元处理成环

当期望 DP 有环（方程相互依赖）时，转化为线性方程组用高斯消元解。

```cpp
// 将 dp[u] = 1 + sum(p[v] * dp[v]) 整理为标准形式：
// dp[u] - sum(p[v] * dp[v]) = 1
// 每个未知数 dp[u] 对应一个方程，高斯消元 O(n³)
```

## 典型例题：CF518D Ilya and Escalator

n 个人在电梯前，每秒有一个人以概率 p 进入。求 t 秒后的期望人数。

```cpp
// dp[i][j] = 前 i 秒有 j 个人在电梯中的概率
// dp[i][j] = (1-p) * dp[i-1][j] + p * dp[i-1][j-1] (j < n)
// dp[i][n] = dp[i-1][n] + p * dp[i-1][n-1]  (满了不再进)
// ans = Σ j * dp[t][j]
```

## ⚠️ 易错点

1. **期望线性性不等价于DP直接取期望**：注意每一步是否独立
2. **有环必须高斯消元**：不能按常规 DP 顺序硬算
3. **概率 + 期望混合**：某状态转移概率不是 1，需乘以对应概率
4. **浮点精度**：期望 DP 常用 double，注意 `1e-8` 比较
5. **初始化**：终点状态期望 = 0，起点期望是答案
6. **分母为 0**：自环或无法到达的状态需要特判
