# 背包问题全集

## 核心思想

背包问题的本质：**在有限容量下，选择物品组合使得总价值最大化。** DP 三要素体现：
- **状态**：`dp[j]` 表示容量为 `j` 时的最大价值
- **转移**：对每件物品，考虑选还是不选
- **边界**：`dp[0] = 0`，其余初始化为 0 或 -INF

## 01背包（每件物品只能选一次）

**核心**：每件物品只能选 0 次或 1 次。

状态转移：`dp[j] = max(dp[j], dp[j - w[i]] + v[i])`，**j 必须逆序枚举**。

为什么逆序？因为 `dp[j-w[i]]` 必须来自"未选第 i 件物品"的状态。正序会导致同一件物品被选多次（变成完全背包）。

```cpp
// 01背包：容量V，n件物品，重量w[i]，价值v[i]
int dp[V + 1] = {0};
for (int i = 0; i < n; i++) {
    for (int j = V; j >= w[i]; j--) {  // 关键：逆序！
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
    }
}
// 答案: dp[V]
```

时间复杂度 O(nV)，空间复杂度 O(V)（滚动数组优化）。

## 完全背包（每件物品无限量）

与 01 背包唯一区别：**j 正序枚举**。

原因：正序时，`dp[j-w[i]]` 可能是已经选了第 i 件物品的状态，允许重复选择。

```cpp
int dp[V + 1] = {0};
for (int i = 0; i < n; i++) {
    for (int j = w[i]; j <= V; j++) {  // 关键：正序！
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
    }
}
```

## 多重背包（每件物品有限量 c[i]）

### 方法一：二进制拆分优化 → 01背包

将每件物品的数量 `c[i]` 拆分为 `1, 2, 4, 8, ..., 剩余`，每份作为独立的 01 背包物品。

原理：1~c[i] 的任意整数都可以由这些 2 的幂次项组合表示。

```cpp
struct Item { int w, v; };
vector<Item> items;

for (int i = 0; i < n; i++) {
    int k = 1;
    while (k <= c[i]) {
        items.push_back({w[i] * k, v[i] * k});
        c[i] -= k;
        k *= 2;
    }
    if (c[i] > 0) {
        items.push_back({w[i] * c[i], v[i] * c[i]});
    }
}

// 对 items 做普通 01 背包
int dp[V + 1] = {0};
for (auto& it : items) {
    for (int j = V; j >= it.w; j--) {
        dp[j] = max(dp[j], dp[j - it.w] + it.v);
    }
}
```

时间复杂度 O(V × Σlog c[i])。

### 方法二：单调队列优化

适用于 `V` 很大但 `n` 较小的场景，时间复杂度 O(nV)，略。

## 分组背包（每组至多选一件）

```cpp
for (int g = 0; g < groupCount; g++) {           // 枚举组
    for (int j = V; j >= 0; j--) {                // 逆序容量
        for (int i : groups[g]) {                 // 枚举组内物品
            if (j >= w[i])
                dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
        }
    }
}
```

注意：容量循环必须在组内物品循环的外层。

## 二维费用背包

每件物品有两种费用（如体积+重量），背包有两种容量限制。

```cpp
int dp[V1 + 1][V2 + 1] = {0};  // dp[i][j] 表示两种容量分别为 i 和 j
for (int i = 0; i < n; i++) {
    for (int j = V1; j >= w1[i]; j--) {       // 逆序
        for (int k = V2; k >= w2[i]; k--) {   // 逆序
            dp[j][k] = max(dp[j][k], dp[j - w1[i]][k - w2[i]] + v[i]);
        }
    }
}
```

## 背包求方案数

将 `max` 改为求和，`dp[0] = 1`，其余为 0。

```cpp
int dp[V + 1] = {0};
dp[0] = 1;
for (int i = 0; i < n; i++) {
    for (int j = V; j >= w[i]; j--) {
        dp[j] = (dp[j] + dp[j - w[i]]) % MOD;
    }
}
// dp[V] 为恰好装满容量 V 的方案数
```

## 背包求具体方案（回溯）

用二维数组 `f[i][j]` 记录"前 i 件物品、容量 j"的最优值，然后逆推。

```cpp
// 假设已计算好 f[n+1][V+1]
int i = n, j = V;
vector<int> chosen;
while (i > 0 && j > 0) {
    if (j >= w[i-1] && f[i][j] == f[i-1][j - w[i-1]] + v[i-1]) {
        chosen.push_back(i - 1);
        j -= w[i-1];
    }
    i--;
}
reverse(chosen.begin(), chosen.end());  // 正序输出
```

## 常见变种对比

| 类型 | 枚举顺序 | 容量循环顺序 |
|------|----------|-------------|
| 01背包 | 每件物品 | 逆序 |
| 完全背包 | 每件物品 | 正序 |
| 多重背包 | 拆分后物品 | 逆序 |
| 分组背包 | 组→容量→组内物品 | 逆序 |

## ⚠️ 易错点

1. **01 背包用正序** → 变成完全背包，逻辑错误
2. **多重背包未拆分直接用二维** → 容易超时
3. **dp 数组未初始化**：求最大值应初始化为 0 或 -INF，求方案数 `dp[0]=1`
4. **dp 数组大小不够**：容量为 V，需要 `V+1` 大小
5. **整数溢出**：涉及方案数取模问题记得 `% MOD`
