## 一、四种进制速查

## 二、进制互转

### 任意进制 → 十进制（按权展开）

$$(a_n\ldots a_0)_R = a_nR^n + \cdots + a_0R^0$$

例：$(1011)_2 = 1×8 + 0×4 + 1×2 + 1×1 = 11$

例：$(1A)_{16} = 1×16 + 10×1 = 26$

### 十进制 → 任意进制（除 R 取余，倒序）

25 → 二进制：
```
25 ÷ 2 = 12 余 1
12 ÷ 2 =  6 余 0
 6 ÷ 2 =  3 余 0
 3 ÷ 2 =  1 余 1
 1 ÷ 2 =  0 余 1
→ (11001)₂（从下到上）
```

### 二↔八↔十六：分组法

| 转换 | 方法 | 例 |
|------|------|-----|
| 二 → 八 | 从右起每 **3 位**一组 | `101110`₂ → `101` `110` → `56`₈ |
| 二 → 十六 | 从右起每 **4 位**一组 | `1011 1010`₂ → `BA`₁₆ |
| 八 → 二 | 每位展开为 3 位 | `35`₈ → `011 101`₂ |
| 十六 → 二 | 每位展开为 4 位 | `A5`₁₆ → `1010 0101`₂ |

## 三、位运算（必考）⭐⭐⭐

### 运算符表

| 运算符 | 含义 | 示例 | 口诀 |
|--------|------|------|------|
| `&` | 按位与 | `5 & 3 = 1` | 全 1 才 1 |
| `|` | 按位或 | `5 | 3 = 7` | 有 1 即 1 |
| `^` | 按位异或 | `5 ^ 3 = 6` | 不同为 1 |
| `~` | 按位取反 | `~5 = -6` | 0 变 1，1 变 0 |
| `<<` | 左移 | `1 << 3 = 8` | 左移 n 位 = ×2ⁿ |
| `>>` | 右移 | `8 >> 2 = 2` | 右移 n 位 = ÷2ⁿ |

### 优先级（从高到低）

`~` → `<<` `>>` → `&` → `^` → `|`

> ⚠️ 位运算优先级低于 `==` 和 `!=`，记得加括号！

## 四、高频位运算技巧 ⭐⭐⭐

这些是复赛写代码时的「肌肉记忆」，也对初赛程序阅读题非常重要。

### 1. 基础运算

| 操作 | 写法 | 说明 |
|------|------|------|
| 乘 2 | `n << 1` | 左移 1 位 |
| 除 2 | `n >> 1` | 右移 1 位（向下取整） |
| 判奇偶 | `n & 1` | 1 = 奇数，0 = 偶数 |
| 取绝对值 | `(n ^ (n >> 31)) - (n >> 31)` | 不依赖分支 |
| 交换两数 | `a ^= b; b ^= a; a ^= b;` | 无需临时变量 |
| 取第 k 位 | `(n >> k) & 1` | k 从 0 起 |
| 第 k 位置 1 | `n | (1 << k)` | |
| 第 k 位清零 | `n & ~(1 << k)` | |

### 2. lowbit：取最低位的 1

```cpp
int lowbit(int x) { return x & -x; }
```

| x | 二进制 | lowbit(x) | 值 |
|---|--------|----------|-----|
| 6 | `0110` | `0010` | 2 |
| 8 | `1000` | `1000` | 8 |
| 12 | `1100` | `0100` | 4 |

> **用途**：树状数组核心操作、判断是否为 2 的幂次

### 3. 消除最低位的 1

```cpp
x = x & (x - 1);
```

| x | 二进制 | x & (x-1) | 效果 |
|---|--------|----------|------|
| 6 | `0110` | `0100` | 消去最低位 1 |
| 8 | `1000` | `0000` | 变成 0（2 的幂次！） |

> **衍生**：
> - 判断 2 的幂次：`x > 0 && (x & (x - 1)) == 0`
> - 统计二进制中 1 的个数（popcount）：
> ```cpp
> int popcount(int x) {
>     int cnt = 0;
>     while (x) { x &= x - 1; cnt++; }
>     return cnt;
> }
> ```

### 4. 异或 XOR 的妙用

| 性质 | 说明 |
|------|------|
| `x ^ x = 0` | 自己异或自己归零 |
| `x ^ 0 = x` | 异或 0 不变 |
| 交换律 + 结合律 | `a ^ b ^ a = b` |

**应用：找出数组中唯一出现奇数次的数**
```cpp
int ans = 0;
for (int x : arr) ans ^= x;
// 最后 ans 就是那个数
```

### 5. 位运算遍历子集（S 组进阶）

```cpp
// 遍历 mask 的所有子集
for (int sub = mask; sub; sub = (sub - 1) & mask) {
    // sub 是 mask 的一个子集
}
```
