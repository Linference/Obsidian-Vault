# 02-C++语法 · MOC

CSP-J/S 初赛的"语法基础盘"——J组约 10~15 分，S组约 8~12 分。**语法不扎实 = 读题都读不明白**。更需要关注意外行为和编译陷阱。

## 笔记索引

| 文件 | 难度 | 核心考点 |
|------|:----:|----------|
| [[数据类型与变量]] | J | 类型字节数、溢出、隐式转换、const |
| [[运算符与表达式]] | J+S | 优先级总表、短路求值、++i vs i++ |
| [[控制结构]] | J | if/switch/for/while、break vs continue |

## 模块特点

- **J 组侧重"能写代码"**——基本类型、循环、数组、简单的STL
- **S 组侧重"理解底层"**——内存对齐、指针、引用、STL复杂度
- **陷阱密度高**——C++ 的"未定义行为""隐式转换""优先级"是出题人的最爱
- **新标准加成**——C++11/14/17 的 auto、range-for、lambda 已多次出现

## 学习路线建议

```
数据类型与变量 → 运算符与表达式 → 控制结构
    ↓                                    ↓
数组与字符串 → 函数与递归 → 结构体与指针引用
    ↓
文件操作 → STL与常用库
```

前 6 篇是基础，必须按顺序吃透。[[文件操作]] 和 [[STL与常用库]] 可以并行学。

> **核心提醒**：CSP 初赛的语法题不是考你能不能写出对的代码，而是考你能不能**找出错的代码**。所以每篇笔记都列出高频陷阱——这些才是真正的考点所在。
