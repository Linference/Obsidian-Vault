# 运算符与表达式

## 一、算术运算符

| 运算符 | 含义 | 示例 | 注意 |
|:------:|------|------|------|
| `+` | 加 | `a + b` | |
| `-` | 减 | `a - b` | |
| `*` | 乘 | `a * b` | 注意溢出 |
| `/` | 除 | `a / b` | **整型截断！** |
| `%` | 取模（余） | `a % b` | 只能用于整数；结果符号同被除数 |

```cpp
int a = 7 / 2;    // 3（截断）
int b = 7 % 2;    // 1
int c = -7 % 3;   // -1（C++ 中符号同被除数）
int d = 7 % -3;   // 1
```

> **注意**：`a % b` 中，b 不能为 0（运行时除以零错误）。模运算结果符号：C++11 起，结果符号与 **a** 相同。

## 二、关系运算符

`>` `<` `>=` `<=` `==` `!=`

返回 `bool` 类型（`true` / `false`，本质是 `1` / `0`）。

```cpp
int x = 5;
bool b1 = (x == 5);   // true (1)
bool b2 = (x != 3);   // true (1)
bool b3 = (x > 10);   // false (0)
```

## 三、逻辑运算符

| 运算符 | 含义 | 真值表 |
|:------:|------|--------|
| `&&` | 逻辑与 | `true && true = true`，其余 false |
| `\|\|` | 逻辑或 | `false \|\| false = false`，其余 true |
| `!` | 逻辑非 | `!true = false`, `!false = true` |

### 短路求值（Short-circuit Evaluation）⚠️ CSP 高频

```cpp
int a = 0;
if (a != 0 && 10 / a > 1) {  // ✅ 安全！a!=0 为 false，后半段不执行
    ...
}

int b = 5;
if (b > 0 || b / 0 > 1) {    // ✅ 安全！b>0 为 true，后半段不执行
    ...
}
```

> **规则**：
> - `A && B`：如果 A 为 false，**B 不会被执行**。
> - `A || B`：如果 A 为 true，**B 不会被执行**。
>
> 出题人最爱：在 B 里放个除以零、数组越界、自增自减，问你"这段代码会崩溃吗"。

```cpp
// 经典真题陷阱
int x = 0;
if (x == 0 || ++x == 1) {
    cout << x;  // 输出 0，不是 1！因为短路求值跳过了 ++x
}
```

## 四、自增自减运算符

| 运算符 | 名称 | 行为 |
|:------:|------|------|
| `++i` | 前置自增 | **先加 1，再返回加后的值** |
| `i++` | 后置自增 | **先返回原值，再加 1** |
| `--i` | 前置自减 | 同理，先减后返回 |
| `i--` | 后置自减 | 同理，先返回后减 |

```cpp
int i = 5;
int a = ++i;  // i=6, a=6
int b = i++;  // b=6, i=7
int c = i--;  // c=7, i=6
int d = --i;  // i=5, d=5
```

```cpp
// 真题陷阱：数组和自增混用
int arr[5] = {1, 2, 3, 4, 5};
int i = 2;
int x = arr[++i];  // x = arr[3] = 4, i=3
int y = arr[i++];  // y = arr[3] = 4, i=4
int z = arr[i--];  // z = arr[4] = 5, i=3
```

## 五、赋值运算符

`=` `+=` `-=` `*=` `/=` `%=` `<<=` `>>=` `&=` `|=` `^=`

```cpp
a += b;   // 等价 a = a + b
a *= b;   // 等价 a = a * b
a /= b;   // 等价 a = a / b（注意整除截断）
```

## 六、三目运算符（条件运算符）

```cpp
int max_val = (a > b) ? a : b;
// 等价于：
// if (a > b) max_val = a; else max_val = b;
```

> **注意**：三目运算符的两个返回表达式类型必须一致，否则会有隐式转换。

## 七、逗号运算符

```cpp
int a = (1, 2, 3);  // a = 3（取最后一个表达式的值）
for (i = 0, j = n; i < j; i++, j--) { ... }  // 常见用法
```

## 八、运算符优先级总表

**从高到低排列**。S 组必背，J 组了解关键几行就行。

| 优先级 | 运算符 | 结合性 |
|:-----:|--------|:-----:|
| 1 | `::` | 左 |
| 2 | `()` `[]` `->` `.` `++` `--` (后置) | 左 |
| 3 | `!` `~` `++` `--` (前置) `+` `-` (一元) `*` `&` (指针) `sizeof` `new` `delete` | **右** |
| 4 | `.*` `->*` | 左 |
| 5 | `*` `/` `%` | 左 |
| 6 | `+` `-` (二元) | 左 |
| 7 | `<<` `>>` | 左 |
| 8 | `<` `<=` `>` `>=` | 左 |
| 9 | `==` `!=` | 左 |
| 10 | `&` (按位与) | 左 |
| 11 | `^` (按位异或) | 左 |
| 12 | `\|` (按位或) | 左 |
| 13 | `&&` | 左 |
| 14 | `\|\|` | 左 |
| 15 | `?:` | **右** |
| 16 | `=` `+=` `-=` ... | **右** |
| 17 | `,` | 左 |

### ⚠️ 优先级三大陷阱

**陷阱一**：位运算 `& ^ |` 优先级低于 `== !=`

```cpp
if (x & 1 == 0)   // ❌ 等价 x & (1 == 0) = x & 0 = 0，永远 false！
if ((x & 1) == 0) // ✅ 加括号
```

**陷阱二**：移位优先级低于加减法

```cpp
int x = 1 << 2 + 1;  // ❌ 等价 1 << (2+1) = 8，不是 (1<<2)+1 = 5
int x = (1 << 2) + 1; // ✅ = 5
```

**陷阱三**：赋值运算优先级最低（除逗号外）

```cpp
int a = 5, b;
b = a == 5;  // ✅ 等价 b = (a == 5)，b = true(1)
```

> **终极口诀**：**不确定优先级的全部加括号！** 加括号不花钱，但错了扣分。

## ⚠️ 易错点

1. **`++i` vs `i++` 在表达式中的行为**——CSP 阅读程序题的最爱。
2. **短路求值**——`A || ++x` 中 `++x` 可能不执行。
3. **位运算优先级超级低**——必须加括号。
4. **`a % b` 中 b 不能为 0**。
5. **除法和取模结果符号**——C++ 中 `/` 向零截断，`%` 结果符号同被除数。
6. **`=` 和 `==` 混淆**——`if (x = 5)` 是赋值，永远为 true；应该 `if (x == 5)`。
7. **三目运算符类型不一致**——`(cond ? 1 : 3.14)` 结果是 `double`，不会保留 int。
