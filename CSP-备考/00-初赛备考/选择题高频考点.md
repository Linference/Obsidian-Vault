## 一、计算机基础（必考 3-5 题）

### 1.1 计算机发展史

| 代 | 核心元件 | 代表 | 常考 |
|----|---------|------|------|
| 第一代 | 电子管 | ENIAC | 世界上第一台通用电子计算机 |
| 第二代 | 晶体管 | — | |
| 第三代 | 集成电路 | — | |
| 第四代 | 大规模集成电路 | — | 当前所处 |

> **冯·诺依曼体系**：运算器、控制器、存储器、输入设备、输出设备。  
> 程序和数据以二进制形式存储，**存储程序**思想。

### 1.2 计算机硬件

| 组件 | 功能 | 常考 |
|------|------|------|
| CPU | 运算器 + 控制器 | ALU 是运算器核心 |
| 内存 | RAM（随机存取，断电丢失）、ROM（只读） | **RAM vs ROM** 高频易错 |
| 外存 | 硬盘、SSD、U 盘 | 速度排序：寄存器 > 缓存 > 内存 > 外存 |
| 总线 | 地址总线、数据总线、控制总线 | |

### 1.3 计算机软件

| 类型 | 例子 | 说明 |
|------|------|------|
| 系统软件 | 操作系统（Windows/Linux）、编译程序 | 管理硬件 + 提供平台 |
| 应用软件 | Word、浏览器、游戏 | 面向用户 |

> **编译 vs 解释**：编译型（C/C++，生成目标代码再执行）vs 解释型（Python，逐行解释执行）

## 二、信息编码（必考 2-3 题）

### 2.1 存储单位

| 单位 | 换算 |
|------|------|
| 1 Byte (B) | 8 bit |
| 1 KB | 1024 B |
| 1 MB | 1024 KB |
| 1 GB | 1024 MB |

> 注意：**硬盘厂商用 1KB=1000B**，但计算机内部用 1024。初赛答案一律用 1024。

### 2.2 ASCII 码

- 标准 ASCII：7 位，128 个字符
- 常见记忆：`'A'=65`，`'a'=97`，`'0'=48`，空格 = 32

### 2.3 汉字编码

- GB2312 / GBK / GB18030：中国标准
- **输入码**（拼音/五笔）→ **机内码**（存储）→ **字形码**（显示）
- 一个汉字占 2 字节（GB 系列）或 1-4 字节（UTF-8）

### 2.4 多媒体文件大小

$$文件大小 = 采样频率 \times 量化位数 \times 声道数 \times 时长$$

> 图像：分辨率 × 位深度 ÷ 8 = 字节数  
> 视频：单帧大小 × 帧率 × 时长 + 音频

## 三、网络基础（必考 1-2 题）

### 3.1 OSI 七层模型（从高到低）

| 层数 | 名称 | 协议/设备 |
|------|------|----------|
| 7 | 应用层 | HTTP, FTP, SMTP, DNS |
| 4 | 传输层 | **TCP**（可靠）/ **UDP**（不可靠但快） |
| 3 | 网络层 | IP, 路由器 |
| 2 | 数据链路层 | MAC 地址, 交换机 |

### 3.2 IP 地址

- IPv4：32 位，4 组十进制（如 192.168.1.1）
- 每段 0-255
- A/B/C 类地址的区分看首段

### 3.3 域名与 URL

- www.example.com：域名
- 顶级域名：.com / .cn / .org / .gov / .edu

## 四、历年必背数字

| 数字 | 含义 |
|------|------|
| 2^8 = 256 | 1 字节取值范围 |
| 2^10 = 1024 | 1 KB |
| 2^16 = 65536 | 16 位范围 |
| 2^31-1 ≈ 2.1×10^9 | int 最大值 |
| 2^63-1 ≈ 9.2×10^18 | long long 最大值 |
| 128 | ASCII 字符数 |
| 65536 | 16 位 Unicode 字符数 |

## 六、C++ 语法选择题

> 详细在 [[02-C++语法/02-MOC]]，常见考点：
> - 数组下标越界（不会报错，UB）
> - 整数除以整数结果还是整数
> - 全局变量默认初始化为 0，局部变量不初始化是垃圾值
> - `i++` vs `++i` 在表达式中
> - `&` 取地址 vs `&` 引用 vs `&&` 逻辑与
> - `>>` 右移一位 = 除以 2（对正整数）
