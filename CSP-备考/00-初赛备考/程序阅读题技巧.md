# 初赛 · 程序阅读题技巧

> CSP-S 初赛 3 大题程序阅读，共 40 分，是拉分最大的题型。

## 题型特征

给出一段 C++ 程序，后面跟着若干道判断题/选择题，要求：
- 判断题：判断给定输入下的输出值
- 选择题：选正确的输出结果

## 三步解题法

### 第 1 步：快速定位算法类型（30 秒）

扫一眼程序，判断它在做什么：

| 代码特征 | 算法判断 |
|---------|---------|
| `for + swap` 嵌套 | 排序（冒泡/选择） |
| `a[i] + a[j] == target` | 两数之和 |
| `dp[i] = max/min(...)` | 动态规划 |
| `while (l < r)` + `mid` | 二分查找 |
| `dfs(...)` 或 `queue+push/pop` | 搜索/图遍历 |
| `fa[x] = find(fa[x])` | 并查集 |
| `% 10 / 10` | 数位处理 |

> 这一步做对了，后面的选择和判断题就有方向了。

### 第 2 步：模拟小数据（3-5 分钟）

取一组**尽量小**的输入（比如 n=3），在草稿纸上**一步一步模拟**程序执行。

```
输入: n = 5
a[] = {3, 1, 4, 1, 5}

模拟笔记格式：
i=0:  a[0]=3  → ...
i=1:  a[1]=1  → ...
```

### 第 3 步：根据题意选答案

- 判断题：以小数据模拟结果为准
- 选择题：先排除明显错误的选项，再在小数据和题给数据中验证

---

## 常见坑

| 坑 | 说明 |
|----|------|
| 循环边界 | `i < n` vs `i <= n`，差一个 |
| 数组下标 0 还是 1 | 审题 + 看数组大小 |
| 全局变量 | 初始化为 0，不要自作多情 |
| `int` 除法 | 3/2 = 1，不是 1.5 |
| `%` 优先级 | `a + b % c` = `a + (b%c)` |
| `for(;;)` 空语句 | 死循环，要执行到 break |
| `return` 提前结束 | 后面代码不会执行 |
